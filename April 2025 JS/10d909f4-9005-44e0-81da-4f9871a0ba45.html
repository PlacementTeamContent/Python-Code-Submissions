String Length Checker


Write the function stringLengthChecker(input) that processes an array of space-separated strings and determines whether each string is short or long based on its length.


    

  • If the string has 5 or fewer characters, classify it as "short".
    • 

  • If the string has more than 5 characters, classify it as "long".
    • 



Input Format


    

  • The input consists of an array of space-separated strings.
    • 



Output Format


    

  • Print space-separated values where each string is classified as "short" or "long" based on its length.
    • 



Constraints


    

  • 1 ≤ N ≤ 100, and each string has a length of at least 1 and at most 10 characters.
    • 



Sample Input:


apple banana cat elephant dog


Sample Output:


short long short long short


Explanation:


In the given space-separated words:


    

  • 'apple' → 5 characters → short
    • 

  • 'banana' → 6 characters → long
    • 

  • 'cat' → 3 characters → short
    • 

  • 'elephant' → 8 characters → long
    • 

  • 'dog' → 3 characters → short
    • 

"use strict";
process.stdin.resume();
process.stdin.setEncoding("utf-8");
let inputString = "";
let currentLine = 0;
process.stdin.on("data", (inputStdin) => {
  inputString += inputStdin;
});

process.stdin.on("end", (_) => {
  inputString = inputString
    .trim()
    .split("\n")
    .map((str) => str.trim());

  main();
});

//Don't modify above code

function stringLengthChecker(input) {
  let output = input.map(word => word.length > 5? "long": "short");
  return output.join(" ");
  
}
//Don't modify below code

function readLine() {
  return inputString[currentLine++];
}

function main() {
  let arr = readLine().split(" ");
  console.log(stringLengthChecker(arr))
}

Check the Rank of a Student


Write the function checkRank(num) that takes a student's score num and returns a string that categorizes the rank based on the following conditions:


    

  • Return "Fail" if the score is less than 50.
    • 

  • Return "Pass" if the score is between 50 and 60 (inclusive).
    • 

  • Return "Good" if the score is between 61 and 75 (inclusive).
    • 

  • Return "Excellent" if the score is greater than 75.
    • 

  • Return "Wrong Evaluation" if the score is outside the range 0 <= num <= 100.
    • 



Input Format


    

  • The input consists of a single integer num representing the student's score.
    • 



Output Format


    

  • Return a string categorizing the student's rank based on the specified conditions.
    • 



Sample Input:


67


Sample Output:


Good


Explanation:


    

  • The score is 67, which is between 61 and 75, so it returns the string Good.
    •
