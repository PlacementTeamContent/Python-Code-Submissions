String Length Checker


Write the function stringLengthChecker(input) that processes an array of space-separated strings and determines whether each string is short or long based on its length.


    

  • If the string has 5 or fewer characters, classify it as "short".
    • 

  • If the string has more than 5 characters, classify it as "long".
    • 



Input Format


    

  • The input consists of an array of space-separated strings.
    • 



Output Format


    

  • Print space-separated values where each string is classified as "short" or "long" based on its length.
    • 



Constraints


    

  • 1 ≤ N ≤ 100, and each string has a length of at least 1 and at most 10 characters.
    • 



Sample Input:


apple banana cat elephant dog


Sample Output:


short long short long short


Explanation:


In the given space-separated words:


    

  • 'apple' → 5 characters → short
    • 

  • 'banana' → 6 characters → long
    • 

  • 'cat' → 3 characters → short
    • 

  • 'elephant' → 8 characters → long
    • 

  • 'dog' → 3 characters → short
    • 

"use strict";
process.stdin.resume();
process.stdin.setEncoding("utf-8");
let inputString = "";
let currentLine = 0;
process.stdin.on("data", (inputStdin) => {
  inputString += inputStdin;
});

process.stdin.on("end", (_) => {
  inputString = inputString
    .trim()
    .split("\n")
    .map((str) => str.trim());

  main();
});

//Don't modify above code

function stringLengthChecker(input) {
  //Write your code here
 let c = "";
 for (let i of input){
     if (i.length>5){
            c = c+"long"+" ";
        
     }
    else {
      c = c+ "short"+" ";
    }
 }
  return c ;
}
//Don't modify below code

function readLine() {
  return inputString[currentLine++];
}

function main() {
  let arr = readLine().split(" ");
  console.log(stringLengthChecker(arr))
}

Modify the Alphabet


Write the function modifyTheString(arr) that processes an array of single-character lowercase alphabets. For each character:


    

  • If the character is a vowel (a, e, i, o, u), replace it with the string "vowel".
    • 

  • If the character is a consonant, replace it with the string "consonant".
    • 



Input Format


    

  • The input consists of a single line containing a space-separated list of lowercase alphabets (a-z), with no numbers or special characters.
    • 



Output Format


    

  • return a space-separated list where each alphabet is replaced with either "vowel" or "consonant" as per the rules.
    • 



Constraints


    

  • The number of characters (N) is between 1 and 100.
    • 



Sample Input:


j i l p d o n


Sample Output:


consonant vowel consonant consonant consonant vowel consonant


Explanation:


In the given space-separated lowercase alphabets:


    

  • 'j' → consonant → consonant
    • 

  • 'i' → vowel → vowel
    • 

  • 'l' → consonant → consonant
    • 

  • 'p' → consonant → consonant
    • 

  • 'd' → consonant → consonant
    • 

  • 'o' → vowel → vowel
    • 

  • 'n' → consonant → consonant
    •
