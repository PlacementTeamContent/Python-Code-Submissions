Number Range Checker


Write the function checkRange(N) to categorize a given integer N into one of the following ranges:


    

  • Return "negative" if the number is less than 0.
    • 

  • Return "small" if the number is between 0 and 10 (inclusive).
    • 

  • Return "medium" if the number is between 11 and 100 (inclusive).
    • 

  • Return "large" if the number is greater than 100.
    • 



You are provided with a prefilled code structure. Your task is to implement the function checkRange(N).


Input Format


    

  • A single integer N.
    • 



Output Format


    

  • The function should return a string based on the categorization of N:
    • 

  • "negative" if N is less than 0.
    • 

  • "small" if N is between 0 and 10 (inclusive).
    • 

  • "medium" if N is between 11 and 100 (inclusive).
    • 

  • "large" if N is greater than 100.
    • 



Constraints


    

  • -10^3 <= N <= 10^3
    • 



Sample Input:


-5


Sample Output:


negative


Explanation:


    

  • The integer -5 is less than 0, so the function returns "negative".
    •

Modify the Alphabet


Write the function modifyTheString(arr) that processes an array of single-character lowercase alphabets. For each character:


    

  • If the character is a vowel (a, e, i, o, u), replace it with the string "vowel".
    • 

  • If the character is a consonant, replace it with the string "consonant".
    • 



Input Format


    

  • The input consists of a single line containing a space-separated list of lowercase alphabets (a-z), with no numbers or special characters.
    • 



Output Format


    

  • return a space-separated list where each alphabet is replaced with either "vowel" or "consonant" as per the rules.
    • 



Constraints


    

  • The number of characters (N) is between 1 and 100.
    • 



Sample Input:


j i l p d o n


Sample Output:


consonant vowel consonant consonant consonant vowel consonant


Explanation:


In the given space-separated lowercase alphabets:


    

  • 'j' → consonant → consonant
    • 

  • 'i' → vowel → vowel
    • 

  • 'l' → consonant → consonant
    • 

  • 'p' → consonant → consonant
    • 

  • 'd' → consonant → consonant
    • 

  • 'o' → vowel → vowel
    • 

  • 'n' → consonant → consonant
    •
