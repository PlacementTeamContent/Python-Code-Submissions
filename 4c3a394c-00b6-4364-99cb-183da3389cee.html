Total Distance Covered


Implement a function total_distance to determine the total distance covered by a car after T hours, starting at an initial distance of D kilometers. The car travels at a constant speed of S kilometers per hour.


Function total_distance will take D, S, and T as first, second, and third arguments respectively and return the total distance covered after T hours.


Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing 3 space-separated integers representing D, S, and T respectively.
    • 





Output Format


    

  • Single line having 1 integer representing the total distance covered after T hours.
    • 





Example 1


Input


10 20 3



Output


70



Explanation


Initial distance is 10 kilometers.
At time 1 hour, total distance will be 10+20 = 30.
At time 2 hours, total distance will be 30+20 = 50.
At time 3 hours, total distance will be 50+20 = 70.




Constraints


    

  • 0 <= D, S, T < 1000
    • 



def total_distance(D, S, T, count=0):
    if count == T:
        return D
    else:
        D = D + S
        return total_distance(D, S, T, count+1)

Message Encryption


You are given a message string M of length N. The message needs to be encrypted before sending it. The message string can be encrypted using the following rules:


    

  • If the k-th character in M is a vowel, change it to the first next character in the alphabetical sequence. For example, the first next character of o is p.
    • 

  • If the k-th character in M is a consonant, change it to the first previous character in the alphabetical sequence. For example, the first previous character of h is g.
    • 



Implement the function encrypt which will take message string M as argument and return encrypted string.


Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing the message string.
    • 





Output Format


    

  • Single line containing the encrypted message string.
    • 





Example 1


Input


cat



Output


bbs



Explanation


c is consonant so change to b

a is vowel so change to b

t is consonant so change to s

So, encrypted message for cat will be bbs.


Example 2


Input


hello



Output


gfkkp



Explanation


h is consonant so change to g

e is vowel so change to f

l is consonant so change to k

l is consonant so change to k

o is vowel so change to p

Constraints

    •

def encrypt(M):
    encrypted_one = ""
    for i in M:
        if i=="a" or i=="e" or i=="o" or i=="i" or i=="u":
            encrypted_one += chr(ord(i)+1)
        else:
            encrypted_one += chr(ord(i)-1)
    return encrypted_one

Difference of Multiples


For a list of integers nums, nums[i] is said to be a Multiple if it is divisible by (i + 1)
Implement the function difference_of_multiples which will take n, size of list and nums as argument and return an integer, the absolute difference between the sum of multiples and the sum of non-multiples.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the absolute difference between sums.
    • 





Example 1


Input


4
2 3 6 8



Output


13



Explanation


    

  • For i = 0, nums[0] = 2, divisible by 1. Multiple.
    • 

  • For i = 1, nums[1] = 3, not divisible by 2. Non-multiple.
    • 

  • For i = 2, nums[2] = 6, divisible by 3. Multiple.
    • 

  • For i = 3, nums[3] = 8, divisible by 4. Multiple.
    • 



Sum of multiples: 2 + 6 + 8 = 16.
Sum of non-multiples: 3.
Absolute difference: |16 - 3| = 13.




Constraints


    

  • 1 <= nums.size() <= 106
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def difference_of_multiples(n, nums):
    int_nums = [int(each_char) for each_char in nums]
    sum_of_multiples = 0
    sum_of_non_multiples = 0
    for i in range(len(int_nums)):
        if (int_nums[i]) % (i+1) == 0:
            sum_of_multiples += int_nums[i]
        else:
            sum_of_non_multiples += int_nums[i]
    if sum_of_multiples > sum_of_non_multiples:
        return sum_of_multiples - sum_of_non_multiples
    return sum_of_non_multiples - sum_of_multiples

- An element is considered matching if it exists in both the main set and the subset.



Example


    

  • For set([1, 2, 3]) and set([1, 4, 5]), matching elements are {1}.
    • 

  • For set([1, 2, 3]) and set([2, 3]), matching elements are {2, 3}.

    • 

  • The second line contains the space-separated elements of the main set F.
    • 

  • The third line contains an integer S that specifies the number of other sets.
    • 

  • The next 2*S lines contain the size of the set followed by space-separated elements of the set.

Output Format


    

  • Print two integers separated by a space.

Alternating Sum


Write a program that traverses an array from left to right, where in each step it examines a sub-array of size k and skips m positions forward after each examination. Your task is to print the alternating sum of values found in each sub-array considered.


Alternating sum of an array can be calculated by adding first value subtracting second and so on.


For example: arr = [a1, a2, a3, a4, a5]
alternating sum of arr will be: (a1-a2+a3-a4-a5)



Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.





Input Format


    

  • The first line contains n size of array.
    • 

  • The second line contains space-separated integers representing the elements of the array arr.
    • 

  • The third line contains an integer k, indicating the size of each sub-array to be considered.
    • 

  • The fourth line contains an integer m, specifying the number of elements to skip after processing each sub-array.
    • 





Output Format


    

  • The output should be a single line consisting of space-separated values, each representing the alternating sum found in the respective sub-array.
    • 





Constraints


    

  • The number of elements in the array arr is greater than zero.
    • 

  • Sub-array size k and the number of skips m are positive integers.
    • 

  • k is less than or equal to the length of the array.
    • 





Example


Let's take a sample input:


8
4 7 3 9 6 1 8 0
3
1



We need to print:


0 13 0 4 13 -7



Explanation


[4 7 3] 9 6 1 8 0 -> alternating sum is 4-7+3 = 0.
4 [7 3 9] 6 1 8 0 -> moving right by 1 place, alternating sum is 7-3+9 = 13.
4 7 [3 9 6] 1 8 0 -> moving right by 1 place, alternating sum is 3-9+6 = 0.
4 7 3 [9 6 1] 8 0 -> moving right by 1 place, alternating sum is 9-6+1 = 4.
4 7 3 9 [6 1 8] 0 -> moving right by 1 place, alternating sum is 6-1+8 = 13.
4 7 3 9 6 [1 8 0] -> moving right by 1 place, alternating sum is 1-8+0 = -7.
