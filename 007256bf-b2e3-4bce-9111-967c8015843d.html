Central Index


You are given a list of integers numbers containing n elements.


A Central Index is an index i where the sum of all elements before it is equal to the sum of all elements after it.


Write a program to find the Central Index in numbers.


Note: The index starts at 0.




Input Format


    

  • The first line contains an integer n, representing the size of the list.
    • 

  • The second line contains n integers, separated by spaces, representing the list numbers.
    • 



Output Format


    

  • A single line containing the Central Index of the list.
    • 





Constraints


    

  • 0 <= n <= 1000
    • 

  • 1 <= numbers[i] <= 1000
    • 

  • There is guaranteed to be exactly one central index in the list.
    • 





Example 1


Input


5
2 2 4 3 1



Output


2



Explanation


Sum of values from index 0 to 1 is 2 + 2 = 4.
Sum of values from index 3 to 4 is 3 + 1 = 4.


Thus, the central index is 2.

b=input().split()
c=[]
d=[]
for i in b:
    if i=="0":
        
        c+=[i]
    else:
        d+=[i]
result=" ".join(c+d)
print(result)

Partition Index


You are given a list of integers elements of size n.


A Partition Index is an index i where the sum of the elements from index 0 to i-1 is equal to the sum of the elements from index i+1 to n-1.


Write a program to find the Partition Index in the list elements.


Note: Indexing starts at 0.




Input Format


    

  • The first line contains an integer n, the size of the list.
    • 

  • The second line contains the list of n integers, separated by spaces.
    • 



Output Format


    

  • Output a single line containing the Partition Index.
    • 





Constraints


    

  • 0 <= n <= 1000
    • 

  • 1 <= elements[i] <= 1000
    • 

  • There will always be exactly one partition index.
    • 





Example 1


Input


8
2 3 5 6 7 6 5 5



Output


4



Explanation


Sum from index 0 to 3 is 2 + 3 + 5 + 6 = 16.
Sum from index 5 to 7 is 6 + 5 + 5 = 16.


Thus, the partition index is 4.


#Write your code here
a=input().split()
first_half=a[:1]
b=a[1:]
z=sorted(b,reverse=True)
second_half=z[:3]
result=first_half+second_half
sum=0
for i in result:
    sum+=int(i)
sum=int(sum/4)
print(sum)

Executive Call Charges


An executive has a list of phone numbers, and you are given call records for this executive in the following format:



HH:MM:SS # 123-123-123 | HH:MM:SS # 234-234-234 | HH:MM:SS # 345-345-345



Details of the records:


-> Every call detail is separated by a '|' character.
 -> The first block before # i.e., (HH : MM : SS) shows the duration of a call for the phone number next to it.
 -> The phone number is written after # and has the format 999-999-999.
 -> There is no 0 at the beginning of a phone number.
 -> Ignore white spaces.


Write a program to calculate the total bill for all calls if the call rate is Rs. 0.02/second and the first 30 seconds of each call are free.


NOTE: The total bill will be the largest whole number less than or equal to the calculated amount. For example, if the calculated amount is 60.25, the total bill will be 60.




Input Format


    

  • A single line containing a string representing call details separated by '|'.
    • 



Output Format


    

  • Print the total bill.
    • 





Constraints


    

  • 00 <=HH<= 23
    • 

  • 00 <=MM, SS<= 59 
    • 

  • The duration of any call will not exceed 24 hours.
    • 

  • The input string will not be empty and will contain at least one call record.
    • 





Example


Input


00:05:00 # 123-123-123 | 00:04:10 # 234-234-234



Output


9



Explanation


For 123-123-123, total duration in seconds is 5*60+0 = 300 seconds. The cost of calling is (300-30)0.02 = 5.4 rupees.
For 234-234-234, total duration in seconds is 4*60+10 = 250 seconds. The cost of calling is (250-30)0.02 = 4.4 rupees.
Total bill amount is 9.
Here we have subtracted the first 30 seconds while calculating the cost.
