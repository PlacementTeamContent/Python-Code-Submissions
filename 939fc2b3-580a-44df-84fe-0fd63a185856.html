Password Encoding


You are given a password string P of length N. The password needs to be encoded before storing it to the database. The password string can be encoded using the following rules:


    

  • If the k-th character in P is a vowel, change it to the second next character in the alphabetical sequence. For example, the second next character of o is q.
    • 

  • If the k-th character in P is a consonant, change it to the second previous character in the alphabetical sequence. For example, the second previous character of h is f.
    • 



Implement the function encode which will take password string P as argument and return encoded string.



1) The second previous character of b is z.



Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing password string.
    • 





Output Format


    

  • Single line containing encoded password string.
    • 





Example 1


Input


abc



Output


cza



Explanation


a is vowel so change to c
b is consonant so change to z
c is consonant so change to a
So, encoded password for abc will be cza


Example 2


Input


iloveniat



Output


kjqtglkcr



Explanation


i is vowel so change to k
l is consonant so change to j
o is vowel so change to q
v is consonant so change to t
e is vowel so change to g
n is consonant so change to l
i is vowel so change to k
a is vowel so change to c
t is consonant so change to r
So, encoded password for iloveniat will be kjqtglkcr




Constraints


    

  • 1 <=N<= 106
    • 

  • The password string P contains only lowercase alphabets.
    • 



def encode(P) :
    value = "aeiou"
    alpha = "abcdefghijklmnopqrstuvwxyz"
    new_string = ""
    for i in P:
        if i in value:
            x = alpha.index(i) + 2 
            if x == 26:
                new_string += "a"
            elif x == 27:
                new_string += "b"
            else:
                new_string += alpha[x] 
        else:
            x = alpha.index(i) - 2
            if x == -1:
                new_string += "z"
            elif x == -2:
                new_string += "y" 
            else:
                new_string += alpha[x]
    return new_string

Total Calories Consumed


Implement a function total_calories to determine the total calories consumed after T meals, starting with an initial calorie intake of C calories. Each subsequent meal adds a constant number of A calories.


Function total_calories will take C, A, and T as first, second, and third arguments respectively and return the total calories consumed after T meals.


Note: You need not write code to handle input and output.




Input Format


    

  • Single line containing 3 space-separated integers representing C, A, and T respectively.
    • 





Output Format


    

  • Single line having 1 integer representing the total calories consumed after T meals.
    • 





Example 1


Input


500 200 4



Output


1300



Explanation


Initial calories consumed is 500.
At meal 1, total calories will be 500+200 = 700.
At meal 2, total calories will be 700+200 = 900.
At meal 3, total calories will be 900+200 = 1100.
At meal 4, total calories will be 1100+200 = 1300.




Constraints


    

  • 0 <= C, A, T < 5000
    • 



def total_calories(C, A, T):
    x = C 
    for i in range(T):
        x += A 
    return x

Count of Divisible Indices


For a list of integers nums, nums[i] is said to be Index Divisible if i is divisible by nums[i].


Implement the function count_divisible_indices which will take n, size of list and nums as argument and return an integer, count of index-divisible numbers in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the count of index-divisible numbers.
    • 





Example 1


Input


4
1 2 1 4



Output


2



Explanation


    

  • For i = 0, nums[0] = 1, 0 % 1 = 0. Index divisible.
    • 

  • For i = 1, nums[1] = 2, 1 % 2 != 0. Not index divisible.
    • 

  • For i = 2, nums[2] = 1, 2 % 1 = 0. Index divisible.
    • 

  • For i = 3, nums[3] = 4, 3 % 4 != 3. Not index divisible.
    • 



Total index divisible numbers: 2.




Constraints


    

  • 1 <= nums.size() <= 106
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def count_divisible_indices(n, nums):
    count = 0 
    for i in range(n):
        if i % nums[i] == 0:
            count +=1 
    return count

- The intersection of two sets is the set of elements common to both.



Example


    

  • set([1, 2, 3]) ∩ set([2, 4]) is set([2]), which is non-empty.
    • 

  • set([1, 2, 3]) ∩ set([4, 5]) is an empty set.
    •

  • The next 2*R lines contain the size of the set followed by space-separated elements of the set.
    • 



Output Format


    

  • Print True or False followed by an integer separated by space.
    •

6

2 
2 
1 3 
3 
7 8 9



Sample Output


False 1



Explanation


Set E = {1,2,3,4,5,6}
Set-1 = {1,3}
Set Intersection with Set E = {1,3}


Set-2 = {7,8,9}
Set Difference with Set E = {}


Intersection with set-2 resulted in empty set.

Neighbor Element


Given an array of integers C, and an integer K find number of Neighbor Elements.


Element Z is a Neighbor Element if there exists at least 1 element whose absolute difference with Z is less than or equal to K i.e. an element Z is a Neighbor Element, if there is another element in the range [Z-K, Z+K] other than Z itself.




Input Format


    

  • First line contains two integers P and K respectively.
    • 

  • Second line contains P integers separated by space.
    • 





Output Format


    

  • Print a single integer denoting the total number of neighbor elements.
    • 





Constraints


    

  • 1 <=P<= 1000
    • 

  • 0 <=K<= 1000
    • 

  • 0 <=C[i]<= 1000
    • 





Explanation


Let's take an example:


4 5
10 21 13 14



arr = [10, 21, 13, 14]


For Z=10 to be Neighbor element, 
there should be atleast one element in range: [10-5, 10+5] = [5, 15].
So, this is a Neighbor element. 


For Z=21 to be Neighbor element, 
there should be atleast one element in range: [21-5, 21+5] = [16, 26].
There is no element in this range.
So, this is a not Neighbor element.


Similarily,
13 and 14 are Neighbor element 
Hence we have 3 Neighbor element.


#Write your code here
n, k = list(map(int, input().split()))
list_a = list(map(int, input().split()))

count = 0 
for i in list_a:
    x, y =  abs(i-k), abs(i+k)
    for j in range(x, y+1):
        if list_a.count(i) > 1:
            if (j in list_a):
                count += 1 
                break 
        else:
            if (j in list_a) and (j != i):
                count += 1 
                break
                
print(count)

Alternating Sum


Write a program that traverses an array from left to right, where in each step it examines a sub-array of size k and skips m positions forward after each examination. Your task is to print the alternating sum of values found in each sub-array considered.


Alternating sum of an array can be calculated by adding first value subtracting second and so on.


For example: arr = [a1, a2, a3, a4, a5]
alternating sum of arr will be: (a1-a2+a3-a4-a5)



Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.





Input Format


    

  • The first line contains n size of array.
    • 

  • The second line contains space-separated integers representing the elements of the array arr.
    • 

  • The third line contains an integer k, indicating the size of each sub-array to be considered.
    • 

  • The fourth line contains an integer m, specifying the number of elements to skip after processing each sub-array.
    • 





Output Format


    

  • The output should be a single line consisting of space-separated values, each representing the alternating sum found in the respective sub-array.
    • 





Constraints


    

  • The number of elements in the array arr is greater than zero.
    • 

  • Sub-array size k and the number of skips m are positive integers.
    • 

  • k is less than or equal to the length of the array.
    • 





Example


Let's take a sample input:


8
4 7 3 9 6 1 8 0
3
1



We need to print:


0 13 0 4 13 -7



Explanation


[4 7 3] 9 6 1 8 0 -> alternating sum is 4-7+3 = 0.
4 [7 3 9] 6 1 8 0 -> moving right by 1 place, alternating sum is 7-3+9 = 13.
4 7 [3 9 6] 1 8 0 -> moving right by 1 place, alternating sum is 3-9+6 = 0.
4 7 3 [9 6 1] 8 0 -> moving right by 1 place, alternating sum is 9-6+1 = 4.
4 7 3 9 [6 1 8] 0 -> moving right by 1 place, alternating sum is 6-1+8 = 13.
4 7 3 9 6 [1 8 0] -> moving right by 1 place, alternating sum is 1-8+0 = -7.
