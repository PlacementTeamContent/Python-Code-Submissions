Smart Document Formatter


A publishing company uses long documents containing specific keywords that need to be replaced with shorthand codes to speed up formatting. Your task is to write a program that processes the given document and replaces every occurrence of the word "Drafting" with "D2" and the word "Draft" with "D1" , while keeping the rest of the document intact.



- Replacements are case-sensitive; only the words "Draft" and "Drafting" exactly as shown should be replaced.
- Punctuation adjacent to the keywords should be preserved and should not affect replacements.





Input Format


    

  • A single line containing the document text.
    • 





Output Format


    

  • A single line containing the modified document after replacements.
    • 





Constraints


    

  • 1 ≤ Length of document ≤ 1000  
    • 





Example


Input


textDraft in Drafting.


Output


textD1 in D2.


Explanation


The words "Drafting" and "Draft" are replaced by "D2" and "D1", result is "textD1 in D2.".


def replace_keywords(text):
    text = text.replace("Drafting","D2")
    text = text.replace("Draft","D1")
    return text
document_text = input()
modified_text = replace_keywords(document_text)
print(modified_text)

Maximum Distance Between Points


You are given n points located on a one-dimensional line, each having a coordinate value. The goal is to find the maximum possible distance between any two points and also determine how many pairs of points have that exact maximum distance.


Given the coordinates of the points in an array where Xi represents the coordinate of the ith point, write a program to calculate the maximum distance and the count of point pairs with the maximum distance.



- If all points have the same coordinate, the maximum distance is 0.





Input Format


    

  • The first line contains the number of points, n.  
    • 

  • The second line contains n space-separated integers, each representing the coordinate of a point.
    • 



Output Format


    

  • A single line containing two space-separated integers: the maximum distance and the number of pairs that have maximum distance.
    • 





Constraints


    

  • 1 ≤ n ≤ 105  
    • 

  • -109 ≤ Xi ≤ 109
    • 





Example 1


Input:


5
1 2 3 4 10


Output:


9 1


Explanation:


    

  • The maximum distance is 10 - 1 = 9.
    • 

  • There is only one pair of points (1, 10) that has this distance.
    • 

  • Hence, the output is 9 1.
    • 



def max_distance_and_count(coords):
    if not coords:
        return 0 , 0
    min_cord = min(coords)
    max_cord = max(coords)
    if min_cord == max_cord:
        return 0,0
    max_dist = max_cord - min_cord
    count = 0
    for i in range(len(coords)):
        for j in range(i+1,len(coords)):
            if abs(coords[i]-coords[j])==max_dist:
                count+=1 
    return max_dist,count
n = int(input())
coords = list(map(int,input().split()))
max_dist,count = max_distance_and_count(coords)
print(max_dist,count)

Lost Item Removal


You are an explorer collecting items represented by numbers indicating their value. Occasionally, you face a barrier marked by the letter B that forces you to lose the item with the lowest value in your collection so far.


Your task is to process the sequence A containing item values and barriers B from Left to Right following these rules:


    

  • If the element is a number in the sequence A, add that item’s value to  newList (where newList is a Empty array).
    • 

  • If the element is B in the sequence A, remove the smallest value from the newList.
    • 

  • After processing all elements, output the values of the remaining items in the order you collected them.
    • 





Input Format


    

  • First line contains an integer k, the length of sequence A.
    • 

  • Second line contains k space-separated elements (numbers or B).
    • 



Output Format


    

  • Single line containing space-separated values after processing A.
    • 





Constraints


    

  • 1 <= k <= 1000
    • 

  • For every B there will be at least one item to remove.
    • 





Example


Input


9
21 17 12 B 30 8 25 B 19




Output


21 17 30 25 19




Explanation


Step-by-step processing of your collection:


    

  • Add value 21 → newList: [21]
    • 

  • Add value 17 → newList: [21, 17]
    • 

  • Add value 12 → newList: [21, 17, 12]
    • 

  • Barrier B encountered → Remove smallest value (12) → newList: [21, 17]
    • 

  • Add value 30 → newList: [21, 17, 30]
    • 

  • Add value 8 → newList: [21, 17, 30, 8]
    • 

  • Add value 25 → newList: [21, 17, 30, 8, 25]
    • 

  • Barrier B encountered → Remove smallest value (8) → newList: [21, 17, 30, 25]
    • 

  • Add value 19 → newList: [21, 17, 30, 25, 19]
    • 



After all operations, the remaining items in your newList have values 21, 17, 30, 25, and 19, in that order.

Thus, the output is: 21 17 30 25 19.
def process_sequence(sequence):
    new_list = []
    for item in sequence:
        if isinstance(item,int):
            new_list.append(item)
        elif item == 'B' and new_list:
            new_list.remove(min(new_list))
    return new_list
k = int(input())
sequence = input()
sequence = [int(x) if x.isdigit() else x for x in sequence]
result = process_sequence(sequence)
print(*result)

Electrician’s Wire Length Conversion


Electricians often receive wire lengths in decimal but need to convert these measurements into mixed fractions for precise cutting and installation. Using mixed fractions helps avoid errors and ensures wire lengths match design specifications exactly.


Write a program that takes a decimal wire length (in meters) as input and converts it into a mixed fraction, reducing the fractional part to its simplest form.



- The input decimal will not contain repeating decimals. (e.g., 1.3333…33).
- If the fractional part is zero, print only the integer part.





Input Format


    

  • A single decimal number representing the wire length in meters.
    • 



Output Format


    

  • The integer part followed by the simplified fractional part in numerator/denominator form.
    • 





Constraints


0 <= N <= 106  




Example 1


Input


15.75


Output


15 3/4


Explanation


    

  • Integer part: 15  
    • 

  • Fractional part: 0.75 = 3/4 after simplification  
    • 

  • Final output: 15 3/4
    • 



import math
def decimal_to_mixed_fraction(decimal_length):
    integer_part = int(decimal_length)
    fractional_part = decimal_length - integer_part
    if fractional_part == 0:
        return str(integer_part)
    numerator = int(fractional_part*1000000)
    denominator = 1000000
    gcd = math.gcd(numerator,denominator)
    numerator //= gcd
    denominator //= gcd
    return f"{integer_part} {numerator}/{denominator}"
wire_length = float(input())
result = decimal_to_mixed_fraction(wire_length)
print(result)

Student Achievement Benchmark


In a school, students are graded based on their academic performance throughout the semester. Each student receives a score reflecting their overall achievement. The academic committee wants to find the first student in the list who can be considered a benchmark achiever, defined as:


    

  • Every student before them has a lower score.
    • 

  • Every student after them has a higher score.
    • 



Identifying this first benchmark achiever will help the school tailor academic support and recognize outstanding progress.


If no such student exists, return -1.



- If all student scores are identical, there will be no student who meets the benchmark condition, so return -1.





Input Format


    

  • The first line contains an integer k, the number of students.
    • 

  • The second line contains k space-separated integers representing the student's scores.
    • 



Output Format


    

  • A single integer representing the score of the first benchmark achiever, or -1 if no such student exists.
    • 





Constraints


    

  • 3 ≤ k ≤ 106
    • 

  • 1 ≤ scores[i] ≤ 106
    • 





Example


Input:


8
4 2 1 8 9 12 15 16


Output:


12


Explanation:


For the array scores = [4, 2, 1, 8, 9, 12, 15,16]

Considering the student with score 12:

- All students before 12 is (4, 2, 1, 8, 9) have lower scores,

- All students after 12 is (15,16) have higher scores.  


Thus, the output is 12.


k = int(input())
a = list(map(int,input().split()))
left = [float('-inf')] + [max(a[:i]) for i in range(1,k)]
right = [min(a[i+1:]) if i<k-1 else float('-inf') for i in range(k)]
for i in range(k):
    if left[i]<a[i]<right[i]:
        print(a[i])
        break
else:
    print(-1)

Jumbled Key Combinations


You are trying to open a high-tech safe that uses a set of keys represented by characters. However, the keys are jumbled, and you don’t know the exact order to unlock the safe. To find the correct combination, you need to generate every possible unique arrangement of these keys and test them.


Given a string s containing the available keys (characters may repeat), list all unique key combinations that could open the safe.




Input Format


    

  • A single string s representing the available keys.
    • 





Output Format


    

  • Print all unique permutations of the keys separated by spaces.
    • 





Constraints


    

  • 1 ≤ |s| ≤ 9  
    • 

  • s may contain duplicate characters.
    • 





Example


Input


ABC


Output:


ABC ACB BAC BCA CAB CBA


Explanation:


Given the string "ABC", the total number of unique permutations is 6. These permutations are:


    

  • "ABC", "ACB", "BAC", "BCA", "CAB", "CBA"
    • 

from itertools import permutations 
s=input().strip()
unique_perms=set(permutations(s))
result=[''.join(p)for p in unique_perms ]
result.sort()
print(' '.join(result))
