- If a palindrome can be formed, return the lexicographically smallest one.

Output Format


    

  • Print all unique permutations of the characters separated by spaces.
    • 





Constraints


    

  • 1 ≤ |s| ≤ 9  
    • 

  • s may contain duplicate characters.
    • 





Example


Input


ABC


Output:


ABC ACB BAC BCA CAB CBA


Explanation:


Given the string "ABC", the total number of unique permutations is 6. These permutations are:


    

  • "ABC", "ACB", "BAC", "BCA", "CAB", "CBA"

# Implement your logic here
import itertools
a = input()
p = set(itertools.permutations(a))
q = sorted(p)
