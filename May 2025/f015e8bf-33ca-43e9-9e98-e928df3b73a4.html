Keyword Compression Tool


A text compression software replaces frequently used keywords with shorter codes to save space. Your task is to write a program that processes a given text and replaces every occurrence of the word "Processing" with "P2" and "Process" with "P1", keeping the rest of the text unchanged.



- Replacements are case-sensitive; only the words "Process" and "Processing" exactly as shown should be replaced.
- Punctuation adjacent to the keywords should be preserved and should not affect replacements.





Input Format


    

  • A single line containing the text to compress.
    • 



Output Format


    

  • A single line containing the compressed text after replacements.
    • 





Constraints


    

  • 1 ≤ Length of text ≤ 1000  
    • 





Example


Input


ScreeningProcess is Processing.


Output


ScreeningP1 is P2.


Explanation


The words "Processing" and "Process" are replaced by "P2" and "P1", result is "ScreeningP1 is P2.".


def compress_text(text):
    text = text.replace("Processing", "P2")
    text = text.replace("Process", "P1")
    return text

Ordered Removal Process


You are given a sequence A containing integers and the letter X.


Your task is to traverse the sequence from Left to Right and apply the following operations:


    

  • 

    If the element is a number in the sequence A, add it to the newList (where newList is a Empty Array).
    

    • 

  • 

    If the element is X in the sequence A, remove the smallest number from the newList.
    

    • 

  • 

    After processing all elements, output the remaining numbers in their original order of appearance.
    

    • 





Input Format


    

  • First line contains an integer m, the length of sequence A.
    • 

  • Second line contains m space-separated elements (numbers or X).
    • 



Output Format


    

  • Single line containing space-separated numbers after processing A.
    • 





Constraints


    

  • 1 <= m <= 1000
    • 

  • For every X there will be at least one number to delete.
    • 





Example


Input


7
8 5 3 X 11 2 X




Output


8 5 11




Explanation


When processing the input sequence A:
Keep the number 8

Keep the number 5

Keep the number 3

Encounter X, remove the smallest number, which is 3  


Keep the number 11

Keep the number 2

Again X is encountered, remove the smallest number, which is 2  


The remaining numbers after processing are 8, 5 and 11

Thus, the output is 8 5 11.
def ordered_removal(m, sequence):
    new_list = []
    for element in sequence:
        if element != 'X':
            new_list.append(int(element))
        else:
            new_list.remove(min(new_list))
    return ' '.join(map(str, new_list))

m = int(input())
sequence = input().split()
print(ordered_removal(m, sequence))

Employee Salary Difference


In a company, there are n employees, each earning a certain salary. The HR department wants to analyze the salary structure by finding the maximum salary difference between any two employees and counting how many pairs of employees have that exact maximum difference.


Given the salaries of the employees in an array where Si represents the salary of the ith employee, write a program to calculate the maximum salary difference and the number of employee pairs with that maximum salary difference.



- If all employees have the same salary, the difference is 0.





Input Format


    

  • The first line contains the number of employees, n.  
    • 

  • The second line contains n space-separated integers, each representing the salary of an employee.
    • 



Output Format


    

  • A single line containing two space-separated integers: the maximum salary difference and the number of pairs that have the maximum difference.
    • 





Constraints


    

  • 1 ≤ n ≤ 105  
    • 

  • 0 ≤ Si ≤ 109
    • 





Example 1


Input:


5
30000 45000 30000 60000 45000


Output:


30000 2


Explanation:


    

  • The maximum salary difference is 60000 - 30000 = 30000.
    • 

  • There are two pairs of employees with this difference: (30000, 60000) and the other (30000, 60000), since there are two employees earning 30000.
    • 

  • Hence, the output is 30000 2.
    • 



def solve():
    n = int(input())
    salaries = list(map(int, input().split()))

    if len(set(salaries)) == 1:
        print("0 0")
        return

    max_salary = max(salaries)
    min_salary = min(salaries)
    max_difference = max_salary - min_salary

    count_max = salaries.count(max_salary)
    count_min = salaries.count(min_salary)

    if max_salary == min_salary:
      pairs = n * (n - 1) // 2
    else:
      pairs = count_max * count_min

    print(max_difference, pairs)

Decorator’s Fabric Length Conversion


Decorators often receive fabric lengths in decimal but prefer measurements in mixed fractions for cutting and pattern alignment. Converting decimals to mixed fractions helps achieve precise fabric usage and reduces wastage.


Write a program that converts a decimal fabric length into a mixed fraction, simplifying the fractional part to its lowest terms.



- The input decimal will not contain repeating decimals. (e.g., 1.3333…33).
- If the fractional part is zero, print only the integer part.





Input Format


    

  • A single decimal number representing the fabric length.
    • 



Output Format


    

  • The integer part followed by the reduced fractional part in the form numerator/denominator.
    • 





Constraints


0 <= N <= 106 




Example 1


Input


4.375


Output


4 3/8


Explanation


    

  • Integer part: 4  
    • 

  • Fractional part: 0.375 = 3/8 after simplification  
    • 

  • Final output: 4 3/8
    • 



def gcd(a, b):
    if b == 0:
        return a
    return gcd(b, a % b)

def decimal_to_mixed_fraction(decimal):
    integer_part = int(decimal)
    fractional_part = decimal - integer_part

    if fractional_part == 0:
        return str(integer_part)

   
    denominator = 1000  
    numerator = int(round(fractional_part * denominator))

    
    common_divisor = gcd(numerator, denominator)
    numerator //= common_divisor
    denominator //= common_divisor

    return f"{integer_part} {numerator}/{denominator}"


decimal = float(input())


print(decimal_to_mixed_fraction(decimal))

Patterned Stock Tags


A retail company uses stock tags made up of lowercase letters. To improve scanning accuracy and avoid confusion, the company wants to rearrange the characters of each stock tag so that it forms a palindrome — a tag that reads the same forwards and backwards.


Given a stock tag string, help the company find the lexicographically smallest palindrome possible by rearranging its characters. If it’s impossible to form a palindrome, print -1.



- A palindrome is a word, phrase, or sequence of characters that reads the same forward and backward, such as "madam" or "racecar". 
- Lexicographically smallest palindrome means the palindrome that comes first in alphabetical order
- If a palindrome can be formed, return the lexicographically smallest one.
- If it's not possible to form a palindrome, return -1.
- The string S contains only lowercase English alphabets (a-z).





Input Format


    

  • The first line contains an integer T, the number of stock tags.
    • 

  • Each of the following T lines contains a string S, representing a stock tag.
    • 





Output Format


    

  • For each stock tag, print the lexicographically smallest palindrome that can be formed by rearranging its characters, or -1 if impossible.
    • 





Constraints


    

  • 1 ≤ T ≤ 10  
    • 

  • 1 ≤ |S| ≤ 1000  
    • 





Example


Input


3
aassddffggh
aabbcc
aabbccd


Output:


adfgshsgfda
abccba
abcdcba


Explanation:


    

  • Test Case 1 : The stock symbol "aassddffggh" can be rearranged to form the palindrome "adfgshsgfda", which is the lexicographically smallest palindrome.
    • 

  • Test Case 2 : The stock symbol "aabbcc" can be rearranged to form the palindrome "abccba", which is the lexicographically smallest palindrome.
    • 

  • Test Case 3 : The stock symbol "aabbccd" can be rearranged to form the palindrome "abcdcba", which is the lexicographically smallest palindrome.
    • 



def solve():
    t = int(input())
    for _ in range(t):
        s = input()
        counts = {}
        for char in s:
            counts[char] = counts.get(char, 0) + 1
        
        odd_chars = []
        for char, count in counts.items():
            if count % 2 != 0:
                odd_chars.append(char)
        
        if len(odd_chars) > 1:
            print("-1")
            continue
        
        middle_char = ""
        if len(odd_chars) == 1:
            middle_char = odd_chars[0]
            counts[middle_char] -= 1
        
        first_half = []
        for char, count in sorted(counts.items()):
            first_half.extend([char] * (count // 2))
        
        palindrome = "".join(first_half) + middle_char + "".join(reversed(first_half))
        print(palindrome)

A ticketing system generates ticket numbers using only the given digits of 0 and 5. Your task is to form the largest possible ticket number divisible by 90 by arranging these digits. If no such ticket number can be formed, print -1.




Input Format


    

  • The first line contains an integer n, the number of digits.
    • 

  • The second line contains n space separated integers, each either 0 or 5.
    • 





Output Format


    

  • Print the largest ticket number divisible by 90 that can be formed using the given digits, or -1 if none exists.
    • 





Constraints


    

  • 1 ≤ n ≤ 105 
    • 

  • Each element in the array is either 0 or 5
    • 





Example


Input


11
5 5 5 5 5 5 5 5 0 5 5


Output


5555555550


Explanation


    

  • It can be shown that 5555555550 is the largest number which can be formed using the given array and conditions
    • 



def solve():
    n = int(input())
    digits = list(map(int, input().split()))
    
    zeros = digits.count(0)
    fives = digits.count(5)
    
    if zeros == 0:
        print(-1)
        return
    
    num_fives = fives // 9 * 9
    
    if num_fives == 0:
        if zeros > 0:
            print(0)
        else:
            print(-1)
        return
    
    result = '5' * num_fives + '0' * zeros
    print(result)

ABC ACB BAC BCA CAB CBA

import itertools

def unique_permutations(s):
    perms = sorted(list(set("".join(p) for p in itertools.permutations(s))))
    return " ".join(perms)

s = input()

Elite Candidate Selection


In a competitive program, candidates receive a ranking score based on their performance in multiple evaluations. The selection committee wants to identify the first candidate in the list who stands out as an Elite candidate, defined by the following criteria:


    

  • Every candidate before them has a lower ranking score.
    • 

  • Every candidate after them has a higher ranking score.
    • 



Spotting this first  Elite candidate early helps the committee to tailor leadership opportunities and mentorship.


If no such candidate exists, return -1.



- If all ranking scores are the same, no candidate fulfills the condition, then return -1.





Input Format


    

  • The first line contains an integer c, the number of candidates.
    • 

  • The second line contains c space-separated integers representing the ranking scores of the each candidate.
    • 



Output Format


    

  • A single integer representing the ranking score of the first elite candidate, or -1 if none exists.
    • 





Constraints


    

  • 3 ≤ c ≤ 106
    • 

  • 1 ≤ scores[i] ≤ 106
    • 





Example


Input:


6
7 5 4 11 14 16


Output:


11


Explanation:


For the ranking scores [7, 5, 4, 11, 14, 16],

The candidate with score 11 is the first to satisfy the condition:

- All candidates before 11 is  (7, 5, 4) have lower scores,

- All candidates after 11 is (14, 16) have higher scores.


Hence, the output is 11.
def find_elite_candidate(scores):
   

    n = len(scores)
    if n < 3:
        return -1

    for i in range(1, n - 1):
        is_elite = True
       
        for j in range(i):
            if scores[j] >= scores[i]:
                is_elite = False
                break

       
        if is_elite:
            for j in range(i + 1, n):
                if scores[j] <= scores[i]:
                    is_elite = False
                    break

        if is_elite:
            return scores[i]

    return -1

    result = find_elite_candidate(scores)
    print(result)
