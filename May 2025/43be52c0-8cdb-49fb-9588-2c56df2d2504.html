Minimum Value Removal Sequence


You are given a sequence Q containing integers and the letter E.


Your task is to traverse the sequence from Left to Right and apply the following operations:


    

  • 

    If the element is a number in the sequence Q, add it to the newList(where newList is a empty array).
    

    • 

  • 

    If the element is E in the sequence Q, remove the smallest number form the newList .
    

    • 

  • 

    After processing all elements, output the remaining numbers in their original order of appearance.
    

    • 





Input Format


    

  • First line contains an integer m, the length of sequence Q.
    • 

  • Second line contains m space-separated elements (numbers or E).
    • 



Output Format


    

  • Single line containing space-separated numbers after processing Q.
    • 





Constraints


    

  • 1 <= m <= 1000
    • 

  • For every E there will be at least one number to delete.
    • 





Example


Input


8
14 9 6 E 11 3 7 E




Output


14 9 11 7




Explanation


When processing the input sequence Q:


Keep the number 14

Keep the number 9

Keep the number 6

Encounter E, remove the smallest number, which is 6  


Keep the number 11

Keep the number 3

Keep the number 7

Again E is encountered, remove the smallest number, which is 3  


The remaining numbers after processing are 14, 9, 11, and 7

Thus, the output is 14 9 11 7.
# Implement your logic here
def solve():
    a = int(input())
    b = input().split()
    c = []
    
    for x in b:
        if x.isdigit():
            c.append(int(x))
        elif x == 'E':
            if c:
                d = min(c)
                c.remove(d)
    
    print(*(c))

Rearranging Shipment IDs


A logistics company assigns shipment IDs using strings of lowercase letters. To avoid errors during scanning, the company wants each shipment ID to be symmetric — meaning it reads the same forwards and backwards.


Given a shipment ID string, help the company rearrange its characters to form the lexicographically smallest palindrome possible. If no such arrangement exists, print -1.



- A palindrome is a word, phrase, or sequence of characters that reads the same forward and backward, such as "madam" or "racecar". 
- Lexicographically smallest palindrome means the palindrome that comes first in alphabetical order
- If a palindrome can be formed, return the lexicographically smallest one.
- If it's not possible to form a palindrome, return -1.
- The string S contains only lowercase English alphabets (a-z).





Input Format


    

  • The first line contains an integer T, the number of shipment IDs.
    • 

  • Each of the following T lines contains a string S, representing a shipment ID.
    • 





Output Format


    

  • For each shipment ID, print the lexicographically smallest palindrome that can be formed by rearranging its characters, or -1 if impossible.
    • 





Constraints


    

  • 1 ≤ T ≤ 10  
    • 

  • 1 ≤ |S| ≤ 1000  
    • 





Example


Input


3
aassddffggh
aabbcc
aabbccd


Output:


adfgshsgfda
abccba
abcdcba


Explanation:


    

  • Test Case 1 : The stock symbol "aassddffggh" can be rearranged to form the palindrome "adfgshsgfda", which is the lexicographically smallest palindrome.
    • 

  • Test Case 2 : The stock symbol "aabbcc" can be rearranged to form the palindrome "abccba", which is the lexicographically smallest palindrome.
    • 

  • Test Case 3 : The stock symbol "aabbccd" can be rearranged to form the palindrome "abcdcba", which is the lexicographically smallest palindrome.
    • 



from collections import Counter

def solve_single_test_case():
    a = input()
    b = Counter(a)

    c = []
    d = ''
    e = 0

    for char_code in range(ord('a'), ord('z') + 1):
        char = chr(char_code)
        if char in b:
            if b[char] % 2 == 1:
                e += 1
                d = char
            c.extend([char] * (b[char] // 2))

    if e > 1:
        print("-1")
    else:
        f = "".join(c)
        g = f + d + f[::-1]
        print(g)

h = int(input())
for _ in range(h):
    solve_single_test_case()

from fractions import Fraction
import math

def solve():
    a = float(input())
    
    # Convert the float to a Fraction
    # Using a string conversion helps avoid floating-point inaccuracies
    f = Fraction(str(a))
    
    b = f.numerator
    c = f.denominator

    # Calculate the integer part
    d = b // c
    
    # Calculate the remaining fractional part
    e = b % c
    
    if e == 0:
        print(d)
    else:
        # Simplify the fractional part by finding the greatest common divisor
        g = math.gcd(e, c)
        h = e // g
        i = c // g
        
        if d == 0:
            print(f"{h}/{i}")
        else:
            print(f"{d} {h}/{i}")

Biggest Phone Number Ending with Zero


A telecom company wants to generate the biggest possible phone number using only the given digits of 0 and 5. The phone number must divisible by 90. Using the given digits, find the largest such number. If no such number can be formed, print -1.




Input Format


    

  • First line contains an integer n, the number of digits available.
    • 

  • Second line contains n space separated integers, each either 0 or 5.
    • 





Output Format


    

  • Single line containing the largest phone number divisible by 90 formed using the digits, or -1 if not possible.
    • 





Constraints


    

  • 1 ≤ n ≤ 105 
    • 

  • Each element in the array is either 0 or 5
    • 





Example


Input


11
5 5 5 5 5 5 5 5 0 5 5


Output


5555555550


Explanation


    

  • It can be shown that 5555555550 is the largest number which can be formed using the given array and conditions
    • 



def solve():
    a = int(input())
    b = list(map(int, input().split()))

    c = b.count(0)
    d = b.count(5)

    if c == 0:
        print("-1")
        return

    while d % 9 != 0:
        d -= 1

    if d == 0:
        print("-1")
        return

    e = ['5'] * d + ['0'] * c
    f = "".join(e)
    print(f)

Product Quality Benchmark


In a manufacturing company, each batch of products is evaluated and assigned a quality score based on various parameters such as durability, finish, and functionality. The Quality Control team wants to identify the first batch in the production line that stands out as a benchmark batch,it means:


    

  • All batches produced before it have lower quality scores.
    • 

  • All batches produced after it have higher quality scores.
    • 



Identifying this first benchmark batch helps the team optimize production processes and improve overall product quality.


If no such batch exists, return -1.



- If all quality scores are the same, there will be no batch that meets the benchmark condition, so return -1.





Input Format


    

  • The first line contains an integer m, the number of batches.
    • 

  • The second line contains m space-separated integers representing the quality scores of the each batch.
    • 



Output Format


    

  • A single integer representing the quality score of the first benchmark batch, or -1 if no such batch exists.
    • 





Constraints


    

  • 3 ≤ m ≤ 106
    • 

  • 1 ≤ scores[i] ≤ 106
    • 





Example


Input:


6
2 1 3 5 7 10


Output:


3


Explanation:


For the input batch = [2, 1, 3, 5, 7, 10] .
Considering the batch with quality score 3:

- All batches before it (2, 1) have lower scores,

- All batches after it (5, 7, 10) have higher scores.  


Hence, the output is 3.



def find_benchmark_batch(arr):
    n = len(arr)
    left_max = [0] * n
    right_min = [0] * n
    
    left_max[0] = float('-inf')
    for i in range(1, n):
        left_max[i] = max(left_max[i-1], arr[i-1])
        
    right_min[n-1] = float('inf')
    for i in range(n-2, -1, -1):
        right_min[i] = min(right_min[i+1], arr[i+1])
        
    for i in range(n):
        if left_max[i] < arr[i] < right_min[i]:
            return arr[i]
    
    return -1

# Example usage:
m = int(input())
arr = list(map(int, input().split()))
print(find_benchmark_batch(arr))
