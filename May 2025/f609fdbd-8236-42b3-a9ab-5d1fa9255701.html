Bench Seating


In a classroom with n students, the teacher has distributed answer sheets. Some students have performed excellently, while others have scored poorly. To help the weaker students, the teacher wants to reorganize the seating arrangement, pairing weaker students with stronger ones.


The teacher wants to assign two students to the every bench such that the difference in their marks is as large as possible. The marks of the students are given in an array where Mi represents the marks of the ith student. Your task is to write a program to calculate the maximum possible difference in marks between any two students and determine how many such pairs exist with that difference.



- If all students have the same marks, then the difference is 0.





Input Format


    

  • The first line contains the number of students, n.
    • 

  • The second line contains n space-separated integers, where each integer represents the marks of a student.
    • 



Output Format


    

  • A single line containing two space-separated integers, those are the maximum possible difference in marks and the number of pairs that have the maximum difference.
    • 





Constraints


    

  • 1 ≤ n ≤ 105
    • 

  • 0 ≤ Mi ≤ 106
    • 





Example 1


Input:


3
2 5 6


Output:


4 1


Explanation:


    

  • The maximum difference is between the marks of the first and third students, which is 6 - 2 = 4.
    • 

  • There is only one valid pair that has this difference: (2, 6).
    • 

  • Hence, the output is 4 1.
    •

Selective Removal


You are given a sequence S containing integers and the letter E.


Your task is to traverse the sequence from Left to Right and apply the following operations:


    

  • 

    If the element is a number in the sequence S , add it to the newList.
    

    • 

  • 

    If the element is E in the sequence S, remove the smallest number from  the newList.
    

    • 

  • 

    After processing all elements, output the remaining numbers in their original order of appearance.
    

    • 





Input Format


    

  • First line contains an integer n, the length of sequence S.
    • 

  • Second line contains n space-separated elements (numbers or E).
    • 



Output Format


    

  • Single line containing space-separated numbers after processing S
    • 





Constraints


    

  • 1 <=n<= 1000
    • 

  • For every E there will be atleast one number to delete
    • 





Example


Input


7
8 5 3 E 10 2 E




Output


8 5 10




Explanation


When processing the input sequence S:
Keep the number 8
Keep the number 5
Keep the number 3
Encounter E, remove the smallest number, which is 3


Keep the number 10
Keep the number 2
Again E is encountered, remove the smallest number, which is 2


The remaining numbers after processing are 8, 5 and 10
Thus, the output is 8 5 10.


n = int(input())
arr = input().split()
new_list = []

for i in arr:
    if i.isdigit():
        new_list.append(i)
    elif i=='E':
        small = min(new_list)
        new_list.remove(small)
print(" ".join(new_list))

Lexicographical Stock Symbols


In a financial system, stock symbols are generated from various characters. However, some stock symbols are not symmetric and may lead to confusion. The application needs to rearrange the characters of a given stock symbol (denoted as S) to form the lexicographically smallest palindrome. If it's not possible to form a palindrome, the system should return -1.


Your task is to develop an application that takes T test cases, where for each test case, a stock symbol S is given. The application should determine if the given stock symbol S can be rearranged into a palindrome. If it can, return the lexicographically smallest palindrome; otherwise, return -1.



- A palindrome is a word, phrase, or sequence of characters that reads the same forward and backward, such as "madam" or "racecar". 
- Lexicographically smallest palindrome means the palindrome that comes first in alphabetical order
- If a palindrome can be formed, return the lexicographically smallest one.
- If it's not possible to form a palindrome, return -1.
- The string S contains only lowercase English alphabets (a-z).





Input Format


    

  • The first line contains an integer T, the number of test cases.
    • 

  • Each of the following T lines contains a string S, representing a stock symbol.
    • 



Output Format


    

  • For each test case, print a single line containing the lexicographically smallest palindrome that can be formed from the characters of the string S, or -1 if no palindrome can be formed.
    • 





Constraints


    

  • 1 ≤ t ≤ 10
    • 

  • 1 ≤ |s| ≤ 1000
    • 





Example 1


Input:


3
aassddffggh
aabbcc
aabbccd


Output:


adfgshsgfda
abccba
abcdcba


Explanation:


    

  • Test Case 1 : The stock symbol "aassddffggh" can be rearranged to form the palindrome "adfgshsgfda", which is the lexicographically smallest palindrome.
    • 

  • Test Case 2 : The stock symbol "aabbcc" can be rearranged to form the palindrome "abccba", which is the lexicographically smallest palindrome.
    • 

  • Test Case 3 : The stock symbol "aabbccd" can be rearranged to form the palindrome "abcdcba", which is the lexicographically smallest palindrome.
