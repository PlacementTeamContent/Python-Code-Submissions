Serial Number Transformation


You are given a serial number string S of length N. The serial number needs to be transformed before use. The serial number string can be transformed using the following rules:


    

  • If the k-th character in S is a vowel, change it to the fifth next character in the alphabetical sequence. For example, the fifth next character of o is t.
    • 

  • If the k-th character in S is a consonant, change it to the fifth previous character in the alphabetical sequence. For example, the fifth previous character of h is c.
    • 



Implement the function transform_serial which will take serial number string S as argument and return transformed string.



1) The fifth previous character of d is y, c is x, b is w.



Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing the serial number string.
    • 





Output Format


    

  • Single line containing the transformed serial number string.
    • 





Example 1


Input


xyz



Output


stu



Explanation


x is consonant so change to s

y is consonant so change to t

z is consonant so change to u

So, transformed serial number for xyz will be stu.


Example 2


Input


violet



Output


qntgjo



Explanation


v is consonant so change to q

i is vowel so change to n

o is vowel so change to t

l is consonant so change to g

e is vowel so change to j

t is consonant so change to o

So, transformed serial number for violet will be qntgjo.




Constraints


    

  • 1 <=N<= 106
    • 

  • The serial number string S contains only lowercase alphabets.
    • 



def transform_serial(S):
    vowels = 'aeiou'
    letters = 'abcdefghijklmnopqrstuvwxyz'
    new = ''
    for i in S:
        ind = letters.index(i)
        if i in vowels:
            new += letters[ind+5]
        else:
            new += letters[ind-5]
    return new

Good Number Sum


For a list of integers nums, nums[i] is said to be Good Number if parity of i and nums[i] will be same.


Implement the function sum_good_number which will take n, size of list and nums as argument and return an integer, sum of good number in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of good numbers.
    • 





Example 1


Input


3
1 2 3



Output


0



Explanation


    

  • For i = 0, nums[0] = 1, 1 % 2 = 1 and i % 2 = 0. Not a good number.
    • 

  • For i = 1, nums[1] = 2, 2 % 2 = 0 and i % 2 = 1. Not a good number.
    • 

  • For i = 2, nums[2] = 3, 3 % 2 = 1 and i % 2 = 0. Not a good number.
    • 



Hence, the answer is 0.


Example 2


Input


4
2 3 4 5



Output


14



Explanation


    

  • For i = 0, values[0] = 2, 2 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 1, values[1] = 3, 3 % 2 = 1 and i % 2 = 1. Good number.
    • 

  • For i = 2, values[2] = 4, 4 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 3, values[3] = 5, 5 % 2 = 1 and i % 2 = 1. Good number.
    • 



Sum of elements added: 2 + 3 + 4 + 5 = 14.
Hence, the answer is 14.




Constraints


    

  • 1 <=nums.size()<= 106
    • 

  • 1 <=nums[i]<= 1000
    • 

  • Index starts from 0
    • 



def sum_good_number(n, nums):
    good_list = []
    for i in range(n):
        if i % 2 == nums[i]% 2 :
            good_list.append(nums[i])
    return sum(good_list)

Common Character Count


Two strings are said to share commonality if they have at least one character in common. For instance, abc and bcd share characters b and c, but xyz and abc do not share any characters.


Given an array of strings strArr of length n, count the number of pairs of strings that have at least one character in common.



1. Each string consists of lowercase English alphabets only.
2. Pairs are considered index-wise, i.e., identical strings at different indices count as separate pairs.
3. A pair of indices (i,j) is the same as the pair (j,i).





Input Format


    

  • The first line contains an integer n, the number of strings in the array.
    • 

  • The second line contains n space-separated strings, representing strArr.
    • 



Output Format


    

  • Print the number of pairs of strings that share at least one character.
    • 





Constraints


    

  • 1 <= n <= 104
    • 

  • a <= strArr[i][j] <= z
    • 





Example


Example input:


4
abc bcd def ghi



Here:


    

  • abc and bcd share characters, making one pair.
    • 

  • bcd and def share character, making one pair.
    • 

  • No other pairs share characters.
    • 



The answer is 2.


#Write your code here
n = int(input())
strings = input().split()
pairs = 0
for i in range(len(strings)-1):
    for j in strings[i+1:]:
        a = any([k in strings[i] for k in j])
        if a:
            pairs += 1
print(pairs)

Proximate Number


Given an array of integers F, and an integer J find the number of Proximate Numbers.


Element V is Proximate if there is at least 1 element whose absolute difference with V is less than or equal to J. Any element V is considered Proximate, if there is another element in the range [V-J, V+J] other than V itself.




Input Format


    

  • First line contains two integers R and J respectively.
    • 

  • Second line contains R integers separated by space.
    • 





Output Format


    

  • Print a single integer denoting the total number of proximate elements.
    • 





Constraints


    

  • 1 <=R<= 1000
    • 

  • 0 <=J<= 1000
    • 

  • 0 <=F[i]<= 1000
    • 





Explanation


Let's take an example:


4 5
10 21 13 14



arr = [10, 21, 13, 14]


For V=10 to be proximate element, 
there should be atleast one element in range: [10-5, 10+5] = [5, 15].
So, this is a proximate element. 


For V=21 to be proximate element, 
there should be atleast one element in range: [21-5, 21+5] = [16, 26].
There is no element in this range.
So, this is a not proximate element.


Similarily,
13 and 14 are proximate element 
Hence we have 3 proximate element.


#Write your code here
r,j = list(map(int,input().split()))
nums = list(map(int,input().split()))
prox = 0
for i in range(len(nums)):
    v = nums[i]
    count = 0
    ranges = [k for k in range(v-j,v+j+1)]
    for l in nums:
        if (l in ranges):
            count += 1 
    if count > 1:
        prox += 1
print(prox)

Average of Sub-arrays


Write a program that traverses an array from left to right, where in each step it examines a sub-array of size k and skips m positions forward after each examination. Your task is to print the average of values found in each sub-array considered.



Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.





Input Format


    

  • The first line contains n size of array.
    • 

  • The second line contains space-separated integers representing the elements of the array arr.
    • 

  • The third line contains an integer k, indicating the size of each sub-array to be considered.
    • 

  • The fourth line contains an integer m, specifying the number of elements to skip after processing each sub-array.
    • 





Output Format


    

  • The output should be a single line consisting of space-separated values, each representing the average found in the respective sub-array.
    • 





Constraints


    

  • The number of elements in the array arr is greater than zero.
    • 

  • Sub-array size k and the number of skips m are positive integers.
    • 

  • k is less than or equal to the length of the array.
    • 





Example


Let's take a sample input:


8
4 7 3 9 6 1 8 0
3
1



We need to print:


4.67 6.33 6.00 5.33 5.00 3.00



Explanation


[4 7 3] 9 6 1 8 0 -> average is 4.67.
4 [7 3 9] 6 1 8 0 -> moving right by 1 place, average is 6.33.
4 7 [3 9 6] 1 8 0 -> moving right by 1 place, average is 6.00.
4 7 3 [9 6 1] 8 0 -> moving right by 1 place, average is 5.33.
4 7 3 9 [6 1 8] 0 -> moving right by 1 place, average is 5.00.
4 7 3 9 6 [1 8 0] -> moving right by 1 place, average is 3.00.


#Write your code here
n = int(input())
nums = list(map(int,input().split()))
k = int(input())
m = int(input())
index = 0
for i in range(n):
    average = (sum(nums[index:index+k])/k)
    if len(str(average)) >3:
        print(round(average,2),end =" ")
    else :
        print(str(average)+'0', end = ' ')
    index += m
    if index >= n-k+1:
        break

On-Off Lamps


In this exercise, you have N lamps that are initially off. In a sequence of rounds, you toggle these lamps according to specific rules. Initially, you turn on all the lamps. In the second round, you turn off every second lamp. In the third round, you toggle every third lamp.


For the jth round, you toggle every jth lamp. For the Nth round, you toggle only the last lamp.


Calculate the number of lamps that remain on after N rounds.




Input Format


    

  • A single integer N
    • 



Output Format


    

  • A single integer representing the number of lamps that remain on after N rounds.
    • 





Constraints


    

  • 1 <= N <= 1000
    • 





Sample Input 1


5



Sample Output 1


2



Explanation


Let's take an example:


5



    

  • At first, the five lamps are [off, off, off, off, off].
    • 

  • After the first round, the five lamps are [on, on, on, on, on].
    • 

  • After the second round, the five lamps are [on, off, on, off, on].
    • 

  • After the third round, the five lamps are [on, off, off, off, on].
    • 

  • After the fourth round, the five lamps are [on, off, off, on, on].
    • 

  • After the fifth round, the five lamps are [on, off, off, on, off].
    • 

  • So you should print 2 because there are two lamps that are on.
    • 



#Write your code here
n = int(input())
num_list = []
new_list = []

for i in range(n):
    num_list.append(i)
    new_list.append('off')

for i in range(n):
    for j in num_list[i::i+1]:
        if new_list[j] == 'off':
            new_list[j] = 'on'
        else:
            new_list[j] = 'off'
print(new_list.count('on'))
