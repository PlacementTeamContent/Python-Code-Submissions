Username Encoding


You are given a username string U of length N. The username needs to be encoded before storing it in the system. The username string can be encoded using the following rules:


    

  • If the k-th character in U is a vowel, change it to the third next character in the alphabetical sequence. For example, the third next character of o is r.
    • 

  • If the k-th character in U is a consonant, change it to the third previous character in the alphabetical sequence. For example, the third previous character of h is e.
    • 



Implement the function encode which will take username string U as argument and return encoded string.



1) The third previous character of c is z, b is y.



Note: You need not to write code to handle input and output.

def encode(U):
    vowels = "aeiou"
    final_word = ""
    for letter in U:
        if letter not in vowels:
            if letter=="b":
                final_word += "y"
            elif letter=="c":
                final_word += "z"
            else:
                final_word += chr(ord(letter)-3)
        else:
            final_word += chr(ord(letter)+3)
    return final_word

Sum of Matching Squares


For a list of integers nums, nums[i] is said to be Matching Square if the square of nums[i] is equal to the square of i.


Implement the function sum_matching_squares which will take n, size of list and nums as argument and return an integer, the sum of all matching square numbers in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of all matching square numbers.
    • 





Example 1


Input


3
0 1 2



Output


3



Explanation


    

  • For i = 0, nums[0] = 0, 0*0 = 0 and i*i = 0. Matching square.
    • 

  • For i = 1, nums[1] = 1, 1*1 = 1 and i*i = 1. Matching square.
    • 

  • For i = 2, nums[2] = 2, 2*2 = 4 and i*i = 4. Matching square.
    • 



Sum of matching squares: 0 + 1 + 2 = 3.




Constraints


    

  • 1 <= nums.size() <= 1000
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def sum_matching_squares(n, nums):
    # n is the length of nums
    count = 0
    for i in range(n):
        if (abs(i)**2)==(abs(nums[i])**2):
            count += i
    return count

Element W is Close Pair if there exists at least 1 element whose absolute difference with W is less than or equal to H. An element W is Close Pair, if there is another element in the range [W-H, W+H] other than W itself.

Output Format

    • 





Explanation

Common Character Count


Two strings are said to share commonality if they have at least one character in common. For instance, abc and bcd share characters b and c, but xyz and abc do not share any characters.


Given an array of strings strArr of length n, count the number of pairs of strings that have at least one character in common.



1. Each string consists of lowercase English alphabets only.
2. Pairs are considered index-wise, i.e., identical strings at different indices count as separate pairs.
3. A pair of indices (i,j) is the same as the pair (j,i).





Input Format


    

  • The first line contains an integer n, the number of strings in the array.
    • 

  • The second line contains n space-separated strings, representing strArr.
    • 



Output Format


    

  • Print the number of pairs of strings that share at least one character.
    • 





Constraints


    

  • 1 <= n <= 104
    • 

  • a <= strArr[i][j] <= z
    • 





Example


Example input:


4
abc bcd def ghi



Here:


    

  • abc and bcd share characters, making one pair.
    • 

  • bcd and def share character, making one pair.
    • 

  • No other pairs share characters.
    •

#Write your code here
no_of_series = int(input())
series = sorted(input().split())
pairs = 0
for i in range(no_of_series-1):
    set1 = set(series[i])
    set2 = set(series[i+1])
    if set1.intersection(set2):
        pairs += 1

- A strict subset must have all its elements in the main set and have fewer elements than the main set.



Example


    

  • The set([1, 2])is a strict subset of set([1, 2, 3]).
    • 

  • The set([1, 2, 3, 4])is not a strict subset of set([1, 2, 3]).

    • 

  • The second line contains the space-separated elements of the main set C.

  • The next 2*P lines contain the size of each set followed by space-separated elements of the set.

Output Format


    

  • Print True or False followed by an integer separated by space.
    •
