Good Number Sum


For a list of integers nums, nums[i] is said to be Good Number if parity of i and nums[i] will be same.


Implement the function sum_good_number which will take n, size of list and nums as argument and return an integer, sum of good number in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of good numbers.
    • 





Example 1


Input


3
1 2 3




Output


0




Explanation


    

  • For i = 0, nums[0] = 1, 1 % 2 = 1 and i % 2 = 0. Not a good number.
    • 

  • For i = 1, nums[1] = 2, 2 % 2 = 0 and i % 2 = 1. Not a good number.
    • 

  • For i = 2, nums[2] = 3, 3 % 2 = 1 and i % 2 = 0. Not a good number.
    • 



Hence, the answer is 0.


Example 2


Input


4
2 3 4 5




Output


14




Explanation


    

  • For i = 0, values[0] = 2, 2 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 1, values[1] = 3, 3 % 2 = 1 and i % 2 = 1. Good number.
    • 

  • For i = 2, values[2] = 4, 4 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 3, values[3] = 5, 5 % 2 = 1 and i % 2 = 1. Good number.
    • 



Sum of elements added: 2 + 3 + 4 + 5 = 14.
Hence, the answer is 14.




Constraints


    

  • 1 <=nums.size()<= 106
    • 

  • 1 <=nums[i]<= 1000
    • 

  • Index starts from 0
    • 



def sum_good_number(n, nums):
    sum1=0
    for i in range(len(nums)):
        if i%2 == nums[i]%2:
            sum1+=nums[i]
    return sum1

Password Encoding


You are given a password string P of length N. The password needs to be encoded before storing it to the database. The password string can be encoded using the following rules:


    

  • If the k-th character in P is a vowel, change it to the second next character in the alphabetical sequence. For example, the second next character of o is q.
    • 

  • If the k-th character in P is a consonant, change it to the second previous character in the alphabetical sequence. For example, the second previous character of h is f.
    • 



Implement the function encode which will take password string P as argument and return encoded string.



1) The second previous character of b is z.



Note: You need not to write code to handle input and output.

def encode(P) :
    vowels="aeiou"
    p = list(P)
    for i in range(len(p)):
        if p[i] in vowels:
            if p[i]=="y":
                p[i]="a"
            elif p[i]=="z":
                p[i]="b"
            else:
                p[i]=chr(ord(p[i])+2)
        else:
            if p[i]=="b":
                p[i]="z"
            elif p[i]=="a":
                p[i]="y"
            else:
                p[i]=chr(ord(p[i])-2)
    return "".join(p)

Vowel-Consonant Similarity


Two strings are similar if they have the same composition of vowels and consonants, regardless of order or frequency. For example, hello and oellh are similar because they both contain vowels e, o and consonants h, l. However, abc and xyz are not similar.


Given an array of strings strArr of size n, find the number of pairs of strings that are similar based on their vowel-consonant composition.



1. Strings consist of lowercase English letters only.
2. Pairs are considered index-wise, and identical strings at different indices count as separate pairs.
3. A pair of indices (i, j) is the same as (j, i).





Input Format


    

  • The first line contains an integer n, the size of the string array.
    • 

  • The second line contains n space-separated strings representing strArr.

Output Format


    

  • Output an integer representing the number of pairs of strings that are similar based on vowel-consonant composition.

Constraints


    

  • 1 <= n <= 104
    • 

  • a <= strArr[i][j] <= z
    • 





Example

4
hello oellh ab xyz




Here:


    

  • hello and oellh are similar because both have same vowels and consonants.
    • 

  • No other strings share the same composition.

Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.

Constraints


    

  • The number of elements in the array arr is greater than zero.

  • Sub-array size k and the number of skips m are positive integers.
