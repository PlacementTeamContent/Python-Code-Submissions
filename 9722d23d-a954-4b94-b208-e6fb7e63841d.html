Password Encoding


You are given a password string P of length N. The password needs to be encoded before storing it to the database. The password string can be encoded using the following rules:


    

  • If the k-th character in P is a vowel, change it to the second next character in the alphabetical sequence. For example, the second next character of o is q.
    • 

  • If the k-th character in P is a consonant, change it to the second previous character in the alphabetical sequence. For example, the second previous character of h is f.
    • 



Implement the function encode which will take password string P as argument and return encoded string.



1) The second previous character of b is z.



Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing password string.
    • 





Output Format


    

  • Single line containing encoded password string.
    • 





Example 1


Input


abc




Output


cza




Explanation


a is vowel so change to c
b is consonant so change to z
c is consonant so change to a
So, encoded password for abc will be cza


Example 2

iloveniat

kjqtglkcr

def encode(P) :
    alpha = "abcdefghijklmnopqrstuvwxyz"
    vowel = "aeiou"
    new_str = ""
    if len(P) == 1 :
        if P[0] in vowel :
            index = alpha.index(P[0]) + 2
        else :
            index = alpha.index(P[0]) - 2
        return alpha[index]
    if P[0] in vowel :
        index = alpha.index(P[0]) + 2
    else :
        index = alpha.index(P[0]) - 2
    new_str += alpha[index] + encode(P[1:])
    return new_str

Sum of Matching Squares


For a list of integers nums, nums[i] is said to be Matching Square if the square of nums[i] is equal to the square of i.


Implement the function sum_matching_squares which will take n, size of list and nums as argument and return an integer, the sum of all matching square numbers in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of all matching square numbers.
    • 





Example 1


Input


3
0 1 2




Output


3




Explanation


    

  • For i = 0, nums[0] = 0, 00 = 0 and ii = 0. Matching square.
    • 

  • For i = 1, nums[1] = 1, 11 = 1 and ii = 1. Matching square.
    • 

  • For i = 2, nums[2] = 2, 22 = 4 and ii = 4. Matching square.
    • 



Sum of matching squares: 0 + 1 + 2 = 3.




Constraints


    

  • 1 <= nums.size() <= 1000
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def sum_matching_squares(n, nums):
    Sum = 0
    for i in range(len(nums)) :
        if i*i == nums[i]*nums[i] :
            Sum += i 
    return Sum

Given an array of integers E, and an integer H find number of Close Pair elements.


Element W is Close Pair if there exists at least 1 element whose absolute difference with W is less than or equal to H. An element W is Close Pair, if there is another element in the range [W-H, W+H] other than W itself.




Input Format


    

  • First line contains two integers Q and H respectively.
    • 

  • Second line contains Q integers separated by space.
    • 





Output Format


    

  • Print a single integer denoting the total number of close pair elements.
    • 





Constraints


    

  • 1 <=Q<= 1000
    • 

  • 0 <=H<= 1000
    • 

  • 0 <=E[i]<= 1000
    • 





Explanation


Let's take an example:


4 5
10 21 13 14


arr = [10, 21, 13, 14]


For W=10 to be Close Pair, 
there should be atleast one element in range: [10-5, 10+5] = [5, 15].
So, this is a Close Pair. 


For W=21 to be Close Pair, 
there should be atleast one element in range: [21-5, 21+5] = [16, 26].
There is no element in this range.
So, this is a not Close Pair.


Similarily,
13 and 14 are Close Pair 
Hence we have 3 Close Pair.

- A strict subset must have all its elements in the main set and have fewer elements than the main set.



Example

    • 

  • The set([1, 2, 3, 4])is not a strict subset of set([1, 2, 3]).

Alternating Sum


Write a program that traverses an array from left to right, where in each step it examines a sub-array of size k and skips m positions forward after each examination. Your task is to print the alternating sum of values found in each sub-array considered.


Alternating sum of an array can be calculated by adding first value subtracting second and so on.


For example: arr = [a1, a2, a3, a4, a5]
alternating sum of arr will be: (a1-a2+a3-a4-a5)



Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.





Input Format


    

  • The first line contains n size of array.
    • 

  • The second line contains space-separated integers representing the elements of the array arr.
    • 

  • The third line contains an integer k, indicating the size of each sub-array to be considered.
    • 

  • The fourth line contains an integer m, specifying the number of elements to skip after processing each sub-array.
    • 





Output Format


    

  • The output should be a single line consisting of space-separated values, each representing the alternating sum found in the respective sub-array.
    • 





Constraints


    

  • The number of elements in the array arr is greater than zero.
    • 

  • Sub-array size k and the number of skips m are positive integers.
    • 

  • k is less than or equal to the length of the array.
    • 





Example


Let's take a sample input:


8
4 7 3 9 6 1 8 0
3
1


We need to print:


0 13 0 4 13 -7


Explanation


[4 7 3] 9 6 1 8 0 -> alternating sum is 4-7+3 = 0.
4 [7 3 9] 6 1 8 0 -> moving right by 1 place, alternating sum is 7-3+9 = 13.
4 7 [3 9 6] 1 8 0 -> moving right by 1 place, alternating sum is 3-9+6 = 0.
4 7 3 [9 6 1] 8 0 -> moving right by 1 place, alternating sum is 9-6+1 = 4.
4 7 3 9 [6 1 8] 0 -> moving right by 1 place, alternating sum is 6-1+8 = 13.
4 7 3 9 6 [1 8 0] -> moving right by 1 place, alternating sum is 1-8+0 = -7.


#Write your code here
lst_len = int(input())
lst = list(map(int,input().split()))
sum_index = int(input())
skip_index = int(input())
ans_lst = []

for i in range(lst_len) :
    Sum = 0
    new_lst = lst[i+skip_index-1:sum_index+i]
    if len(new_lst) == sum_index :
        for i in range(sum_index) :
            if i % 2 == 0 :
                Sum += new_lst[i]
            else :
                Sum -= new_lst[i]
        ans_lst.append(str(Sum))

print(" ".join(ans_lst))

Toggle Switches


In this task, there are N switches that are initially turned off. You first turn on all the switches, then turn off every second switch. On the third round, you toggle every third switch (turning on if it's off or turning off if it's on).


For the jth round, you toggle every jth switch. For the Nth round, you only toggle the last switch.


Print the number of switches that are on after N rounds.




Input Format


    

  • A single integer N
    • 



Output Format


    

  • A single integer representing the number of switches that are on after N rounds.
    • 





Constraints


    

  • 1 <= N <= 1000
    • 





Sample Input 1


4


Sample Output 1


2


Explanation


Let's take an example:


4


    

  • At first, the four switches are [off, off, off, off].
    • 

  • After the first round, the four switches are [on, on, on, on].
    • 

  • After the second round, the four switches are [on, off, on, off].
    • 

  • After the third round, the four switches are [on, off, off, off].
    • 

  • After the fourth round, the four switches are [on, off, off, on].

    • 



#Write your code here
N = int(input())
new_set = []
num_set = []
for i in range(N) :
    new_set.append("off")
    num_set.append(i)
for i in range(N):
    for j in num_set[i::i+1]:
        if new_set[j] == "off" :
            new_set[j] = "on"
        else :
            new_set[j] = "off"
