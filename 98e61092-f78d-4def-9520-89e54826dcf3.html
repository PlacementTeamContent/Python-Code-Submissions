Message Encryption


You are given a message string M of length N. The message needs to be encrypted before sending it. The message string can be encrypted using the following rules:


    

  • If the k-th character in M is a vowel, change it to the first next character in the alphabetical sequence. For example, the first next character of o is p.
    • 

  • If the k-th character in M is a consonant, change it to the first previous character in the alphabetical sequence. For example, the first previous character of h is g.
    • 



Implement the function encrypt which will take message string M as argument and return encrypted string.


Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing the message string.
    • 





Output Format


    

  • Single line containing the encrypted message string.
    • 





Example 1


Input


cat




Output


bbs




Explanation


c is consonant so change to b

a is vowel so change to b

t is consonant so change to s

So, encrypted message for cat will be bbs.


Example 2


Input


hello




Output


gfkkp




Explanation


h is consonant so change to g

e is vowel so change to f

l is consonant so change to k

l is consonant so change to k

o is vowel so change to p

So, encrypted message for hello will be gfkkp.




Constraints


    

  • 1 <=N<= 106
    • 

  • The message string M contains only lowercase alphabets.
    • 



def encrypt(M):
    e=""
    for i in M:
        if i=="a" or i=="e" or i=="i" or i=="o" or i=="u" :
            a=ord(i)
            e+=chr(a+1)
        else:
            a=ord(i)
            e+=chr(a-1)
    return e

Sum of Matching Squares


For a list of integers nums, nums[i] is said to be Matching Square if the square of nums[i] is equal to the square of i.


Implement the function sum_matching_squares which will take n, size of list and nums as argument and return an integer, the sum of all matching square numbers in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of all matching square numbers.
    • 





Example 1


Input


3
0 1 2




Output


3




Explanation


    

  • For i = 0, nums[0] = 0, 00 = 0 and ii = 0. Matching square.
    • 

  • For i = 1, nums[1] = 1, 11 = 1 and ii = 1. Matching square.
    • 

  • For i = 2, nums[2] = 2, 22 = 4 and ii = 4. Matching square.
    • 



Sum of matching squares: 0 + 1 + 2 = 3.




Constraints


    

  • 1 <= nums.size() <= 1000
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def sum_matching_squares(n, nums):
    r=0
    for i in range(n):
        v=nums[i]**2
        if v==i*i:
            r+=nums[i]
    return r

2500

Distinct Subset Union


You are given a set C and P other sets.


Check if all P sets are strict subsets of the set C then print True, followed by the total number of elements in the union of all P sets. Otherwise, print False, followed by the number of sets that failed the strict subset condition.

- A strict subset must have all its elements in the main set and have fewer elements than the main set.



Example


    

  • The set([1, 2])is a strict subset of set([1, 2, 3]).
    • 

  • The set([1, 2, 3, 4])is not a strict subset of set([1, 2, 3]).

Average of Sub-arrays


Write a program that traverses an array from left to right, where in each step it examines a sub-array of size k and skips m positions forward after each examination. Your task is to print the average of values found in each sub-array considered.



Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.





Input Format


    

  • The first line contains n size of array.
    • 

  • The second line contains space-separated integers representing the elements of the array arr.
    • 

  • The third line contains an integer k, indicating the size of each sub-array to be considered.
    • 

  • The fourth line contains an integer m, specifying the number of elements to skip after processing each sub-array.
    • 





Output Format


    

  • The output should be a single line consisting of space-separated values, each representing the average found in the respective sub-array.
    • 





Constraints


    

  • The number of elements in the array arr is greater than zero.
    • 

  • Sub-array size k and the number of skips m are positive integers.
    • 

  • k is less than or equal to the length of the array.
    • 





Example


Let's take a sample input:


8
4 7 3 9 6 1 8 0
3
1


We need to print:


4.67 6.33 6.00 5.33 5.00 3.00


Explanation


[4 7 3] 9 6 1 8 0 -> average is 4.67.
4 [7 3 9] 6 1 8 0 -> moving right by 1 place, average is 6.33.
4 7 [3 9 6] 1 8 0 -> moving right by 1 place, average is 6.00.
4 7 3 [9 6 1] 8 0 -> moving right by 1 place, average is 5.33.
4 7 3 9 [6 1 8] 0 -> moving right by 1 place, average is 5.00.
4 7 3 9 6 [1 8 0] -> moving right by 1 place, average is 3.00.
