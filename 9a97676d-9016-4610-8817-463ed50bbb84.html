Good Number Sum


For a list of integers nums, nums[i] is said to be Good Number if parity of i and nums[i] will be same.


Implement the function sum_good_number which will take n, size of list and nums as argument and return an integer, sum of good number in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of good numbers.
    • 





Example 1


Input


3
1 2 3




Output


0




Explanation


    

  • For i = 0, nums[0] = 1, 1 % 2 = 1 and i % 2 = 0. Not a good number.
    • 

  • For i = 1, nums[1] = 2, 2 % 2 = 0 and i % 2 = 1. Not a good number.
    • 

  • For i = 2, nums[2] = 3, 3 % 2 = 1 and i % 2 = 0. Not a good number.
    • 



Hence, the answer is 0.


Example 2


Input


4
2 3 4 5




Output


14




Explanation


    

  • For i = 0, values[0] = 2, 2 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 1, values[1] = 3, 3 % 2 = 1 and i % 2 = 1. Good number.
    • 

  • For i = 2, values[2] = 4, 4 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 3, values[3] = 5, 5 % 2 = 1 and i % 2 = 1. Good number.
    • 



Sum of elements added: 2 + 3 + 4 + 5 = 14.
Hence, the answer is 14.




Constraints


    

  • 1 <=nums.size()<= 106
    • 

  • 1 <=nums[i]<= 1000
    • 

  • Index starts from 0
    • 



def sum_good_number(n, nums):
    sum=0 
    for i in range(n):
        if int(nums[i])%2 == int(i)%2:
            sum+=int(nums[i])
    return sum

Total Population Growth


Implement a function total_population to determine the total population of a city after T years, starting at an initial population of P. The population grows at a constant rate of R individuals per year.


Function total_population will take P, R, and T as first, second, and third arguments respectively and return the total population after T years.


Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing 3 space-separated integers representing P, R, and T respectively.
    • 





Output Format


    

  • Single line having 1 integer representing the total population after T years.
    • 





Example 1


Input


1000 200 5




Output


2000




Explanation


Initial population is 1000.
At time 1 year, total population will be 1000+200 = 1200.
At time 2 years, total population will be 1200+200 = 1400.
At time 2 years, total population will be 1400+200 = 1600.
At time 2 years, total population will be 1600+200 = 1800.
At time 5 years, total population will be 1800+200 = 2000.




Constraints


    

  • 0 <= P, R, T < 10000
    • 



def total_population(P, R, T):
    for i in range(T):
        P+=R
    return P

Toggle Lights

For the jth round, you toggle every jth light. For the Nth round, you toggle only the last light.


Determine how many lights are on after N rounds.




Input Format


    

  • A single integer N
    • 



Output Format


    

  • A single integer representing the number of lights that are on after N rounds.
    • 





Constraints


    

  • 1 <= N <= 1000
    •

5


Sample Output 1


2


Explanation


Let's take an example:


5
