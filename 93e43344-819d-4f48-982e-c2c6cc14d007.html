Sum of Matching Squares


For a list of integers nums, nums[i] is said to be Matching Square if the square of nums[i] is equal to the square of i.


Implement the function sum_matching_squares which will take n, size of list and nums as argument and return an integer, the sum of all matching square numbers in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of all matching square numbers.
    • 





Example 1


Input


3
0 1 2



Output


3



Explanation


    

  • For i = 0, nums[0] = 0, 0*0 = 0 and i*i = 0. Matching square.
    • 

  • For i = 1, nums[1] = 1, 1*1 = 1 and i*i = 1. Matching square.
    • 

  • For i = 2, nums[2] = 2, 2*2 = 4 and i*i = 4. Matching square.
    • 



Sum of matching squares: 0 + 1 + 2 = 3.




Constraints


    

  • 1 <= nums.size() <= 1000
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def sum_matching_squares(n, nums):
    r=0
    for i in range(n):
        if i*i == nums[i]*nums[i]:
            r+=nums[i]
    return r      
def main():
    n=int(input())
    m=list(map(int,input().split()))
    print(sum_matching_squares(n,m))

Total Steps Walked


Implement a function total_steps to determine the total steps walked after T days, starting with an initial count of S steps. The number of steps increases by a constant rate of I steps per day.


Function total_steps will take S, I, and T as first, second, and third arguments respectively and return the total steps walked after T days.


Note: You need not write code to handle input and output.




Input Format


    

  • Single line containing 3 space-separated integers representing S, I, and T respectively.
    • 





Output Format


    

  • Single line having 1 integer representing the total steps walked after T days.
    • 





Example 1


Input


1000 500 3



Output


2500



Explanation


Initial steps walked is 1000.
At day 1, total steps will be 1000+500 = 1500.
At day 2, total steps will be 1500+500 = 2000.
At day 3, total steps will be 2500.




Constraints


    

  • 0 <= S, I, T < 20000
    • 



def total_steps(S, I, T):
    for i in range(T):
        S+=I
    return S    
def main():
    n=list(map(int,input().split()))
    print(total_steps(n[0],n[1],n[2]))

Alternating Sum


Write a program that traverses an array from left to right, where in each step it examines a sub-array of size k and skips m positions forward after each examination. Your task is to print the alternating sum of values found in each sub-array considered.


Alternating sum of an array can be calculated by adding first value subtracting second and so on.


For example: arr = [a1, a2, a3, a4, a5]
alternating sum of arr will be: (a1-a2+a3-a4-a5)



Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.





Input Format


    

  • The first line contains n size of array.
    • 

  • The second line contains space-separated integers representing the elements of the array arr.
    • 

  • The third line contains an integer k, indicating the size of each sub-array to be considered.
    • 

  • The fourth line contains an integer m, specifying the number of elements to skip after processing each sub-array.
    • 





Output Format


    

  • The output should be a single line consisting of space-separated values, each representing the alternating sum found in the respective sub-array.
    • 





Constraints


    

  • The number of elements in the array arr is greater than zero.
    • 

  • Sub-array size k and the number of skips m are positive integers.
    • 

  • k is less than or equal to the length of the array.
    • 





Example


Let's take a sample input:


8
4 7 3 9 6 1 8 0
3
1



We need to print:


0 13 0 4 13 -7



Explanation


[4 7 3] 9 6 1 8 0 -> alternating sum is 4-7+3 = 0.
4 [7 3 9] 6 1 8 0 -> moving right by 1 place, alternating sum is 7-3+9 = 13.
4 7 [3 9 6] 1 8 0 -> moving right by 1 place, alternating sum is 3-9+6 = 0.
4 7 3 [9 6 1] 8 0 -> moving right by 1 place, alternating sum is 9-6+1 = 4.
4 7 3 9 [6 1 8] 0 -> moving right by 1 place, alternating sum is 6-1+8 = 13.
4 7 3 9 6 [1 8 0] -> moving right by 1 place, alternating sum is 1-8+0 = -7.

Flipping Bulbs


You have N bulbs that start in the off state. In each round, you apply a specific rule to toggle the bulbs. Initially, you turn on all the bulbs. In the second round, you turn off every second bulb. In the third round, you toggle every third bulb.


For the jth round, you toggle every jth bulb. For the Nth round, you toggle only the last bulb.


Return the count of bulbs that remain on after N rounds.




Input Format


    

  • A single integer N
    • 



Output Format


    

  • A single integer representing the number of bulbs that remain on after N rounds.
    • 





Constraints


    

  • 1 <= N <= 1000
    • 





Sample Input 1


5



Sample Output 1


2



Explanation


Let's take an example:


5



    

  • At first, the five bulbs are [off, off, off, off, off].
    • 

  • After the first round, the five bulbs are [on, on, on, on, on].
    • 

  • After the second round, the five bulbs are [on, off, on, off, on].
    • 

  • After the third round, the five bulbs are [on, off, off, off, on].
    • 

  • After the fourth round, the five bulbs are [on, off, off, on, on].
    • 

  • After the fifth round, the five bulbs are [on, off, off, on, off].
    •

    • 



n=int(input())
m=[]
for i in range(n):
    m+=["off"]
for j in range(1,n+1):
    if j==1:
        for k in range(len(m)):
            m[k]="on"
    else:
        for a in range(n):
            if (a+1)%j==0:
                if m[a]=="on":
                    m[a]="off"
                else:
                    m[a]="on"
