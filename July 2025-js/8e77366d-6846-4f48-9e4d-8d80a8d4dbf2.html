Alternate Merge of Arrays


You should design a factory function to merge two Arrays into a single Array.


The factory function createMerger() should:
 - Accept two arrays  array1 and array2 as an argument.
 - The mergeArrays() method should return combined of both Arrays by alternating their elements. If one Array is shorter, append the remaining elements of the longer Array at the end.


Input Format:


    

  • The first line of input contains an Array representing array1.
    • 

  • The second line of input contains an Array representing array2.
    • 



Output Format:


    

  • A single Array containing the merged elements of array1 and array2. If one Array is shorter, the remaining elements from the longer Array are appended at the end.
    • 



Constraints:


    

  • Both Arrays can have different lengths.
    • 



Sample Input:


10 20 30
40 50 60 70 80


Sample Output:


10 40 20 50 30 60 70 80


Explanation:


The function begins by inserting the first element of array1 (10) and then the first element of array2 (40).


Next, it adds the second element of array1 (20) followed by the second element of array2 (50).


This process repeats until array1 is exhausted (after adding 30).


Then, the remaining elements of array2 (70 and 80) are appended to the result.


Hence, the final output is: 10 40 20 50 30 60 70 80
function createMerger(array1, array2) {
    return{
        mergeArrays: function(){
            const result =[];
            const maxLength= Math.max(array1.length,array2.length);
            for (let i=0; i<maxLength; i++){
                if(i<array1.length){
                    result.push(array1[i]);
                }
                if (i<array2.length){
                    result.push(array2[i]);
                }
            }
            return result;
        }
    }
}

module.exports = createMerger;

Rotate Elements


Implement a constructor function that performs a specific right rotation on a sequence of integers. This rotation affects the sequence k times.


The constructor function SequenceRotator() needs to:
- Accept a sequence arr and a count k.
- Have a method rightRotate() that shifts the sequence to the right k times.


Input Format:


    

  • A sequence arr of integers.
    • 

  • An integer k, which is the number of rightward rotations to apply.
    • 



Constraints:


    

  • The arr contains at least one integer.
    • 

  • All elements in the arr are integers.
    • 

  • 1 ≤ arr.length ≤ 1000
    • 

  • 1 ≤ k ≤ 1000
    • 



Output Format:


    

  • The rightRotate() method returns the sequence post-rotation.
    • 



Sample Input:


6 7 8 9 10
3


Sample Output:


8 9 10 6 7


Explanation:


Given array = [6, 7, 8, 9, 10]


Number of rotation = 3


Now, perform 3 right rotations:


    

  • 

    After 1st rotation → [10, 6, 7, 8, 9]
    

    • 

  • 

    After 2nd rotation → [9, 10, 6, 7, 8]
    

    • 

  • 

    After 3rd rotation → [8, 9, 10, 6, 7]
    

    • 



So after 3 right rotations, the array becomes:
[8, 9, 10, 6, 7]
function SequenceRotator(arr,k) {
  this.arr= arr;
  this.k = k;
  
  this.rightRotate = function(){
      const n = this.arr.length;
      const effectiveRotations = this.k % n;
      const rotated= this.arr.slice(-effectiveRotations).concat(this.arr.slice(0, n - effectiveRotations));
      return rotated;
  }
}

module.exports = SequenceRotator;

Recursive Depth Flattening


Create a function flattenNestedArray() that processes a nested array up to a required depth level.


Your goal is to implement flattenNestedArray(arr, n) which follows these principles:


    

  • The function takes an array arr and an integer n which representing the depth to flatten the array.
    • 

  • It should recursively condense the array up to the specified depth n.
    • 

  • The resulting array, flattened to the desired depth level, should be returned.
    • 



Using the flat() method is not allowed. Resolve the task without it.


Input Format


    

  • The first line is an array arr, which may include nested arrays.
    • 

  • The second line is an integer n, indicating how deep the array should be flattened.
    • 



Constraints


    

  • The array can have multiple nested layers.
    • 

  • The depth n is a positive number determining the flattening levels.
    • 

  • Array may vary in length between 1 and 100.
    • 



Output Format


    

  • Return the array flattened to a depth of n.
    • 



Sample Input:


[[1, [2, [3, 4]]], 5, [6]]
1


Sample Output:


[ 1, [ 2, [3, 4] ], 5, 6 ]


Explanation:


    

  • 

    The input array [[1, [2, [3, 4]]], 5, [6]] is flattened up to depth 1, meaning the first level of nesting is removed while preserving deeper levels.
    

    • 

  • 

    At depth 1, the first nested level [1, [2, [3, 4]]] becomes 1, [2, [3, 4]], and [6] is simply flattened to 6.
    

    • 

  • 

    Final Output: [ 1, [ 2, [3, 4] ], 5, 6 ].
    

    • 

function flattenNestedArray(arr, n) {
    function flatten(array,depth){
        if (depth==0){
            return array;
        }
        let result =[];
        for (let i =0; i<array.length;i++){
            if (Array.isArray(array[i])){
                result.push(...flatten(array[i], depth - 1));
            }
            else{
                result.push(array[i])
                }
        }
        return result;
    }
    return flatten(arr,n);
}
