You are tasked to create a factory function  with alternately combining two arrays into a single array.


The factory function createCombiner() should:


    

  • Accept two arrays  array1 and array2 as an argument.
    • 

  • The combineArray() method should return merged array of both array by alternating their elements. If one array is shorter, append the remaining elements from the longer array at the end.
    • 



Input Format:


    

  • The first line of input contains a array representing array1.
    • 

  • The second line of input contains a array representing array2.
    • 



Output Format:


    

  • A single array containing the combined elements of both array1 and array2 in an alternating fashion. If one array is shorter, the remaining elements from the longer array are appended at the end.
    • 



Constraints:


    

  • Both arrays can have different lengths.
    • 



Sample Input:


22 44
11 33 55


Sample Output:


22 11 44 33 55


Explanation:


The function starts by adding the first element of array1 (22) and then the first element of array2 (11).


Next, it adds the second element of array1 (44) followed by the second element of array2 (33).


Since array1 is now exhausted, the remaining elements of array2 (55) are appended.


Therefore, the final output is: 22 11 44 33 55
function createCombiner(array1, array2) {
    //Write your code here  
    return {
        combineArray : function() {
            let arr = [];
            let max_length = Math.max(array1.length,array2.length);
            for (let i=0;i<max_length;i++) {
                if(i<array1.length) {
                    arr.push(array1[i])
                }
                if(i<array2.length) {
                    arr.push(array2[i])
                }
            }
            return arr;
        }
    }
}

//Ignore the below line and don't modify it
module.exports = createCombiner;

Circular Shift


Create a constructor function to perform a rightward circular shift on an integer array. This shift is executed k times.


The function CircularShifter() must:
- Take an array arr and a rotation count k as input.
- Include a method circularShiftRight() that shifts the array rightward, wrapping around, k times.


Input Format:


    

  • An array arr with integers.
    • 

  • An integer k, the number of times to shift the array.
    • 



Constraints:


    

  • The arr contains at least one element.
    • 

  • All number in the arr are integers.
    • 

  • 1 ≤ arr.length ≤ 1000
    • 

  • 1 ≤ k ≤ 1000
    • 



Output Format:


    

  • The circularShiftRight() method returns the shifted array.
    • 



Sample Input:


3 1 4 1 5
4


Sample Output:


1 4 1 5 3


Explanation:


Given Array = [3, 1, 4, 1, 5]


Number of rotation = 4


Now, perform 4 right rotations:


    

  • 

    After 1st rotation → [5, 3, 1, 4, 1]
    

    • 

  • 

    After 2nd rotation → [1, 5, 3, 1, 4]
    

    • 

  • 

    After 3rd rotation → [4, 1, 5, 3, 1]
    

    • 

  • 

    After 4th rotation → [1, 4, 1, 5, 3]
    

    • 



Final Result = [1, 4, 1, 5, 3]


Each rotation moves the last element to the front and shifts the rest to the right.
function CircularShifter(arr,k) {
  //Write your code here
  this.circularShiftRight = function() {
      let length = arr.length
      if(k===0 || k===length || length === 0) {
          return arr;
      }
      k = k%length;
      let first_part = arr.slice(-k);
      let second_part = arr.slice(0,length-k);
      let result = first_part.concat(second_part);
      return result;
  }
}

module.exports = CircularShifter;
