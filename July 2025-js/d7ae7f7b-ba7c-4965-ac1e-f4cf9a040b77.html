Zigzag Merge


You should implement a  factory function that performs a zigzag merge of two Arrays into one.


The factory function createZigzagMerger() should:


    

  • Accept two arrays  array1 and array2 as an argument.
    • 

  • The zigzagMerge() method should interlace both Arrays by alternating their elements. If one Array is shorter, append the remaining elements of the longer Array at the end.
    • 



Input Format:


    

  • The first line of input contains a Array representing array1.
    • 

  • The second line of input contains a Array representing array2.
    • 



Output Format:


    

  • A single Array containing the elements of both array1 and array2, zigzagged. If one Array is shorter, the remaining elements from the longer Array are added at the end.
    • 



Constraints:


    

  • Both Arrays may have different lengths.
    • 



Sample Input:


7 14 21
8 16 24 32 40


Sample Output:


7 8 14 16 21 24 32 40


Explanation:


The merging begins with the first item of array1 (7) and the first item of array2 (8).


Then, it continues with the next item from array1 (14) and the next item from array2 (16).


This continues until array1 runs out (after adding 21).


The remaining items of array2 (32 and 40) complete the zigzag pattern.


Thus, the final output is: 7 8 14 16 21 24 32 40
function createZigzagMerger(array1, array2) {
    return{
        zigzagMerge:function(){
        let merged=[];
    let maxLength=Math.max(array1.length,array2.length);
    for(let i=0;i<=maxLength;i++){
        if(i<array1.length){
            merged.push(array1[i]);
        }
        if(i<array2.length){
            merged.push(array2[i]);
        }
       
    }
    return merged;
        }
    };

}

Array Reposition


Design a constructor function to carry out a rightward reposition on an array of integers. This repositioning is conducted k times.


The ArrayReposition() function should:
- Accept an array arr and integer k.
- Include a method repositionRight() that repositions the array elements k times to the right.


Input Format:


    

  • An integer array arr.
    • 

  • An integer k, signifying times to shift the Array to the right.
    • 



Constraints:


    

  • The array contains one or more integers.
    • 

  • Only integers are in the array.
    • 

  • 1 ≤ arr.length ≤ 1000
    • 

  • 1 ≤ k ≤ 1000
    • 



Output Format:


    

  • repositionRight() returns the restructured array.
    • 



Sample Input:


13 14 15 16
3


Sample Output:


14 15 16 13


Explanation:


Given array = [13, 14, 15, 16]


Number of rotation = 3


Now, perform 3 right rotations:


    

  • 

    After 1st rotation → [16, 13, 14, 15]
    

    • 

  • 

    After 2nd rotation → [15, 16, 13, 14]
    

    • 

  • 

    After 3rd rotation → [14, 15, 16, 13]
    

    • 



Final Result = [14, 15, 16, 13]


Each right rotation moves the last element to the front and shifts the rest of the elements to the right.
function ArrayReposition(arr,k) {
    this.arr=arr;
    this.k=k;
    this.repositionRight=function(){
        var n=this.arr.length;
        var effective=k%n;
        if(effective===0){
            return arr;
        }
    var rotate=this.arr.slice(n-effective);
    var remain=this.arr.slice(0,n-effective);
    var part=rotate.concat(remain);
    return part;
    
    }

}

Compressed Array Conversion


Create a function flattenArray() that processes a nested array up to a required depth level.


Your goal is to implement flattenArray(arr, n) which follows these principles:


    

  • Accepts an array arr and an integer depth n, designating the levels to flatten.
    • 

  • It should streamline the array based on the depth n specified.
    • 

  • The function returns an array compressed to the given depth n.
    • 



Do not use the flat() method for this solution. Achieve it without it.


Input Format


    

  • The first line is an array arr, which may include nested arrays.
    • 

  • The second line is an integer n, indicating how deep the array should be flattened.
    • 



Constraints


    

  • The array arr can have multiple nested layers.
    • 

  • The depth n is a positive number determining the flattening levels.
    • 

  • Array may vary in length between 1 and 100.
    • 



Output Format


    

  • Return the array flattened to a depth of n.
    • 



Sample Input:


[[1, [2, [3, 4]]], 5, [6]]
2


Sample Output:


[ 1, 2, [ 3, 4 ], 5, 6 ]


Explanation:


    

  • At Depth 1:
    • 

  • Initial sub-array: [ 1, [ 2, [ 3, 4 ] ] ] and [ 6 ].
    • 

  • The first level of nesting is flattened.
    • 

  • The sub-array [ 1, [ 2, [ 3, 4 ] ] ] is now flattened to 1, [ 2, [ 3, 4 ] ].
    • 

  • 

    The sub-array [ 6 ] stays as 6 after flattening.
    

    • 

  • 

    At Depth 2:
    

    • 

  • Flattening the next level: [ 2, [ 3, 4 ] ] gets flattened to 2, [ 3, 4 ]
    • 

  • The array elements like 1,5 and 6 remain unchanged, and only the inner nested elements are affected.
    • 

  • So, the result at depth 2 will be [ 1, 2, [ 3, 4 ], 5, 6 ] .
    • 

function flattenArray(arr, n) {
    if(n===0){
        return arr;
    }
    let result=[];
    for(let elem of arr){
        if(Array.isArray(elem) && n>0){
            result=result.concat(flattenArray(elem,n-1));
        }
        else{
            result.push(elem);
        }
        
    }
    return result;

}
