Blend Arrays


Design a factory function to blend two arrays by alternately mixing their elements into one array.


The factory function createBlender() should:


    

  • Accept two arrays  array1 and array2 as an argument.
    • 

  • The blendArrays() method should combine both arrays by alternating their elements. If one array is shorter, the elements left in the longer array should be appended at the end.
    • 



Input Format:


    

  • The first line of input contains an array representing array1.
    • 

  • The second line of input contains an array representingarray2.
    • 



Output Format:


    

  • Return a  single array containing the elements of both array1 and array2, alternated. If one array is shorter, the remaining elements from the longer array are added at the end.
    • 



Constraints:


    

  • Both arrays may have different lengths.
    • 



Sample Input:


5 15 25
10 20 30 40


Sample Output:


5 10 15 20 25 30 40


Explanation:


Initially, the function adds the first element of array1 (5) and then the first element of array2 (10).


Following this, it adds the second element of array1 (15) and then the second element of array2 (20).


This pattern continues until array1 is done (after adding 25).


The remaining elements of array2 (30 and 40) are then appended to the resulting array.


Thus, the output is 5 10 15 20 25 30 40
function createBlender(array1, array2) {
    return{
        array1,
        array2,
        blendArrays(){
            const newArray=[];     
            const max_len= Math.max(array1.length,array2.length);
            
            for(let i=0;i<max_len;i++){
                if(i<array1.length){
                    newArray.push(array1[i]);
                }
                if(i<array2.length){
                    newArray.push(array2[i])
                }
            }
            return newArray
            
        }
    };
}

module.exports = createBlender;

Nested Array Unwrapping


Create a function unwrapNestedArray() that processes a nested array up to a required depth n level.


Your goal is to implement unwrapNestedArray(arr, n) which follows these principles:


    

  • Accepts an array arr and an integer depth n, designating the levels to flatten.
    • 

  • It should streamline the array based on the depth n specified.
    • 

  • The function returns an array compressed to the given depth n.
    • 



Do not use the flat() method for this solution. Achieve it without it.


Input Format


    

  • The first line is an array arr, which may include nested arrays.
    • 

  • The second line is an integer n, indicating how deep the array should be flattened.
    • 



Constraints


    

  • The array can have multiple nested layers.
    • 

  • The depth n is a positive number determining the flattening levels.
    • 

  • Array may vary in length between 1 and 100.
    • 



Output Format


    

  • Return the array flattened to a depth of n.
    • 



Sample Input:


[[1, [2, [3, 4]]], 5, [6]]
2


Sample Output:


[ 1, 2, [ 3, 4 ], 5, 6 ]


Explanation:


    

  • At Depth 1:
    • 

  • Initial sub-array: [ 1, [ 2, [ 3, 4 ] ] ] and [ 6 ].
    • 

  • The first level of nesting is flattened.
    • 

  • The sub-array [ 1, [ 2, [ 3, 4 ] ] ] is now flattened to 1, [ 2, [ 3, 4 ] ].
    • 

  • 

    The sub-array [ 6 ] stays as 6 after flattening.
    

    • 

  • 

    At Depth 2:
    

    • 

  • Flattening the next level: [ 2, [ 3, 4 ] ] gets flattened to 2, [ 3, 4 ]
    • 

  • The array elements like 1,5 and 6 remain unchanged, and only the inner nested elements are affected.
    • 

  • So, the result at depth 2 will be [ 1, 2, [ 3, 4 ], 5, 6 ] .
    • 

function unwrapNestedArray(arr, n) {
        if(n===0){
            return arr;
        }
        let res=[]
        for(let item of arr){
            if(Array.isArray(item)){
                res.push(...unwrapNestedArray(item,n-1))
            }
            else{
                res.push(item)
            }
        }
            
        return res    
    //Write you code here
}

module.exports = unwrapNestedArray;

Shift Right


Design a constructor function to carry out a right shift on a array of integers, with the shift happening k times.


The constructor ArrayShifter() should:
- Accept a Array arr and an integer k.
- Include a shiftRight() method that executes the rightward shifts k times.


Input Format:


    

  • An Array arr of integers.
    • 

  • An integer k, signifying times to shift the Array to the right.
    • 



Constraints:


    

  • The Array has at least one integer.
    • 

  • All numbers in the Array are integers.
    • 

  • 1 ≤ arr.length ≤ 1000
    • 

  • 1 ≤ k ≤ 1000
    • 



Output Format:


    

  • shiftRight() returns the adjusted Array after shifting.
    • 



Sample Input:


7 0 9
1


Sample Output:


9 7 0


Explanation:


    

  • 

    Right Shift: Each rotation involves taking the last element and moving it to the first position while shifting the others one position to the right.
    

    • 

  • 

    After 1 rotations, the array [7, 0, 9] becomes [9, 7, 0].
    

    • 

function ArrayShifter(arr,k) {
    this.arr=arr;
    this.k=k;
    this.shiftRight=function(){
        const n = this.arr.length
        const rs= this.k 
        for(let i=0;i<rs;i++){
            const last=this.arr.pop()
            this.arr.unshift(last)
        }
        return arr
    }
    
    
}
