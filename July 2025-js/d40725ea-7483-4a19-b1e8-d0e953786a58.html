Merge Arrays


You are tasked with creating a factory function that merges two arrays alternately into one array.


The factory function createArrayMerger should:


    

  • Accept two arrays  array1 and array2 as an argument.
    • 

  • Have a method mergeArrays() that returns the merged arrays by alternating their elements. If one array is shorter, the remaining elements from the longer array should be added at the end.
    • 



Input Format:


    

  • First line of input contains an array represents array1.
    • 

  • Second line of input contains an array represents  array2.
    • 



Output Format:


    

  • A single array containing the elements of both array1 and array2, alternated. If one array is shorter, the remaining elements from the longer array are added at the end.
    • 



Constraints:


    

  • Both arrays can have different lengths.
    • 



Sample Input:


1 3 5
2 4 6 7 8


Sample Output:


1 2 3 4 5 6 7 8


Explanation:


It starts by adding the first element of array1 (1) and then the first element of array2 (2).


Next, it adds the second element of array1 (3) and then the second element of array2 (4).


This continues until array1 is exhausted (after adding 5).


Once array1 is empty, the remaining elements of array2 (7 and 8) are added to the result.


Hence the final output will be 1 2 3 4 5 6 7 8 
function createArrayMerger(array1, array2) {
    return{
        mergeArrays:function(){
            const result=[];
            const maxLength=Math.max(array1.length,array2.length);
            for (let i=0;i<maxLength;i++){
                if (i<array1.length){
                    result.push(array1[i]);
                }
                if (i<array2.length){
                    result.push(array2[i])
                }
            }
            return result
        }
    } ;
}

Array Flattening Based on Depth


You are tasked with creating a function flatten() that flattens a nested array up to a specified depth.


Your task is to implement the flatten() function based on the following logic:


    

  • The function flatten(arr, n) should accept an array arr and an integer n as arguments.
    • 

  • The function should recursively flatten the array up to the given depth n. 
    • 

  • If n is 1, the array should be flattened one level.
    • 

  • If n is 2, it should be flattened to a second level, and so on.
    • 

  • The function should return a flattened array up to the given depth.
    • 



You can't use flat() method try to solve without using flat() method.


Input Format


    

  • The first line contains an array arr, which may contain nested arrays.
    • 

  • The second line contains an integer n, representing the depth to which the array should be flattened.
    • 



Constraints


    

  • The array can contain multiple levels of nested arrays.
    • 

  • The depth n is a positive integer that determines how many levels deep the array should be flattened.
    • 

  • The array length can range from 1 to 100.
    • 



Output Format


    

  • The function should return the array flattened up to the given depth n.
    • 



Sample Input:


[[1, [2, [3, 4]]], 5, [6]]
2


Sample Output:


[ 1, 2, [ 3, 4 ], 5, 6 ]


Explanation:


    

  • At Depth 1:
    • 

  • Initial sub-array: [ 1, [ 2, [ 3, 4 ] ] ] and [ 6 ].
    • 

  • The first level of nesting is flattened.
    • 

  • The sub-array [ 1, [ 2, [ 3, 4 ] ] ] is now flattened to 1, [ 2, [ 3, 4 ] ].
    • 

  • 

    The sub-array [ 6 ] stays as 6 after flattening.
    

    • 

  • 

    At Depth 2:
    

    • 

  • Flattening the next level: [ 2, [ 3, 4 ] ] gets flattened to 2, [ 3, 4 ]
    • 

  • The array elements like 1,5 and 6 remain unchanged, and only the inner nested elements are affected.
    • 

  • So, the result at depth 2 will be [ 1, 2, [ 3, 4 ], 5, 6 ] .

Right Rotation on an Array


You are tasked with creating a constructor function that performs a right rotation on an array of integers. The array is to be rotated k number of times.


The constructor function ArrayProcessor() should:
- Accept an array arr,k as an argument.
- Have a method rightRotate() that performs a right rotation on the array by the specified number of times.


Input Format:


    

  • A single array arr containing integers.
    • 

  • A single integer k, the number of times to rotate the array to the right.
    • 



Constraints:


    

  • The array will contain at least one element.
    • 

  • The array will contain only integers.
    • 

  • 1 ≤ arr.length ≤ 1000
    • 

  • 1 ≤ k ≤ 1000
    • 



Output Format:


    

  • The method rightRotate() should return the rotated array.
    • 



Sample Input:


1 2 3 4 5
2


Sample Output:


4 5 1 2 3


Explanation:


    

  • 

    When you rotate the array [1, 2, 3, 4, 5] for 2 times to the right, each element moves 2 positions forward, and the elements at the end wrap around to the beginning.
    

    • 

  • 

    Here’s the step-by-step breakdown:
    

      

    • 

      Original array: [1, 2, 3, 4, 5]
      

      • 

    • 

      After 1st right rotation: [5, 1, 2, 3, 4]
      

      • 

    • 

      After 2nd right rotation: [4, 5, 1, 2, 3]
      

      • 

    • 

      So, after 2 rotations to the right, the array becomes [4, 5, 1, 2, 3].
      

      • 

    

    • 

function ArrayProcessor(arr,k) {
  this.arr=arr;
  this.k=k;
  this.rightRotate=function(){
      const n=this.arr.length;
      const r = this.k % n;
      return this.arr.slice(-r).concat(this.arr.slice(0,n-r));
  };
}
