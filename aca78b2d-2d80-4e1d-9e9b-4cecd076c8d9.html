Good Number Sum


For a list of integers nums, nums[i] is said to be Good Number if parity of i and nums[i] will be same.


Implement the function sum_good_number which will take n, size of list and nums as argument and return an integer, sum of good number in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of good numbers.
    • 





Example 1


Input


3
1 2 3



Output


0



Explanation


    

  • For i = 0, nums[0] = 1, 1 % 2 = 1 and i % 2 = 0. Not a good number.
    • 

  • For i = 1, nums[1] = 2, 2 % 2 = 0 and i % 2 = 1. Not a good number.
    • 

  • For i = 2, nums[2] = 3, 3 % 2 = 1 and i % 2 = 0. Not a good number.
    • 



Hence, the answer is 0.


Example 2


Input


4
2 3 4 5



Output


14



Explanation


    

  • For i = 0, values[0] = 2, 2 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 1, values[1] = 3, 3 % 2 = 1 and i % 2 = 1. Good number.
    • 

  • For i = 2, values[2] = 4, 4 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 3, values[3] = 5, 5 % 2 = 1 and i % 2 = 1. Good number.
    • 



Sum of elements added: 2 + 3 + 4 + 5 = 14.
Hence, the answer is 14.




Constraints


    

  • 1 <=nums.size()<= 106
    • 

  • 1 <=nums[i]<= 1000
    • 

  • Index starts from 0
    • 



def sum_good_number(n, nums):
    count=0
    for i in range(n):
        if i%2==nums[i]%2:
            count+=nums[i]

    return count

Total Energy Generated


Implement a function total_energy to determine the total energy generated by combining two nuclear chemicals after T seconds, producing initial energy as E units. This energy E increases at a constant rate of R units every second. 


Function total_energy will take E, R and T as first, second and third argument respectively and returns total energy after T seconds.


Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing 3 space-separated integers representing E, R and T respectively.
    • 





Output Format


    

  • Single line having 1 integer representing total unit of energy after T seconds.
    • 





Example 1


Input


5 2 3



Output


11



Explanation


Initial energy is 5 units
at time 1 second total energy will be 5+2 = 7
at time 2 second total energy will be 7+2 = 9
at time 3 second total energy will be 9+2 = 11




Constraints


    

  • 0<=E, R, T<1000
    • 



def total_energy(E, R, T):
    total=E
    for i in range(T):
        total+=R
    return total

Distinct Subset Union


You are given a set C and P other sets.


Check if all P sets are strict subsets of the set C then print True, followed by the total number of elements in the union of all P sets. Otherwise, print False, followed by the number of sets that failed the strict subset condition.

- A strict subset must have all its elements in the main set and have fewer elements than the main set.



Example


    

  • The set([1, 2])is a strict subset of set([1, 2, 3]).
    • 

  • The set([1, 2, 3, 4])is not a strict subset of set([1, 2, 3]).
    • 





Input Format


    

  • The First Line contains an integer representing the number of elements in the main set C.
    • 

  • The second line contains the space-separated elements of the main set C.
    • 

  • The third line contains an integer P that specifies the number of other sets.
    • 

  • The next 2*P lines contain the size of each set followed by space-separated elements of the set.
    • 



Output Format


    

  • Print True or False followed by an integer separated by space.
    • 





Sample Input


8 
1 2 3 4 5 6 7 8 
2 
2 
1 2

3 4 5



Sample Output


True 5



Explanation


Set C = {1,2,3,4,5,6,7,8}
set-1 = {1,2} is a strict subset of set C
set-2 = {3,4,5} is a strict subset of C.


All the sets are strict subsets of set C

Substring Commonality


Two strings are said to be related if one is a substring of the other. For example, abc and abcd are related because abc is a substring of abcd, but xyz and abc are not.


Given an array strArr of size n, determine how many pairs of strings are related through substring commonality.



1. Strings are composed of lowercase English letters only.
2. Index-wise evaluation is required, and identical strings at different indices count as separate pairs.
3. A pair of indices (i, j) is the same as (j, i).





Input Format


    

  • First line: Integer n (number of strings in the array).
    • 

  • Second line: n space-separated strings (strArr).
    • 



Output Format


    

  • Output the number of related pairs of strings.
    • 





Constraints


    

  • 1 <= n <= 104
    • 

  • a <= strArr[i][j] <= z
    • 





Example


Example input:


4
abc ab abcde xyz



Here:


    

  • abc is a substring of abcde resulting in 1 pair.
    • 

  • ab is a substring of abc and abcde resulting in 2 pairs.
    • 



The output is 3.


#Write your code here
n=int(input())
substrings=(input().split()) 
count=0
for i in range(1,n):
    for j in range(1,n):
        if set(substrings[i]).issubset(set(substrings[j])):

Average of Sub-arrays


Write a program that traverses an array from left to right, where in each step it examines a sub-array of size k and skips m positions forward after each examination. Your task is to print the average of values found in each sub-array considered.



Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.





Input Format


    

  • The first line contains n size of array.
    • 

  • The second line contains space-separated integers representing the elements of the array arr.
    • 

  • The third line contains an integer k, indicating the size of each sub-array to be considered.
    • 

  • The fourth line contains an integer m, specifying the number of elements to skip after processing each sub-array.
    • 





Output Format


    

  • The output should be a single line consisting of space-separated values, each representing the average found in the respective sub-array.
    • 





Constraints


    

  • The number of elements in the array arr is greater than zero.
    • 

  • Sub-array size k and the number of skips m are positive integers.
    • 

  • k is less than or equal to the length of the array.
    • 





Example


Let's take a sample input:


8
4 7 3 9 6 1 8 0
3
1



We need to print:


4.67 6.33 6.00 5.33 5.00 3.00



Explanation


[4 7 3] 9 6 1 8 0 -> average is 4.67.
4 [7 3 9] 6 1 8 0 -> moving right by 1 place, average is 6.33.
4 7 [3 9 6] 1 8 0 -> moving right by 1 place, average is 6.00.
4 7 3 [9 6 1] 8 0 -> moving right by 1 place, average is 5.33.
4 7 3 9 [6 1 8] 0 -> moving right by 1 place, average is 5.00.
4 7 3 9 6 [1 8 0] -> moving right by 1 place, average is 3.00.


#Write your code here
n=int(input())
arr=[int(i) for i in input().split()]
k=int(input())  #size of sub array
m=int(input())  #no. of elements to skip
eq=n-k+m

for i in range(eq):
    a=sum(arr[m*i:m*i+k])
    num=round(a/k,2)
    if len(str(num).split(".")[-1])<=1:
        result=str(num)+"0"
    else:
        result=str(num)
    print(result,end=" ")
