Total Energy Generated


Implement a function total_energy to determine the total energy generated by combining two nuclear chemicals after T seconds, producing initial energy as E units. This energy E increases at a constant rate of R units every second. 


Function total_energy will take E, R and T as first, second and third argument respectively and returns total energy after T seconds.


Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing 3 space-separated integers representing E, R and T respectively.
    • 





Output Format


    

  • Single line having 1 integer representing total unit of energy after T seconds.
    • 





Example 1


Input


5 2 3




Output


11




Explanation


Initial energy is 5 units
at time 1 second total energy will be 5+2 = 7
at time 2 second total energy will be 7+2 = 9
at time 3 second total energy will be 9+2 = 11




Constraints


    

  • 0<=E, R, T<1000
    • 



def total_energy(E, R, T):
    c=E 
    for i in range(T):
        c=c+R 
    return c

1) The second previous character of b is z.

def encode(P) :
    alpha="abcdefghijklmnopqrstuvwxyz"
    c=""
    for i in P.lower():
        if i=="a" or i=="e" or i=="i" or i=="o" or i=="u":
            s=(alpha.index(i))+2
            c=c+alpha[s]
        else:
            s=(alpha.index(i))-2
            c=c+alpha[s]    
    return c

Count of Divisible Indices


For a list of integers nums, nums[i] is said to be Index Divisible if i is divisible by nums[i].


Implement the function count_divisible_indices which will take n, size of list and nums as argument and return an integer, count of index-divisible numbers in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the count of index-divisible numbers.
    • 





Example 1


Input


4
1 2 1 4




Output


2




Explanation


    

  • For i = 0, nums[0] = 1, 0 % 1 = 0. Index divisible.
    • 

  • For i = 1, nums[1] = 2, 1 % 2 != 0. Not index divisible.
    • 

  • For i = 2, nums[2] = 1, 2 % 1 = 0. Index divisible.
    • 

  • For i = 3, nums[3] = 4, 3 % 4 != 3. Not index divisible.
    • 



Total index divisible numbers: 2.




Constraints


    

  • 1 <= nums.size() <= 106
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def count_divisible_indices(n, nums):
    c=0
    for i in range(n):
        if i % nums[i]==0:
            c=c+1 
    return c

Write a program that traverses an array from left to right, where in each step it examines a sub-array of size k and skips m positions forward after each examination. Your task is to print the sum of values found in each sub-array considered.



Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.

Constraints
