Average of Multiples


you are tasked with calculating the average of a series of numbers. Specifically, you need to compute the average of the first P multiples of a given integer N. The integer N represents a base number, and P represents how many multiples of N we need to consider.


Your task is to complete the given function that receives  N and  P, and calculates the average of the first P multiples of N.



- Ensure that you discard any decimal part of the result when calculating the average.





Input Format


    

  • The first line contains an integer N, which represents the base number whose multiples will be considered.
    • 

  • The second line contains an integer P, which represents the number of multiples of N to include in the calculation.
    • 



Output Format


    

  • The function should print an integer representing the average of the first P multiples of N.
    • 





Constraints


    

  • 

    1 ≤ N ≤ 105
    

    • 

  • 

    1 ≤ P ≤ 105
    

    • 





Example


Sample Input:


6
5


Sample Output:


18


Explanation:


    

  • In this case, the first 5 multiples of 6 are: 6, 12, 18, 24, 30.
    • 

  • The sum of these multiples is: 6 + 12 + 18 + 24 + 30 = 90.
    • 

  • The average is: 90 / 5 = 18.
    • 



#include <iostream>
using namespace std;

class solution {
public:
    void averageOfMultiples(int N, int P) {
        if (P <= 0) {
            cout << 0 << endl;
            return;
        }

        int sum = 0;
        for (int i = 1; i <= P; ++i) {
            sum += N * i;
        }
        cout << sum / P << endl;
    }
};

Prime Number Range


You are tasked with developing an application to evaluate whether numbers in a given range are prime numbers or not.

The function should evaluate every number in the range (inclusive) to determine whether it is prime or not prime. For each number in the range, print "[number] is prime." if the number is prime, and "[number] is not prime." if it is not.




    

  • A prime number is a number greater than 1 that is divisible only by 1 and itself.

    • 





Input Format


    

  • Two space-separated integers, start and end. 
    • 



Output Format


    

  • A range of lines from start to end, where each line is a string
    • 





Constraints


    

  • 1 ≤ start ≤ end ≤ 106
    • 





Example


Input


5 10


Output


5 is prime.
6 is not prime.
7 is prime.
8 is not prime.
9 is not prime.
10 is not prime.


Explanation


    

  1. Evaluating Primes:  
    2. 

  2. The numbers in the range are: 5, 6, 7, 8, 9, 10.
    3. 

  3. 5 is prime.
    4. 

  4. 6 is not prime.
    5. 

  5. 7 is prime.
    6. 

  6. 8 is not prime.
    7. 

  7. 9 is not prime.
    8. 

  8. 10 is not prime.
    9.

5 is prime.

6 is not prime.

7 is prime.

8 is not prime.

9 is not prime.

10 is not prime.


#include <iostream>
#include <cmath>
using namespace std;

class Solution {
public:
    void evaluatePrimes(int start, int end) {
        // Implement your logic here
        for (int num = start; num <= end; ++num) {
            if (num <= 1) {
                cout << num << " is not prime." << endl;
                continue;
            }
            bool isPrime = true;
            for (int i = 2; i <= sqrt(num); ++i) {
                if (num % i == 0) {
                    isPrime = false;
                    break;
                }
            }
            if (isPrime) {
                cout << num << " is prime." << endl;
            } else {
                cout << num << " is not prime." << endl;
            }
        }
    }
};

Data Compression for File Storage


You are working on a project where you need to compress text data in order to save storage space. 
To do compression you will replace consecutive identical characters with the character followed by the number of times it appears. 


Implement the given function evaluateCompression(string s) that takes a string s representing file data and print its compressed version. 


Note: If the compressed string is not shorter than the original string, print the original string to avoid unnecessary storage overhead.




Input Format


    

  • Single line containing string s representing the data in the file.
    • 



Output Format


    

  • Single line having compressed string. 
    • 





Constraints


    

  • 1 ≤s.size()≤ 105.
    • 

  • s contains only english alphabets.
    • 





Example


Input


aaabbccaa




Output


a3b2c2a2




Explanation


    

  • The string "aaabbccaa" is compressed into "a3b2c2a2":
    • 

  • "a3" means 'a' appears 3 times consecutively,
    • 

  • "b2" means 'b' appears 2 times consecutively,
    • 

  • "c2" means 'c' appears 2 times consecutively,
    • 

  • "a2" means 'a' appears 2 times consecutively.
    • 



#include <iostream>
#include <string>
#include <sstream>
using namespace std;

class Solution {
public:
    void evaluateCompression(string s) {
        if (s.empty()) {
            cout << s << endl;
            return;
        }

        stringstream compressed;
        char currentChar = s[0];
        int count = 1;

        for (int i = 1; i < s.length(); ++i) {
            if (s[i] == currentChar) {
                count++;
            } else {
                compressed << currentChar;
                compressed << count;
                currentChar = s[i];
                count = 1;
            }
        }
        compressed << currentChar;
        compressed << count;

        string compressedString = compressed.str();
        if (compressedString.length() < s.length()) {
            cout << compressedString << endl;
        } else {
            cout << s << endl;
        }
    }
};

Array Equilibrium Index


You are given an array of integers. Your task is to find the equilibrium index of the array. An equilibrium index is an index where the sum of elements to its left equals the sum of elements to its right. If no such index exists, return -1.


Equilibrium Index Criteria:
1. The sum of elements before the index (left sum) should equal the sum of elements after the index (right sum).
2. If multiple equilibrium indices exist, return the one with the smallest index.
3. The sum of elements to the left of the first index is considered 0
4. The sum of elements to the right of the last index is considered 0.


Complete the given function findEquilibriumIndex that takes an array and its size, then returns the equilibrium index or -1.


Note: Consider 0 based indexing




Input Format


    

  • The first line contains a single integer N, the size of the array.
    • 

  • The second line contains N space-separated integers representing the array elements.
    • 



Output Format


    

  • A single integer representing the equilibrium index or -1.
    • 





Constraints


    

  • 1 ≤N≤ 105
    • 

  • -109 ≤array[i]≤ 109
    • 





Example


Input


5
1 2 3 2 1




Output


2




Explanation


    

  • At index 2:
    • 

  • Left sum = 1 + 2 = 3
    • 

  • Right sum = 2 + 1 = 3
    • 

  • Since left sum equals right sum, index 2 is an equilibrium index.
    • 



#include <iostream>
using namespace std;

class Solution {
public:
    int findEquilibriumIndex(int N, int arr[]) {
        // Implement your logic here
        if (N == 0) {
            return -1;
        }

        long long totalSum = 0;
        for (int i = 0; i < N; ++i) {
            totalSum += arr[i];
        }

        long long leftSum = 0;
        for (int i = 0; i < N; ++i) {
            long long rightSum = totalSum - leftSum - arr[i];
            if (leftSum == rightSum) {
                return i;
            }
            leftSum += arr[i];
        }
        return -1;
    }
};

Next Weaker Number


You are required to complete the function next_weaker_number that takes an array of integers nums and its size n as input arguments, such that it will calculate and print Next Weaker Number (NWN) for each number in nums.


The Next Weaker Number (NWN) for a number nums[i] is defined as the first number after nums[i] that is smaller than nums[i]. If no such number exists, the result for nums[i] should be -1.




Input Format


    

  • The first line consists of an integer n, representing the number of integers in nums.
    • 

  • The second line contains n space-separated integers representing the array nums.
    • 



Output Format


    

  • Print n space-separated integers representing the Next Weaker Number (NWN) for each of the numbers in nums
    • 





Constraints


    

  • 1 ≤n≤ 1000
    • 

  • 0 ≤nums[i]≤ 1000
    • 





Example


Input


5
9 4 5 2 1




Output


4 2 2 1 -1




Explanation


For the given array:


    

  • The Next Weaker Number for 9 is 4 (since 4 < 9 and 4 comes after 9).
    • 

  • The Next Weaker Number for 4 is 2 (since 2 < 4 and 2 comes after 4).
    • 

  • The Next Weaker Number for 5 is 2 (since 2 < 5 and 2 comes after 5).
    • 

  • The Next Weaker Number for 2 is 1 (since 1 < 2 and 1 comes after 2).
    • 

  • There is no Next Weaker Number for 1, so it is considered -1.
    • 



Thus, the output is: 4 2 2 1 -1


#include <iostream>
using namespace std;

class Solution {
public:
    void next_weaker_number(int n, int nums[]) {
        // Implement your logic here
        for (int i = 0; i < n; ++i) {
            int nwn = -1;
            for (int j = i + 1; j < n; ++j) {
                if (nums[j] < nums[i]) {
                    nwn = nums[j];
                    break;
                }
            }
            cout << nwn << " ";
        }
        cout << endl;
    }
};

Warehouse Inventory


You are working with an inventory management system for a warehouse. The warehouse keeps track of products in a 2D grid layout, where each cell contains the quantity of a specific product at a particular location. The system needs to rotate the entire inventory grid 90 degrees clockwise, as part of optimizing the warehouse's storage system.


Your task is to complete the given function rotateInventory that:


    

  • Takes the size of the matrix, N, and the inventory matrix (2D array) as input.
    • 

  • Rotates the matrix 90 degrees in a clockwise direction.
    • 

  • Prints the rotated matrix, which represents the updated warehouse layout.
    • 




- The input matrix will always be square (i.e., the number of rows will be equal to the number of columns).






Input Format


    

  • First line represents integer N (size of the square matrix, representing the layout of the warehouse).
    • 

  • Next N lines each  contains N space-separated integers representing the quantity of products in the corresponding grid cells of the warehouse.
    • 



Output Format


    

  • N lines each  contains N space-separated integers, which is rotated matrix  after a 90-degree clockwise rotation, representing the updated product layout in the warehouse.
    • 





Constraints


    

  • 

    1 ≤ N ≤ 100
    

    • 

  • 

    -109 ≤ matrix[i][j] ≤ 109
    

    • 





Example


Input


3
1 2 3
4 5 6
7 8 9


Output


7 4 1
8 5 2
9 6 3


Explanation


In this test case, we are given a 3x3 matrix. We need to rotate the matrix 90 degrees clockwise.


    

  • The original matrix is:
    • 



1 2 3
4 5 6
7 8 9


To rotate the matrix 90 degrees clockwise, follow these steps:


1. First Row Becomes the Last Column:


    

  • The first row [1, 2, 3] becomes the last column in the rotated matrix:  
    • 



2. Second Row Becomes the Second Last Column:


    

  • The second row [4, 5, 6] becomes the second last column in the rotated matrix:  
    • 



3. Third Row Becomes the First Column:


    

  • The third row [7, 8, 9] becomes the first column in the rotated matrix:  
    • 



Resulting Matrix after Rotation:


After rotating 90 degrees clockwise, the new matrix looks like this:


7 4 1
8 5 2
9 6 3


#include <iostream>
using namespace std;

class Solution {
public:
    void rotateInventory(int N, int arr[][100]) {
        // Implement your logic here
        int rotated[N][100];

    
        for (int i = 0; i < N; ++i) {
            for (int j = 0; j < N; ++j) {
                rotated[j][N - 1 - i] = arr[i][j];
            }
        }

        
        for (int i = 0; i < N; ++i) {
            for (int j = 0; j < N; ++j) {
                cout << rotated[i][j] << " ";
            }
            cout << endl;
        }
    }
};
