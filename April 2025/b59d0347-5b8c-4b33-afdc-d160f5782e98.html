The app will display the smallest string to the user for various purposes such as decision-making or comparison.

    • 



Output Format


    

  • The function should output a single string that is the smallest of the three strings.

apple banana cherry

apple

  1. Comparing the Strings:  
    2.

public:

        int sum2 = str2.length();

        cout << str2;

        cout << str1;
    }
    else{
        cout << str3;
    }
    }

Equal Split of Water Tanks


You are managing a set of water tanks in a small town, and the tanks are arranged in a linear fashion. Each tank is represented by an integer, where each number indicates the amount of water it holds. Your task is to find a point where the total water in the tanks before that point is equal to the total water in the tanks after it. If no such point exists, return -1.


Equilibrium Point Criteria:
1. The total water before the index (left sum) should equal the total water after the index (right sum).
2. If multiple equilibrium points exist, return the one with the smallest index.
3. The water before the first tank is considered 0.
4. The water after the last tank is considered 0.


Complete the given function findEquilibriumPoint that takes an array representing the water levels in the tanks and its size, then returns the equilibrium point or -1.


Note: Consider 0 based indexing




Input Format


    

  • The first line contains a single integer N, the size of the array.
    • 

  • The second line contains N space-separated integers representing the amount of water in each tank.
    • 



Output Format


    

  • A single integer representing the equilibrium point or -1.
    • 





Constraints


    

  • 1 ≤ N ≤ 105
    • 

  • -109 ≤ tanks[i] ≤ 109
    • 





Example


Input


5
1 2 3 2 1




Output


2




Explanation


    

  • At index 2:
    • 

  • Left sum = 1 + 2 = 3
    • 

  • Right sum = 2 + 1 = 3
    • 

  • Since left sum equals right sum, index 2 is the equilibrium point.
    • 



#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    int findEquilibriumPoint(int N, int arr[]) {
        
        int sum = 0;
        for(int i=0;i<N;i++){
            sum+=arr[i];
        }
        int leftsum = 0;
        for (int i=0;i<N;i++){
            sum -= arr[i];
            if (leftsum == sum) {
                return i;
            }
            leftsum += arr[i];
        }
        return -1;
        
    }
};

Flight Simulation Rotation


You are working on a flight simulation system where the data is represented as a square matrix. Each element in the matrix represents a specific parameter of the flight at a certain time, such as speed, altitude, or fuel consumption. The system needs to rotate this matrix by 90 degrees clockwise to simulate the movement of the flight over time.


Your task is to complete the given function rotateFlightData that:


    

  • Takes the size of the matrix, N, and the flight simulation data matrix (2D array) as input.
    • 

  • Rotates the matrix 90 degrees in a clockwise direction.
    • 

  • Prints the rotated matrix, which represents the updated flight simulation data.
    • 




- The input matrix will always be square (i.e., the number of rows will be equal to the number of columns).





Input Format:


    

  • First line represents integer N (size of the square matrix, representing the flight simulation data).
    • 

  • Next N lines represent each line containing N space-separated integers representing the flight parameters in the corresponding simulation matrix.
    • 



Output Format:


    

  • The rotated matrix printed after a 90-degree clockwise rotation.
    • 





Constraints:


    

  • 1 ≤ N ≤ 100
    • 

  • -109 ≤ matrix[i][j] ≤ 109
    • 





Example:


Input:


3
1 2 3
4 5 6
7 8 9


Output:


7 4 1
8 5 2
9 6 3


Explanation:


In this case, we are given a 3x3 flight simulation matrix. We need to rotate the matrix 90 degrees clockwise.


    

  • The original matrix is:
    • 



1 2 3
  4 5 6
  7 8 9


To rotate the matrix 90 degrees clockwise, follow these steps:


1. First Row Becomes the Last Column:


    

  • The first row [1, 2, 3] becomes the last column in the rotated matrix:
    
     The elements 1, 4, 7 move to the last column.
    • 



2. Second Row Becomes the Second Last Column:


    

  • The second row [4, 5, 6] becomes the second last column in the rotated matrix:
    
     The elements 2, 5, 8 move to the second column.
    • 



3. Third Row Becomes the First Column:


    

  • The third row [7, 8, 9] becomes the first column in the rotated matrix:
    
     The elements 3, 6, 9 move to the first column.
    • 



Resulting Matrix after Rotation:


7 4 1
8 5 2
9 6 3


#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    void rotateFlightData(int N, int arr[][100]) {
        for (int i=0;i<N;i++){
            for (int j=i;j<N;j++){
                swap(arr[i][j],arr[j][i]);
            }
        }
        for (int i=0;i<N;i++){
            reverse(arr[i],arr[i]+N);
        }
        for (int i=0;i<N;i++){
            for(int j=0;j<N;j++){
                cout << arr[i][j]<<" ";
            }
            cout << endl;
        }
    }
};

Rainfall Data Analysis


You are required to complete the function next_lower_rainfall that takes an array of integers rainfall and its size n as input arguments. This function will calculate and print Next Lower Rainfall (NLR) for each rainfall reading in rainfall.


The Next Lower Rainfall (NLR) for a reading rainfall[i] is defined as the first rainfall value after rainfall[i] that is smaller than rainfall[i]. If no such value exists, the result for rainfall[i] should be -1.




Input Format


    

  • The first line consists of an integer n, representing the number of rainfall readings.
    • 

  • The second line contains n space-separated integers representing the array rainfall.
    • 



Output Format


    

  • Print n space-separated integers representing the Next Lower Rainfall (NLR) for each reading in rainfall.
    • 





Constraints


    

  • 1 ≤ n ≤ 1000
    • 

  • 0 ≤ rainfall[i] ≤ 1000
    • 





Example


Input


5
40 30 35 25 15




Output


30 25 25 15 -1




Explanation


For the given array:


    

  • The Next Lower Rainfall for 40 is 30 (since 30 < 40 and 30 comes after 40).
    • 

  • The Next Lower Rainfall for 30 is 25 (since 25 < 30 and 25 comes after 30).
    • 

  • The Next Lower Rainfall for 35 is 25 (since 25 < 35 and 25 comes after 35).
    • 

  • The Next Lower Rainfall for 25 is 15 (since 15 < 25 and 15 comes after 25).
    • 

  • There is no Next Lower Rainfall for 15, so it is considered -1.
    • 



Thus, the output is: 30 25 25 15 -1


#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    void next_lower_rainfall(int n, int rainfall[]) {
        
        vector<int> result(n,-1);
        stack<int>st;
        for (int i=0;i<n;i++){
            while(!st.empty()&&rainfall[i]<rainfall[st.top()]){
                result[st.top()] = rainfall[i];
                st.pop();
            }
            st.push(i);
        }
        for(int i=0;i<n;i++) {
            cout << result[i];
            if (i!=(n-1))cout << " ";
        }
        cout << endl;
        
    }
};
