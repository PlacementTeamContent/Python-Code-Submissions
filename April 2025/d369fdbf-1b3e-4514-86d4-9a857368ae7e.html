Your task is to find the point in time where the total expenditure/income before that point equals the total expenditure/income after that point. If no such point exists, return -1.


Equilibrium Point Criteria:
1. The total expenditure/income before the point (left sum) should equal the total expenditure/income after the point (right sum).
2. If multiple equilibrium points exist, return the one with the smallest index.
3. The total expenditure/income before the first point is considered 0.
4. The total expenditure/income after the last point is considered 0.


Complete the given function findEquilibriumPoint that takes an array representing the budget and its size, then returns the equilibrium point or -1.


Note: Consider 0 based indexing




Input Format


    

  • The first line contains a single integer N, the size of the budget array.
    • 

  • The second line contains N space-separated integers representing the expenditure/income at each point.
    • 



Output Format


    

  • A single integer representing the equilibrium point or -1.
    • 





Constraints


    

  • 1 ≤ N ≤ 105
    • 

  • -109 ≤ budget[i] ≤ 109
    • 





Example


Input


5
3 1 2 1 3




Output


2




Explanation


    

  • At index 2:
    • 

  • Left sum = 3 + 1 = 4
    • 

  • Right sum = 1 + 3 = 4
    • 

  • Since left sum equals right sum, index 2 is the equilibrium point.
    • 



#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    int findEquilibriumPoint(int N, int arr[]) {
        long long totalSum =0;
        for (int i=0; i<N; i++){
            totalSum += arr[i];
        }
        long long leftSum =0;
        for(int i=0; i<N; i++){
            totalSum -= arr[i];
            
        if (leftSum == totalSum){
            return i;
        }
            leftSum +=arr[i];
        }
        return -1;
    }
};

Product of Prime Numbers


You are tasked with developing an application to calculate the product of all prime numbers in a given range. The user will input two integers: a starting number and an ending number, and the program should compute the product of all prime numbers within the range (inclusive).


Complete the given function productPrimes, that receives starting number and ending number and prints the product of all prime numbers in that range.




    

  • A prime number is a number greater than 1 that is divisible only by 1 and itself.

    • 





Input Format


    

  • Two space-separated integers start and end, representing the range of numbers to evaluate.
    • 



Output Format


    

  • A single line integer, which is the product of all prime numbers in the given range.
    • 





Constraints


1 ≤ start ≤ end ≤ 106




Example


Input


5 10


Output


35


Explanation


    

  1. Evaluating the Numbers:  
    2. 

  2. The numbers in the range are 5, 6, 7, 8, 9, 10.
    3. 

  3. The prime numbers in this range are 5, 7.
    4. 

  4. The product of these prime numbers is 5 * 7 = 35.
    5. 



Thus, the output is: 35.


#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    bool isPrime(int n){
        if (n<=1) return false;
        for (int i =2; i*i <=n; i++){
            if (n%i==0) return false;
        }
        return true;
    }
    void productPrimes(int start, int end) {
        long long product = 1;
        bool foundPrime = false;
        for (int i = start; i<=end;i++){
            if(isPrime(i)){
                product *=i;
                foundPrime = true;
            }
        }
        if (foundPrime)
            cout<<product<<endl;
        else
            cout<<0<<endl;
    }
};

Simplifying Data Entries in an E-commerce System


You are designing a system for an e-commerce website where users make repeated purchases of the same item. To optimize storage, consecutive identical items are logged as the item followed by the number of times it was purchased consecutively.


Implement the function evaluateCompression(string s) that takes a string s representing a list of items purchased in sequence and print the compressed version.


Note: If the compressed string is not shorter than the original, print the original string to avoid wasting storage.




Input Format


    

  • A single line containing string s, which represents the sequence of items purchased.
    • 



Output Format


    

  • A single line containing the compressed version of the purchased items.
    • 





Constraints


    

  • 1 ≤ s.size() ≤ 105.
    • 

  • s contains only lowercase English letters.
    • 





Example


Input


abbbcccaaaa




Output


a1b3c3a4




Explanation


    

  • The string "abbbcccaaaa" is compressed into "a1b3c3a4":
    • 

  • "a1" means 'a' is purchased once,
    • 

  • "b3" means 'b' is purchased 3 times consecutively,
    • 

  • "c3" means 'c' is purchased 3 times consecutively,
    • 

  • "a4" means 'a' is purchased 4 times consecutively.
    • 



#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    
    void evaluateCompression(string s) {
        string v="";
       int count =1;
       for (int i =1; i<=s.length();i++){
           if(i<s.length() && s[i] ==s[i-1]){
               count++;
           }
           else{
               v +=s[i-1];
               if(count>1) v+=to_string(count);
               count=1;
           }
       }
       cout <<v<<endl;
    }
};

Robot Path Planning


You are working on a robot navigation system where the robot's movements are tracked in a 2D matrix. Each element in the matrix represents the position of the robot in a grid-based environment, such as obstacles, free space, or waypoints. As part of path planning, the robot needs to rotate its view of the environment by 90 degrees clockwise to update its navigation strategy based on a new orientation.


Your task is to complete the given function rotateRobotPathData that:


    

  • Takes the size of the matrix, N, and the robot path matrix (2D array) as input.
    • 

  • Rotates the matrix 90 degrees clockwise to simulate a change in the robot's orientation.
    • 

  • Prints the rotated matrix, which represents the updated view for navigation and path planning.
    • 




- The input matrix will always be square (i.e., the number of rows will be equal to the number of columns).






Input Format:


    

  • First line represents integer N (size of the square matrix, representing the robot's navigation grid).
    • 

  • Next N lines represent each line containing N space-separated integers representing the robot's environment or path parameters (such as obstacles, waypoints, or free space).
    • 



Output Format:


    

  • The rotated matrix printed after a 90-degree clockwise rotation, which simulates the robot's new orientation.
    • 





Constraints:


    

  • 1 ≤ N ≤ 100
    • 

  • -109 ≤ matrix[i][j] ≤ 109
    • 





Example:


Input:


3
1 2 3
4 5 6
7 8 9


Output:


7 4 1
8 5 2
9 6 3


Explanation:


In this case, we are given a 3x3 robot path matrix. We need to rotate the matrix 90 degrees clockwise to simulate the robot's updated view after adjusting its orientation.


    

  • The original matrix is:
    • 



1 2 3
  4 5 6
  7 8 9


To rotate the matrix 90 degrees clockwise, follow these steps:


1. First Row Becomes the Last Column:


    

  • The first row [1, 2, 3] becomes the last column in the rotated matrix:
    
     The elements 1, 4, 7 move to the last column.
    • 



2. Second Row Becomes the Second Last Column:


    

  • The second row [4, 5, 6] becomes the second last column in the rotated matrix:
    
     The elements 2, 5, 8 move to the second column.
    • 



3. Third Row Becomes the First Column:


    

  • The third row [7, 8, 9] becomes the first column in the rotated matrix:
    
     The elements 3, 6, 9 move to the first column.
    • 



Resulting Matrix after Rotation:


7 4 1
8 5 2
9 6 3


#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    void rotateRobotPathData(int N, int arr[][100]) {
        // Implement your logic here
        for (int i=0; i<N; i++){
            for (int j=i; j<N; j++){
                swap(arr[i][j], arr[j][i]);
            }
        }
        for(int i=0; i<N; i++){
            reverse(arr[i] , arr[i]+N);
        }
        for (int i=0; i<N; i++){
            for (int j = 0; j < N ; j++){
                cout << arr[i][j] << " ";
            }
            cout << endl;
        }
    }
};

Temperature Monitoring System


You are tasked with completing the function next_lower_temperature that takes an array of integers temperatures and its size n as input arguments. This function will calculate and print the Next Lower Temperature (NLT) for each temperature in temperatures.


The Next Lower Temperature (NLT) for a temperature temperatures[i] is defined as the first temperature after temperatures[i] that is smaller than temperatures[i]. If no such temperature exists, the result for temperatures[i] should be -1.




Input Format


    

  • The first line consists of an integer n, representing the number of temperature readings.
    • 

  • The second line contains n space-separated integers representing the array temperatures.
    • 



Output Format


    

  • Print n space-separated integers representing the Next Lower Temperature (NLT) for each temperature in temperatures.
    • 





Constraints


    

  • 1 ≤ n ≤ 1000
    • 

  • 0 ≤ temperatures[i] ≤ 1000
    • 





Example


Input


5
34 28 30 25 18




Output


28 25 25 18 -1




Explanation


For the given array:


    

  • The Next Lower Temperature for 34 is 28 (since 28 < 34 and 28 comes after 34).
    • 

  • The Next Lower Temperature for 28 is 25 (since 25 < 28 and 25 comes after 28).
    • 

  • The Next Lower Temperature for 30 is 25 (since 25 < 30 and 25 comes after 30).
    • 

  • The Next Lower Temperature for 25 is 18 (since 18 < 25 and 18 comes after 25).
    • 

  • There is no Next Lower Temperature for 18, so it is considered -1.
    • 



Thus, the output is: 28 25 25 18 -1


#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    void next_lower_temperature(int n, int temperatures[]) {
        stack<int>st;
        vector<int>result(n,-1);
        for (int i = n-1;i>=0; i--){
            while (!st.empty()&&st.top()>=temperatures[i]){
                st.pop();
            }
            if(!st.empty()){
                result[i]=st.top();
            }
            st.push(temperatures[i]);
        }
        for(int i=0;i<n;i++){
            cout<<result[i]<<" ";
        }
        cout<<endl;
    }
};
