Longest Common Prefix


Given an array of strings, write a function to find the longest common prefix string amongst them.


    

  • If there is no common prefix, return "NO PREFIX".
    • 

  • If all strings share a common prefix, return that prefix.
    • 





Input Format:


    

  • The first line contains the integer n (the number of strings).
    • 

  • The second line contains n strings separated by space.
    • 





Output Format:


    

  • A string representing the longest common prefix.
    • 

  • If there is no common prefix, return "NO PREFIX".
    • 





Constraints:


    

  • 1 <= n <= 200  
    • 

  • 0 <= length of each string <= 200
    • 

  • Strings consist of only lowercase English letters if non-empty.
    • 





Sample Input:


3
flower flow flight


Sample Output:


fl


Explanation:


    

  • The longest common prefix in the strings ["flower", "flow", "flight"] is "fl", because all three strings start with "fl".
    • 



import java.util.Arrays;

class Solution {
    public static String longestCommonPrefix(String[] strs) {
        if(strs==null || strs.length==0){
            return "NO PREFIX";
        }
        String prefix=strs[0];
        for (int i=1;i<strs.length;i++ ){ 
        while(!strs[i].startsWith(prefix)){
        prefix=prefix.substring(0,prefix.length()-1);
        if(prefix.isEmpty()){
            return "NO PREFIX";
        }
        }
        }
        return prefix;
    }
}

Count Excel Sheet Cells


Given a range in an Excel sheet, determine the number of cells within that range. The range is provided in the format start:end, where start and end represent the top-left and bottom-right cells respectively.




Input Format


    

  • A single line representing the range in the format start:end.
    • 



Output Format


    

  • A single integer representing the number of cells in the given range.
    • 





Constraints


    

  • The cells are within a valid Excel range (from A1 to XFD1048576).
    • 





Example 1


Input


A1:C8


Output


24


Explanation


    

  • The range is from A1 to C8. This range includes 3 columns (A, B, C) and 8 rows (1 to 8). 
    • 

  • Total cells = 3 * 8 = 24.
    • 



import java.util.Scanner;

public class main {
    public static void main(String[] args) {
        // Make your logic here
        Scanner sc=new Scanner(System.in);
        String range=sc.nextLine();
        String[] parts =range.split(":");
        String start=parts[0];
        String end=parts[1];
        int startCol=colToNumber(start.replaceAll("[0-9]",""));
        int endCol=colToNumber(end.replaceAll("[0-9]",""));
        int startRow=Integer.parseInt(start.replaceAll("[A-Z]",""));
        int endRow=Integer.parseInt(end.replaceAll("[A-Z]",""));
        int colCount=endCol-startCol+1;
        int rowCount=endRow-startRow+1;
        int totalCells=colCount*rowCount;
        System.out.println(totalCells);
    }
    static int colToNumber(String col){
        int num=0;
        for (int i=0;i<col.length() ;i++ ){ 
        num=num*25+(col.charAt(i)-'A'+1);
    }
    
    return num;
}
}

Minimum Cable Length for Bulb Connection


In an auditorium, the lighting system of N bulbs is placed in a row. Due to some technical fault in the power supply, only some of the bulbs remain ON and the rest of them go OFF. Due to this flaw in the network, the bulbs which are OFF are unable to work. So till the technical team finds the actual cause of the fault, the technical head Akshay makes some temporary arrangements for the OFF bulbs at a minimum cost. Akshay decides to connect all the OFF bulbs to the nearby ON bulbs so that the length of the cable used to connect them is minimum. He calculates the distance of the systems from the first system.


Write a program to help Akshay find the minimum length of the cable used to turn all the bulbs ON.




Input Format


    

  • The first line of the input consists of an integral num,  representing the number of bulbs (N).
    • 

  • The second line consists of N space-separated integers representing the initial state of each bulb, ON(1) or OFF(0)
    • 

  • The last line consists of N space-separated integers representing the distance of the bulbs from the first bulb.
    • 



Output Format


    

  • Print an integer representing the minimum length of the cable used to turn all the bulbs ON.
    • 





Sample Input:


3
1 0 0
1 5 6


Sample Output:


5


Explanation:


Length of the cable required to connect the 2nd bulb to the 1st bulb =4
Length of the cable required to connect the 3rd bulb to the 2nd bulb =1
The total length of the cable = 5(4+1)


import java.util.*;

public class main {

    public static void main(String[] args) {
      //Write your code here
      Scanner sc=new Scanner(System.in);
      int n=sc.nextInt();
      int[] state=new int[n];
      for(int i=0;i<n;i++){
          state[i]=sc.nextInt();
      }
      int[] position=new int[n];
      for (int i=0;i<n ;i++ ){
          position[i]=sc.nextInt();
      } 
      int totalCAbleLength=0;
      int lastCable=-1;
      for(int i=0;i<n;i++){
      if(state[i]==1){
          lastCable=position[i];
      }
      else if(lastCable!=-1){
          totalCAbleLength+= Math.abs(position[i]-lastCable);
          lastCable=position[i];
      }
      }
    System.out.println(totalCAbleLength);
}
