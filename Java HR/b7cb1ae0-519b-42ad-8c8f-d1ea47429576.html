Count of Unique Student IDs


A Monitor of the class wanted to take the attendance, so he circulated a blank page in the classroom asking students to write there student_Id. But some naughty students wrote their id multiple times to trouble the monitor. Monitor went to the teacher and reported the incident, the Teacher asked him to mark the absence of students whose id is repeated in the attendance sheet and give him the number of students with repeated id in the attendance sheet.


Note: Student id is nothing but an alphabet from a-z.




Input Format


    

  • The input consists of a string representing the ids of students.
    • 



Output Format


    

  • Print an integer representing the count of characters that are not repeated in the string.
    • 





Sample Input:


ghygbvghyghnhjuhjumnj


Sample Output:


3


Explanation:


In the given string, the student IDs that do not repeat are:

- b, v, and m (3 unique IDs).


The rest of the IDs appear multiple times. Hence, the output is 3.


import java.util.Scanner;

public class main {

    public static void main(String[] args) {
      Scanner sc=new Scanner(System.in);
      String s=sc.nextLine();
      int len=s.length();
      char a[]=s.toCharArray();
      int c=0;
      for (int i=0;i<len;i++){
          char p=a[i];
          int temp=0;
          for (int j=0;j<len;j++){
              if(p==a[j]){
                  temp++;
              }
          }
          if(temp==1){
              c++;
          }
      }
     System.out.print(c);
    }
}

3.
Your task is to create a program that takes an integer array arr of numbers and a certain threshold number. Then, it compares each number in the array with the threshold. If a number is greater than the threshold, it goes into the "larger" group. If it's smaller, it goes into the "smaller" group.



- Don't consider the elements equal to the threshold.





Input Format:


    

  • First line of input contains an integer representing the size of the array.
    • 

  • Second line contains the elements of the array arr.
    • 

  • Third line determines the threshold  for which the elements are compared.
    • 





Output Format:


The program should print a string:


    

  • "larger" if there are more integers in array larger than threshold.
    • 

  • "smaller" if there are more integers in array smaller than threshold.
    • 

  • "equal" if the number of integers larger than threshold is the same as the number of integers less than threshold.
    • 





Constraints:


    

  • 1 <= Length of array <= 105 
    • 

  • -109 <= arr[i] <= 109
    • 

  • -109 <= threshold <= 109
    • 





Example:


Input:


6
1 3 0 -1 4 3
2


Output:


equal


Consider the array [1, 3, 0, -1, 4, 3] with a threshold of 2.


    

  • Count of integers greater than threshold:
    • 

  • 

    There are 3 integers (3, 4, and 3) greater than threshold 2.
    

    • 

  • 

    Count of integers less than threshold:
    

    • 

  • There are 3 integers (1, 0, and -1) less than threshold 2.
    • 



Since there are an equal number of integers greater and smaller than the threshold, the output is equal.
import java.util.Scanner;
 
public class main {
    public static void main(String[] args) {
        Scanner sc=new Scanner(System.in);
        int n=sc.nextInt();
        int arr[]=new int[n];
        for(int i=0;i<n;i++){
            arr[i]=sc.nextInt();
        }
        int t=sc.nextInt();
        int cg=0;
        int cl=0;
        int k=0;
        for(int i=0;i<n;i++){
            if(arr[i]>t){
                cg++;
            }else if(arr[i]<t){
                cl++;
            }else{
                k++;
            }
        }
        if(cg==cl){
            System.out.println("equal");
        }else if(cg>cl){
            System.out.println("larger");
        }else{
            System.out.println("smaller");
        }
    }
}
