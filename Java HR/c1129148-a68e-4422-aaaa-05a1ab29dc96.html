1.
Psychic


Raman was playing a game, he starts with x coins. Now in every step, he wins and loses some money.

He came in contact with a psychic who can see the future. The psychic predicted the outcomes after each step.  


Now Raman wants to start the game with the minimum wage such that he never runs out of money (i.e., his coins should never become negative at any point).  


Help Raman to find the minimum starting amount required so that he can play all the steps without going into credit.




Input Format


    

  • First line contains an integer n, the number of steps in the game.  
    • 

  • Next n lines each contain one integer representing the outcome of each step:  
    • 

  • Positive integer: Money won  
    • 

  • Negative integer: Money lost
    • 





Output Format


    

  • A single integer representing the minimum starting amount to ensure Raman never goes into credit.
    • 





Constraints


    

  • 1 <= n <= 109  
    • 

  • -1000 <= Outcome per step <= 1000
    • 





Sample Input:


4
2
-9
15
2




Sample Output:


7




Explanation:


    

  • If Raman starts with 7 rupees:  
    • 

  • After step 1 (+2): 7 → 9  
    • 

  • After step 2 (-9): 9 → 0  
    • 

  • After step 3 (+15): 0 → 15  
    • 

  • After step 4 (+2): 15 → 17  
    • 



At no point does the balance become negative. Hence, the minimum starting amount is 7. 


import java.util.*;

public class main {

    public static void main(String[] args) {
      //Write your code here
      Scanner sc=new Scanner(System.in);
      int n=sc.nextInt();
      int sum=0;
      //int max=Integer.MIN_VALUE;
      int[] arr=new int[n];
      for(int i=0;i<n;i++){
          int a=sc.nextInt();
          arr[i]=a;
      }
      int c=0,f=0,i=0;
      while(i<Integer.MAX_VALUE){
          sum=i;c=0;
          for(int j=0;j<n;j++){
              sum=sum+arr[j];
              if(sum<0){
                  break;
              }else{
                  c++;
              }
          }
          if(c==n){
              f=i;
              break;
          }i++;
      }
      System.out.print(f);
    }
}

Find Dividend


you are given array A, having some dividends. Further, you are given 3 numbers D, Q and R. A dividend can be found using a rule that states:


    

  • Dividend = Divisor x Quotient + Remainder
    • 



Your task is to find and print an integer value representing the index of the dividend if present in array. If dividend not found print -1.




Input Format


    

  • First Line of Input of contains An integer ,representing the size of array A.
    • 

  • Second Line of Input of contains An integer array A, containing dividends.
    • 

  • Third Line of Input of contains An integer D, representing divisor.
    • 

  • Fourth Line of Input of contains An integer Q representing quotient.
    • 

  • Fifth Line of Input of contains An integer R, representing remainder.
    • 



Output Format


    

  • Output contains an integer value representing the index of the dividend if present in array. It dividend not found print - 1.
    • 





Sample Input:


4
26 32 41 36
5
5
1


Sample Output:


0


Explanation:


Here A = [26,32,41,36] and D = 5 and Q is 5 and R is 1 and N is 4.


Now according to rule


    

  • 26 = 5 × 5 + 1 and the index value is 0
    • 



The index where the rule is followed 0. Therefore, 0 is printed as output.


import java.util.Scanner;

public class main {

    public static void main(String[] args) {
      //Write your code here
      Scanner sc=new Scanner(System.in);
      int n=sc.nextInt();
      int[] arr=new int[n];
      for(int i=0;i<n;i++){
          arr[i]=sc.nextInt();
      }
      int d=sc.nextInt();
      int q=sc.nextInt();
      int r=sc.nextInt();
      int D=d*q+r;

      int index=-1;
      for(int i=0;i<n;i++){
          if(D==arr[i]){
              index=i;
              break;
          }
      }
      System.out.print(index);
    }
}
