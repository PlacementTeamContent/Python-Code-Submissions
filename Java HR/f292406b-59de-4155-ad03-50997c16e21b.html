Special Fibonacci


Alex is exploring a series and she came across a speciel series, in which


    

  • f(N) = F(N-1) * F(N-1) + F(N 2) * F(N-2)
    • 

  • where F(0) = 1, F(1) = 1
    • 



Your task is to help Alex find and print an integer value, representing the Nth number in this special series.


Note: Print the output modulo 47.




Input Format


Input contains an integer value N.


Output Format


Output contains an integer value, representing the Nth number in this special series.




Sample Input:


2


Sample Output:


2


Explanation:


The series is defined by:


F(N) = F(N-1) * F(N-1) + F(N-2) * F(N-2)
Base cases: F(0) = 1, F(1) = 1


F(2)=F(1)*F(1)+F(0)F(0)
as F(1),F(0) value is 1
F(2)=1*1+11
F(2)=2


import java.util.*;

public class main {

    public static void main(String[] args) {
      //Write your code here
      Scanner sc = new Scanner(System.in);
      int n = sc.nextInt();
      int MOD = 47;
      if(n==0 || n==1) {
          System.out.println(1);
          return;
      }
      int[] dp = new int[n+1];
      dp[0] = 1;
      dp[1] = 1;
      
      for(int i=2;i<=n;i++) {
          dp[i] = (dp[i-1] * dp[i-1] + dp[i-2] * dp[i-2]) % MOD;
      }
      System.out.println(dp[n]);
    }
}

You are given a time series of stock prices for N days. Implement a function to compute the Simple Moving Average (SMA) over a window of K days. The SMA on day i is the average of the stock prices from day i-K+1 to day i.


 Input
The first line contains two integers, N (1 ≤ N ≤ 10^5) and K (1 ≤ K ≤ N).
The second line contains N space-separated floating-point numbers representing the stock prices.


Output
Print N-K+1 space-separated floating-point numbers consists 2 decimal points representing the SMA for each valid day.


Explanation


5 3


10.0 20.0 30.0 40.0 50.0


Calculate the first SMA
consider first 3 positions i.e 10.0 20.0 30.0
The average of these(10.0 20.0 30.0) is 20.0


Slide the window to the next Position:then it will be 20.0 30.0 40.0
The average of these(20.0 30.0 40.0) is 30.0


Slide the window again to the next position :then it will be 30.0 40.0 50.0
The average of these(30.0 40.0 50.0) is 40.0


The output should be rounded upto 2 decimal points Now the output is 20.00 30.00 40.00
import java.util.*;
import java.text.DecimalFormat;
 
public class main {
    public static void main(String[] args) {
        //Write your code below
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        int K = sc.nextInt();
        
        double[] prices = new double[N];
        for(int i=0; i<N; i++){
            prices[i] = sc.nextDouble();
        }
      
        DecimalFormat df = new DecimalFormat("#0.00");
        double windowSum = 0.0;
        
        for(int i=0; i<K; i++) {
            windowSum += prices[i];
        }
        System.out.print(df.format(windowSum/K)+" ");
        
        for(int i=K;i<N;i++) {
            windowSum += prices[i] - prices[i-K];
            System.out.print(df.format(windowSum/K)+" ");
        }
        sc.close();
        
    }
}

Distinct Count Queries


Rahul has an array a of size n. For each query, Rahul wants to know how many distinct integers are present from a given index li to the end of the array. Help Rahul determine the answer for each query efficiently.

Output Format

import java.util.*;
class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        int n = sc.nextInt();
        int m = sc.nextInt();
        
        int arr[] = new int[n];
        int ans[] = new int[n];
        
        for(int i=0;i<n;i++) {
            arr[i] = sc.nextInt();
        }
        Set<Integer> seen = new HashSet<>();
        for(int i=n-1;i>=0;i--) {
          seen.add(arr[i]);
          ans[i] = seen.size();
        }
        for(int i=0;i<m;i++) {
            int query = sc.nextInt();
            System.out.println(ans[query-1]);
        }
        sc.close();
    }
}
