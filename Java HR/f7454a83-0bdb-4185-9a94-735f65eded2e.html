Clean String


You are given two strings A and B. Your task is to find and print a string representing the left over string in A after removing all the letters that exsist in string B . Print nothing if the output does not contain any value.


Note :


    

  • Strings A and B contain English alphabets in upper case only.
    • 

  • A single alphabet in B can replace all the occurrences of that alphabet in A
    • 





Input Format


    

  • First line of Input contains String A.
    • 

  • Second line of Input contains String B.
    • 



Output Format


Print a string representing the left over string in A after removing all the letters that exist in string B. 
Print nothing if the output does not contain any value.




Sample Input:


AABBCC
AB


Sample Output:


CC


Explanation:


After removing all occurences of A,B from the string A , there will be CC.


import java.util.Scanner;

public class main {

    public static void main(String[] args) {
      //Write your code here
      Scanner sc=new Scanner(System.in);
      String A=sc.nextLine();
      String B=sc.nextLine();
      boolean[] toRemove=new boolean[26];
      for(char c:B.toCharArray()){
          toRemove[c - 'A']=true;
      }
      StringBuilder result=new StringBuilder();
      for( char c:A.toCharArray()){
          if(!toRemove[c - 'A']){
              result.append(c);
          }
      }
      if(result.length()>0){
          System.out.println(result);
      }
      sc.close();
    }
}
