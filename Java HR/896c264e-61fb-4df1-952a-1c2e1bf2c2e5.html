Island Life


You are stuck on an island where they sell and eat coconut sweets only. A person can buy at most 1 box per day with each box containing N pieces .To remain alive you must consume E coconut sweets daily for D days, but the catch is that you cannot purchase sweets on Sundays. Your task is to find and return an integer value representing the minimum number of times you have to buy coconut sweets in order to stay alive. If not possible, return -1


Note: The day starts from Monday




Input Format


    

  • First line of input contains an integer value N, representing the number of coconut sweets per box.
    • 

  • Second line of input contains an integer value E, representing the number of coconut sweets you must
eat daily.
    • 

  • Third line of input contains an integer value D, representing the number of days you have to spend
on the island.
    • 



Output Format


    

  • The output should contains an integer value representing the minimum number of times you have to
buy coconut sweets in order to stay alive. if not possble return -1.
    • 





Sample Input:


16
2
10


Sample Output:


2


Explanation:


Here, there are 16 coconut sweets per box. You have to live for 10 days on the island and each day you will have to eat 2 sweets. So in 8 days you wil run out of sweets and buy another box on the 9th day.
Now, this second box will last for the next 8 days, which means that you con eat till 16th day but you will have to leave the island after 10th day, so you don't need to buy any more boxes. Since you have bought sweet boxes 2 times, 2 is returned as the output


import java.util.Scanner;

public class main {

    public static void main(String[] args) {
      Scanner sc=new Scanner(System.in);
      int box=sc.nextInt();
      int eat=sc.nextInt();
      int days=sc.nextInt();
      
     int cnt=1;
     while(cnt!=0){
         int ans=box*cnt;
         int s=ans/eat;
         if(s>=days){
             break;
         }
         cnt++;
     }
     System.out.println(cnt);
    }
}

Given a sequence of integers and a common ratio q, your task is to count the number of triplets within the sequence that can be arranged to form a geometric progression with the ratio q.



- A triplet subset from the sequence should consist of three elements in increasing order that follow the condition (a, a x q, a x q^2) to form a geometric progression with ratio q.





Input Format


- The first line contains two space-separated integers: N, which is the size of the sequence, and q, the common ratio of the geometric progression.
- The second line contains N space-separated integers representing the elements of the sequence.




 Output Format


- Output a single integer representing the number of triplet subsets from the sequence that can be ordered to form a geometric progression with the given common ratio q.




Constraints


 - 1<= N <= 10^6 
 - 1 <= q<= 10^8 
 - 1<= sequence[i] <= 10^8




 Example


For the given Input:


6 3
1 3 9 9 27 81


Output:


6


Explanation:


    

  • Possible triplets forming a geometric progression with the common ratio q = 3 are:
    • 

  • (1, 3, 9) corresponding to indices (0, 1, 2)
    • 

  • (1, 3, 9) corresponding to indices (0, 1, 3)
    • 

  • (3, 9, 27) corresponding to indices (1, 2, 4)
    • 

  • (3, 9, 27) corresponding to indices (1, 3, 4)
    • 

  • (9, 27, 81) corresponding to indices (2, 4, 5)
    • 

  • 

    (9, 27, 81) corresponding to indices (3, 4, 5)
    

    • 

  • 

    Each of these triplets can be arranged to show that they follow a geometric progression. The count of such triplets is 6.
    

    • 



import java.util.*;
class Main {
    public static void main(String[] args) { 
        Scanner sc=new Scanner(System.in);
        int n=sc.nextInt();
        int q=sc.nextInt();
        int[] arr=new int[n];
        for(int i=0;i<n;i++){
            arr[i]=sc.nextInt();
        }
        int cnt=0;
        for(int i=0;i<n-2;i++){
            int a=arr[i];
            for(int j=i+1;j<n-1;j++){
                if(a*q==arr[j]){
                    for(int k=j+1;k<n;k++){
                        if(a * Math.pow(q,2)==arr[k]){
                            cnt++;
                        }
                    }
                }
            }
        }
        System.out.println(cnt);
    }
}

Device Name System


Rocky is a software engineer and he is creating his own operating system called “myFirst OS”. myFirst OS is a GUI (Graphical User Interface) based operating system where everything is stored in files and folders. He is facing issues with creating unique folder names for the operating system. 


Help Rocky create unique folder names for his OS. If a folder name already exists in the system, an integer number is added to the name to make it unique. The integer starts with 1 and is incremented by 1 for each new request of an existing folder name. Given a list of folder names, process all requests and return an array of corresponding folder names.


Input Format:


    

  • The first line contains an integer n, denoting the size of the array foldername.
    • 

  • Each of the next n lines contains a string foldername[i] representing a folder name request in the order received.
    • 



Output Format:


    

  • The output should be a list of n strings, where each string represents the folder name after processing the request and ensuring uniqueness.
    • 



Constraints:


    

  • 1 <= n <= 104
    • 

  • 1 <= length of foldername[i] < 20
    • 

  • foldername[i] contains only lowercase English letters in the range ascii[a-z].
    • 



Sample Input:


4
home
download
first
first


Sample Output:


home
download
first
first1


Explanation:


    

  1. foldername[0] = 'home' is unique.
    2. 

  2. foldername[1] = 'download' is unique.
    3. 

  3. foldername[2] = 'first' is unique.
    4. 

  4. foldername[3] = 'first' already exists, so first1 is assigned.
    5. 



Thus, the unique folder names are ['home', 'download', 'first', 'first1'].


Notes:


    

  • The integer added to the folder name starts at 1 and is incremented by 1 for each subsequent request for an existing folder name.
    • 

  • The folder name first1 is unique after the second request for first because first already exists in the system.
    • 

import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;
import java.util.Map;

class Solution {
    public List<String> getUniqueFolderNames(String[] foldernames) {
        
        List<String> uniqueNames = new ArrayList<>();
        int n=foldernames.length;
        int cnt=1;
        for(String s:foldernames){
            
            if(uniqueNames.indexOf(s)==-1){
                uniqueNames.add(s);
            }else{
                uniqueNames.add(s+cnt);
                cnt++;
            }
        }
        return uniqueNames;
    }
}
