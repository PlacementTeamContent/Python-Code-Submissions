1.
Psychic


Raman was playing a game, he starts with x coins. Now in every step, he wins and loses some money.

He came in contact with a psychic who can see the future. The psychic predicted the outcomes after each step.  


Now Raman wants to start the game with the minimum wage such that he never runs out of money (i.e., his coins should never become negative at any point).  


Help Raman to find the minimum starting amount required so that he can play all the steps without going into credit.




Input Format


    

  • First line contains an integer n, the number of steps in the game.  
    • 

  • Next n lines each contain one integer representing the outcome of each step:  
    • 

  • Positive integer: Money won  
    • 

  • Negative integer: Money lost
    • 





Output Format


    

  • A single integer representing the minimum starting amount to ensure Raman never goes into credit.
    • 





Constraints


    

  • 1 <= n <= 109  
    • 

  • -1000 <= Outcome per step <= 1000
    • 





Sample Input:


4
2
-9
15
2




Sample Output:


7




Explanation:


    

  • If Raman starts with 7 rupees:  
    • 

  • After step 1 (+2): 7 → 9  
    • 

  • After step 2 (-9): 9 → 0  
    • 

  • After step 3 (+15): 0 → 15  
    • 

  • After step 4 (+2): 15 → 17  
    • 



At no point does the balance become negative. Hence, the minimum starting amount is 7. 


import java.util.Scanner;

public class main {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int[] arr = new int[n];
        for(int i=0;i<n;i++){
            arr[i] = sc.nextInt();
        }
        int neg=0;
        int count =0;
        int index=0;
        boolean isZero = true;
        for (int i=0;i<arr.length ;i++ ){
            if(arr[i]!=0)
                isZero=false;
        }
        if(isZero){
            System.out.println(0);
        }
        else{
        for (int i=0;i<arr.length ;i++ ) {
            if (arr[i]<0){
                neg+=arr[i];
                index=i;
                count++;
            }
        }
        int min=0;
        for(int i=0;i<index;i++){
           if(arr[i]>0)
           min+=arr[i];
        }

        System.out.println(-(neg+min));
    }
    }
}
2.
Write a program that counts the Consonants in the string M and print the Consonant that has the highest number of counts.


The program will also convert the Vowel to the next Consonant as per the Alphabet order, and that converted Consonant will be added to the previous count of the same Consonant. 


If two or more Consonants have the same number of counts then we will print the bigger letter.




Input Format:


The input contains a string representing M.

Output Format:


The output contains an uppercase string representing Consonant that has the highest count.


Constraints: 


    

  • 1<= Length of M <= 105
    • 






Explanation


Let's take an example:
Input:


bana


Output:


B


The input "bana" yields:
'b' has a count of 1.
'n' has a count of 1.
'a' has a count of 2, but since 'a' becomes 'b' (its next consonant), the counts of 'a' are added to 'b'.


Thus, 'b' now has a count of 3, and 'n' remains at 1.
Therefore, 'b' has the highest count in the string. 
The uppercase for 'b' is "B". So the output is "B".


import java.util.Scanner;
 
public class main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String s = sc.next();
        char [] ch = s.toCharArray();
        
        int count =0;
        char ch1='a';
        for(int i=0;i<ch.length-1;i++){
            for(int j=i+1;j<ch.length-1;j++){
            if(ch[i]>ch[j]){
                count++;
                ch1=ch[i];
            }
        }
        }
        String s1=""+ch1;
        
        System.out.println(s1.toUpperCase());
    }
}
3.
A password string, pwd, consists of binary characters (0s and 1s). A cybersecurity expert is trying to determine the minimum number of changes required to make the password secure.


Definition of a Secure Password
To be considered secure, the password must be divided into non-overlapping even-length substrings, where:


    

  • Each substring must consist of only 0s or only 1s (i.e., it should be uniform).
    • 

  • No substring should contain both 0s and 1s.
    • 



The task is to determine the minimum number of character flips required to transform pwd into a secure password.
A flip is defined as changing 0 → 1 or 1 → 0.




Input Format


    

  • Input contains a single string pwd consisting of 0s and 1s.
    • 



Output Format


    

  • Output contains an integer representing the minimum number of flips required.
    • 





Sample Input:


1100101


Sample Output:


1


Explanation:


The given string 1100101 must be split into even-length uniform substrings. The optimal transformation:


    

  • These are the even substrings(11,00,10) in this only one substring needs to flip(10)
    • 

  • Flip one 1 in 10 to make it uniform (11)
    • 



Total flips required: 1.


import java.util.Scanner;

public class main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String s = sc.next();
        int count=0;
        for(int i=0;i<s.length()-1;i++)
        {
            if(s.charAt(i)==s.charAt(i+1)){
                i++;
                break;
            }
            else{
                count++;
                i++;
            }
        }
        System.out.println(count);
    }
}
