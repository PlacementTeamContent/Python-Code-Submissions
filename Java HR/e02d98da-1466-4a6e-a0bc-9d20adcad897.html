Matched Socks Pairs


Alex works at a clothing store. There is a large pile of socks that must be paired by color for sale. Given an array of integers representing the color of each sock, determine how many pairs of socks with matching colors there are.


For example, there are n=7 socks with colors ar = {1,2,1,2,1,3,2}. There is one pair of color 1 and one of color 2. There are three odd socks left, one of each color. The number of pairs is 2.




Input Format


    

  • The first line contains an integer n, the number of socks represented in ar.
    • 

  • The second line contains n space-separated integers describing the colors ar[i] of the socks in the pile.
    • 





Output Format


    

  • Print the total number of matching pairs of socks that Alex can sell.
    • 





Constraints


    

  • 1 <= n <= 100
    • 

  • 1 <= ar[i] <= 100 and 0 <= i < n
    • 





Sample Input


9
10 20 20 10 10 30 50 10 20


Sample Output


3


Explanation


    

  • Alex can match 3 pairs of socks i.e. 10-10, 10-10, 20-20 while the left out socks are 50, 30, and 20.
    •

Network Stream


A stream of n data packets arrives at a server. This server can only process packets that are exactly 2n units long for some non-negative integer value of n (0 ≤ n).

All packets are repackaged in order to the largest possible value of 2n units. The remaining portion of the packet is added to the next arriving packet before it is repackaged.

Find the size of the largest repackaged packet in the given stream.




Input Format


    

  • The first line contains an integer n, the number of packets.
    • 

  • The second line contains n packets, separeted with spaces.
    • 



Output Format


    

  • Print the size of the largest packet that is streamed.
    • 





Constraints


    

  • 1 ≤ n ≤ 105
    • 

  • 1 ≤ arrivingPackets[i] size ≤ 109
    • 





Sample Input 1:


5
13 25 12 2 8


Sample Output 1:


16


Explanation:


    

  • First packet: 13 → maximum 2n = 8 → remaining 13 - 8 = 5.
    • 

  • Second packet: 25 + 5 = 30 → maximum 2n = 16 → remaining 30 - 16 = 14.
    • 

  • Third packet: 14 + 12 = 26 → maximum 2n = 16 → remaining 26 - 16 = 10.
    • 

  • Fourth packet: 10 + 2 = 12 → maximum 2n = 8 → remaining 12 - 8 = 4.
    • 

  • Fifth packet: 4 + 8 = 12 → maximum 2n = 8 → remaining 12 - 8 = 4.
    • 



The largest packet repackaged was 16 units.

k-Anagram


Two strings are called k-anagrams if both of the below conditions are true:
1. Both have the same number of characters.
2. Two strings can become anagrams by changing at most k characters in a string.


Given two strings of lowercase alphabets and an integer k, implement a function to determine if the two strings are k-anagrams of each other.




Input Format


    

  • A string s1 consisting of lowercase English letters.
    • 

  • A string s2 consisting of lowercase English letters.
    • 

  • An integer k.
    • 



Output Format


    

  • Print true if s1 and s2 are k-anagrams, otherwise print false.
    • 





Constraints


    

  • 1 ≤ s1.size(), s2.size() ≤ 10⁵
    • 

  • 1 ≤ k ≤ 10⁵
    • 





Example 1


Input


fodr
gork
2


Output


True


Explanation


We can change 'f' → 'g' and 'd' → 'k' in s1 to make it an anagram of s2.




Example 2


Input


geeks
eggkf
1


Output


False


Explanation


We can modify only 1 character, but 2 modifications are needed ('g' and 'f' in s2).


import java.util.*;

class Solution {
    public boolean areKAnagrams(String s1, String s2, int k) 
    {
        if(s1.length()!=s2.length()) return false;
        char [] a = s1.toCharArray();
        char [] b = s2.toCharArray();
        Arrays.sort(a);
        Arrays.sort(b);
        int c =0;
        for(int i=0;i<a.length;i++)
        {
            if(a[i]==b[i]) continue;
            else
            c++;
        }
        return c<=k;
    }
}
