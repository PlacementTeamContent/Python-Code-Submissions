You are provided with an array of integers named dataset. Your objective is to determine the number of unique pairs (m, n) such that 0 <= m < n < dataset.length, and dataset[m] can be derived from dataset[n] by swapping at most two digits of dataset[n].



  If two numbers are the same then don't consider this pair in swapped pair





Input Format:


    

  • First line of input contains an integer representing the number of elements in the array. 
    • 

  • Second line contains the array elements separated by space.
    • 





Output Format:


    

  • The output is a single integer value that represents the count of distinct pairs (m, n) .
    • 





Constraints


    

  • 1 <= dataset.length <= 10^5
    • 

  • 1 <= dataset[m] <= 10^8
    • 





Example


Input:


Consider the following dataset array:


7
1 23 156 1650 651 165 32


Output:


3


    

  • dataset[1] = 23 can be obtained from dataset[6] = 32 by swapping the two digits.
    • 

  • dataset[2] = 156 can be obtained from dataset[4] = 651 by swapping the '6' and '1'.
    • 

  • dataset[2] = 156 can also be obtained from dataset[5] = 165 by swapping the '6' and '5'.
    • 



So there are 3 pairs. The Output is 3.

Island Life


You are stuck on an island where they sell and eat coconut sweets only. A person can buy at most 1 box per day with each box containing N pieces .To remain alive you must consume E coconut sweets daily for D days, but the catch is that you cannot purchase sweets on Sundays. Your task is to find and return an integer value representing the minimum number of times you have to buy coconut sweets in order to stay alive. If not possible, return -1


Note: The day starts from Monday




Input Format


    

  • First line of input contains an integer value N, representing the number of coconut sweets per box.
    • 

  • Second line of input contains an integer value E, representing the number of coconut sweets you must
eat daily.
    • 

  • Third line of input contains an integer value D, representing the number of days you have to spend
on the island.
    • 



Output Format


    

  • The output should contains an integer value representing the minimum number of times you have to
buy coconut sweets in order to stay alive. if not possble return -1.
    • 





Sample Input:


16
2
10


Sample Output:


2


Explanation:


Here, there are 16 coconut sweets per box. You have to live for 10 days on the island and each day you will have to eat 2 sweets. So in 8 days you wil run out of sweets and buy another box on the 9th day.
Now, this second box will last for the next 8 days, which means that you con eat till 16th day but you will have to leave the island after 10th day, so you don't need to buy any more boxes. Since you have bought sweet boxes 2 times, 2 is returned as the output


import java.util.Scanner;

Sum Number Divisible


You are required to implement the following function:


Int SumNumberDivisible(int m, int n);


The function accepts two positive integers m and n as its arguments. You are required to calculate the sum of numbers divisible by both 3 and 5, between m and n (both inclusive), and return the sum.




Input Format:


    

  • The first line contains two integers m and n representing the range of numbers (inclusive).
    • 





Output Format:


    

  • Return the sum of numbers divisible by both 3 and 5 in the range from m to n (inclusive).
    • 





Constraints:


    

  • 0 < m <= n
    • 

  • 1 <= m, n <= 106
    • 





Sample Input 1:


12
50


Sample Output 1:


90


Explanation:


The numbers divisible by both 3 and 5, between 12 and 50 (both inclusive) are:


    

  • {15, 30, 45}, and their sum is 15 + 30 + 45 = 90.
    • 



class Solution {
    public int sumNumberDivisible(int m, int n) {
        int sum=0;
        for(int i=m;i<=n;i++){
            int lastval=i%10;
            if(lastval==0 || lastval==5){
                if(i%3==0){
                    sum+=i;
                }
            }
        }
        return sum;
    }
}
