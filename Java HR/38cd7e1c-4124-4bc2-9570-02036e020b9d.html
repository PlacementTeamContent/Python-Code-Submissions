1.
Prime Sum Range


You will be given a left limit and a right limit. Your task is to calculate the sum of all prime numbers between a given range x and y, both inclusive.




Input Format


    

  • The first line contains an integer x representing the left limit.
    • 

  • The second line contains an integer y representing the right limit.
    • 



Output Format


    

  • Output a single integer which is the sum of all prime numbers in the given range.
    • 





Constraints


    

  • 1 <= x <= y <= 1000
    • 





Example 1


Input


30 
40




Output


68




Explanation


    

  • Prime numbers between 30 and 40 are 31 and 37. Their sum is 31 + 37 = 68.
    • 



import java.util.*;
 
class Main {
    public static void main(String[] args) {
        Scanner sc=new Scanner(System.in);
        int x=sc.nextInt();
        int y=sc.nextInt();
        int sum=0;
        for(int i=x;i<=y;i++){
            if(isprime(i)){
                sum+=i;
            }
        }
        System.out.println(sum);
        sc.close();
    }
static boolean isprime(int n){
    if(n<2)
        return false;
    
    for(int i=2;i<=Math.sqrt(n);i++){
        if(n%i==0)
            return false;
    } 
        return true;
    }
}

Count Elements Greater Than All Prior Elements


You are given an array of N integers. Your task is to count how many elements in the array are greater than all the elements before them. The first element is always counted because there are no previous elements to compare. As you move through the array, check if a number is greater than all its previous numbers. If yes, include it in the count. Your goal is to find and print this count.





Input Format


    

  • An integer N (size of the array).
    • 

  • Next N lines contains N inputs , representing the elements of the array.
    • 






Output Format


    

  • An integer, representing the count of elements satisfying the condition.
    • 






Constraints


    

  • 1 <= N <= 20
    • 

  • 1 <= arr[i] <= 10,000
    • 






Example 1


N=5 and Arr = [7, 4, 8, 2, 9].


The first element 7 is included.

8 is greater than all previous elements [7, 4].

9 is greater than all previous elements [7, 4, 8, 2].  


Input  


5
7
4
8
2
9


Output  


3





Example 2


N=5 and Arr = [3, 4, 5, 8, 9].


All elements are greater than their prior elements.


Input  


5
3
4
5
8
9


Output  


5


import java.util.Scanner;
class Main {
    public static void main(String[] args) {
        Scanner sc= new Scanner(System.in);
        int N=sc.nextInt();
        int[]arr = new int[N];
        for (int i=0;i<N;i++){
            arr[i]=sc.nextInt();
        }
        int count=1;
        int maxSoFar=arr[0];
        for(int i=1;i<N;i++){
            if(arr[i]>maxSoFar){
                count++;
                maxSoFar=arr[i];
            } 
        }
        System.out.println(count);
    }
}

You are given elements of two sets, A  and B.


Your job is to find whether setA   is a subset of setB.


If setA   is subset of set B, print True.


If set A  is not a subset of setB , print False.




Input


    

  • 

    The first line will contain the number of test cases, T.
    

    • 

  • 

    The first line of each test case contains the number of elements in setA.
    

    • 

  • 

    The second line of each test case contains the space separated elements of setA.
    

    • 

  • 

    The third line of each test case contains the number of elements in setB.
    

    • 

  • The fourth line of each test case contains the space separated elements of setB.
    • 





Output


Output should contain True or False for each test case on separate lines.

Explanation


Here's the explanation for the provided test case:

Input:

True


    

  • In this test case, set A is {1, 2, 3, 5, 6} and set B is {9, 8, 5, 6, 3, 2, 1, 4, 7}. All the elements of set A are present in set B, so set A is a subset of set B. The output is True.

Output:


False


    

  • In this test case, set A is {2} and set B is {3, 6, 5, 4, 1}. Set A has an element (2) that is not present in set B, so set A is not a subset of set B. The output is False.

import java.util.*;
class Main {
    public static void main(String[] args) { 
        Scanner sc=new Scanner(System.in);
        int T=sc.nextInt();
        for(int t=0;t<T;t++){
            int sizeA=sc.nextInt();
            Set <Integer> setA = new HashSet<>();
            for(int i=0;i<sizeA;i++){
                setA.add(sc.nextInt());
            }
            int sizeB=sc.nextInt();
            Set <Integer> setB = new HashSet<>();
            for(int i=0;i<sizeB;i++){
                setB.add(sc.nextInt());
        }
        if(setB.containsAll(setA)){
            System.out.println("True");
        }
        else{
             System.out.println("False");
        }
    }
    }
}
