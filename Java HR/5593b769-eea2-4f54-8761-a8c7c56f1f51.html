Possible Password Combinations


Elliot created a KeyLogger for his friend Romero to monitor his passwords. The KeyLogger records the keys pressed by the user without their consent, which is unethical. Romero writes his passwords in lowercase characters only, and the KeyLogger records the corresponding numeric value for each key. For example:
- 'a' corresponds to '1',
- 'b' corresponds to '2',
- and so on until 'z' which corresponds to '26'.


Given a number, Elliot can generate all possible password combinations using the recorded values and perform a dictionary-based password attack. Your task is to print all the possible password combinations in lexicographic order.




Input Format


    

  • A single line containing the value recorded by the KeyLogger (a string of digits where each digit or pair of digits corresponds to a letter).
    • 





Output Format


    

  • All possible password combinations corresponding to the number, printed in lexicographical order.
    • 





Constraints


    

  • 2≤Length of input string≤1000
    • 





Sample Input


1234


Sample output


abcd
awd
lcd


Explanation


For the input 1234, you can generate the following possible passwords:


    

  • 1 corresponds to 'a' and 2 corresponds to 'b', forming the combination "ab".
    • 

  • 1 corresponds to 'a' and 23 corresponds to 'w', forming the combination "aw".
    • 

  • 12 corresponds to 'l' and 3 corresponds to 'c', forming the combination "lc".
    • 

  • 12 corresponds to 'l' and 4 corresponds to 'd', forming the combination "ld".
    • 



Thus, the output would be the lexicographically sorted combinations:


    

  • abcd
    • 

  • awd
    • 

  • lcd
    • 



import java.util.*;
 class Main {
    static List<String> result = new ArrayList<>();
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String input =sc.nextLine();
        decode(input,0,"");
        Collections.sort(result);
        for(String s:result){
            System.out.println(s);
        }
    }
    static void decode(String s, int index, String current){
        if(index==s.length()){
            result.add(current);
            return;
        }
        int num1=s.charAt(index)-'0';
        if(num1>=1 && num1<=9){
            decode(s,index+1,current+(char)('a'+num1-1));
        }
        if(index+1<s.length()){
           int num2 = Integer.parseInt(s.substring(index, index+2));
            if(num2>=10 && num2<=26){
                decode(s,index+2, current+(char)('a'+num2-1));
        
            }
        }
        
    }
}

import java.util.*;
class Main {
    public static void main(String[] args) { 
        // Write your code here
        Scanner sc = new Scanner(System.in);
        
        int mainSetSize = sc.nextInt();
        Set<Integer> mainSet = readSet(sc,mainSetSize);
        
        int n=sc.nextInt();
        int failedCount =0;
        
        for(int i=0;i<n;i++){
            int subsetSize = sc.nextInt();
            
            Set<Integer>subSet = readSet(sc,subsetSize);
            
            if(!(mainSet.containsAll(subSet)&&mainSet.size()>subSet.size())){
                failedCount++;
            }
        }
        if(failedCount==0){
            System.out.println("True" +" "+ mainSet.size());
        }else{
            System.out.println("False"+" "+ failedCount);
        }
        sc.close();
    }
    public static Set<Integer> readSet(Scanner sc, int size){
        Set<Integer> set = new HashSet<>();
        for(int i=0;i<size;i++){
            set.add(sc.nextInt());
        }
        return set;
    }
}
