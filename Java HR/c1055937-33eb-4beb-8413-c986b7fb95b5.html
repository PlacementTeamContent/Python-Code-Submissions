Count of Unique Student IDs


A Monitor of the class wanted to take the attendance, so he circulated a blank page in the classroom asking students to write there student_Id. But some naughty students wrote their id multiple times to trouble the monitor. Monitor went to the teacher and reported the incident, the Teacher asked him to mark the absence of students whose id is repeated in the attendance sheet and give him the number of students with repeated id in the attendance sheet.


Note: Student id is nothing but an alphabet from a-z.




Input Format


    

  • The input consists of a string representing the ids of students.
    • 



Output Format


    

  • Print an integer representing the count of characters that are not repeated in the string.
    • 





Sample Input:


ghygbvghyghnhjuhjumnj


Sample Output:


3


Explanation:


In the given string, the student IDs that do not repeat are:

- b, v, and m (3 unique IDs).


The rest of the IDs appear multiple times. Hence, the output is 3.


import java.util.*;

public class main {

    public static void main(String[] args) {
     Scanner inpt = new Scanner(System.in);
     String name=inpt.nextLine();
    Map<Character,Integer> charCounts= new HashMap<>();
    for(char c:name.toCharArray()){
        charCounts.put(c,charCounts.getOrDefault(c,0)+1);
    }
    int count=0;
    for(Map.Entry<Character,Integer> entry:charCounts.entrySet()){
        if(entry.getValue()==1){
            count++;
        }
    }
    System.out.println(count);
    }
}

Two strings are said to be similar if they are composed of the same characters. For example, abababc and accccbaa are similar since both of them are composed of characters a, b and c. However, cabca and badcd are not similar since they do not share all of the same letters.


Given an array of strings strArr of length m, find the number of pairs of strings that are similar.



Keep in mind :


    

  1. Each string is composed of lowercase English characters only.
    2. 

  2. Pairs are considered index-wise, i.e, two equal strings at different indices are counted as separated pairs
    3. 

  3. A pair of indices (i,j) is same as the pair (j,i);

    4. 



Your task is to provide an efficient algorithm to find the number of pairs of strings that are similar.
Input
• The first line consists of an Integer M denoting size of string array strArr.
• The second line consists of M space separated Strings representing the elements of strArr array.
Output
• Print an Integer denoting count the number of pairs of strings that are similar in strArr array.
Constraints
• 1 <= M <= 10^4
 • 'a'<= strArr[i][j]<= 'z'
Example :
Let's take an example:


3
xyz foo of


Here, the strings "foo" and "of" are similar because they are composed of the same characters ['o', 'f']. There are no other similar pairs so the answer is 1.
import java.util.*;

class Main {
    
    private static String normalize(String s){
        char[] ch = s.toCharArray();
        Arrays.sort(ch);
        return new String(ch);
    }
    public static void main(String[] args) { 
      Scanner inpt = new Scanner(System.in);
      int n = Integer.parseInt(inpt.nextLine());
      String[] strings = inpt.nextLine().split(" ");
      
      Map<String,Integer> freqMap = new HashMap<>();
      
      for(String s:strings){
          String key = normalize(s);
          freqMap.put(key,freqMap.getOrDefault(key,0)+1);
      }
      
      int totalPairs = 0;
      for(int count:freqMap.values()){
          if(count>0){
              totalPairs+=count*(count-1)/2;
          }
      }
      System.out.println(totalPairs);
    }
}

Count Excel Sheet Cells


Given a range in an Excel sheet, determine the number of cells within that range. The range is provided in the format start:end, where start and end represent the top-left and bottom-right cells respectively.




Input Format


    

  • A single line representing the range in the format start:end.
    • 



Output Format


    

  • A single integer representing the number of cells in the given range.
    • 





Constraints


    

  • The cells are within a valid Excel range (from A1 to XFD1048576).
    • 





Example 1


Input


A1:C8


Output


24


Explanation


    

  • The range is from A1 to C8. This range includes 3 columns (A, B, C) and 8 rows (1 to 8). 
    • 

  • Total cells = 3 * 8 = 24.
    • 



import java.util.*;

public class main {
    public static void main(String[] args) {
        // Make your logic here
        Scanner inpt = new Scanner(System.in);
        String val = inpt.nextLine();
        String[] parts = val.split(":");
        if(parts.length!=2){
            System.out.println("Mot valid");
        }
        
        int[] start = parseCell(parts[0]);
        int[] end= parseCell(parts[1]);
        int colcount = end[1]-start[0]+1;
        int rowcount = end[0]-start[0]+1;
        System.out.println(colcount*rowcount);
    }
    
    private static int[] parseCell(String cell){
        int col=0;
        int rowIndex=0;
        int i=0;
        while(i<cell.length()&&Character.isLetter(cell.charAt(i))){
            col=col*26+(cell.charAt(i)-'A'+1);
            i++;
        }
        String rowStr = cell.substring(i);
        rowIndex = Integer.parseInt(rowStr);
        return new int[]{col-1,rowIndex-1};
    }
}
