Count of Unique Student IDs


A Monitor of the class wanted to take the attendance, so he circulated a blank page in the classroom asking students to write there student_Id. But some naughty students wrote their id multiple times to trouble the monitor. Monitor went to the teacher and reported the incident, the Teacher asked him to mark the absence of students whose id is repeated in the attendance sheet and give him the number of students with repeated id in the attendance sheet.


Note: Student id is nothing but an alphabet from a-z.




Input Format


    

  • The input consists of a string representing the ids of students.
    • 



Output Format


    

  • Print an integer representing the count of characters that are not repeated in the string.
    • 





Sample Input:


ghygbvghyghnhjuhjumnj


Sample Output:


3


Explanation:


In the given string, the student IDs that do not repeat are:

- b, v, and m (3 unique IDs).


The rest of the IDs appear multiple times. Hence, the output is 3.



import java.util.*;
public class main {

    public static void main(String[] args) {
     Scanner sc=new Scanner(System.in);
     String input=sc.nextLine();
     ArrayList<String> hset=new ArrayList<>();
     for(int i=0;i<input.length();i++){
         char ch=input.charAt(i);
         String s=Character.toString(ch);
         hset.add(s);
        //  System.out.println(s);
     }
     int count=0;
     for(int i=0;i<hset.size();i++){
        int a= Collections.frequency(hset,hset.get(i));
        if(a==1){
            count++;
        }
        
     }
     System.out.println(count);
    }
}

Find nth Number In Series


You want to find the nth number in a series using the following parameters:


    

  • The first four numbers in the series are always set as n1=1, n2=2, n3=3, n4=4.
    • 

  • Subsequent numbers in the series are obtained by summing the four preceding numbers.
    • 



Write a program that takes an integer n as input and print the value of the nth number in the series.


If the input n is less than 1, your function should print the string Invalid.




Input format


    

  • A single integer n representing the position of the number in the series to find.
    • 





Output format


    

  • 

    A single integer nth position in the series.
    

    • 

  • 

    If n < 1, print: Invalid
    

    • 





**Constraints: **


    

  • 1 ≤ n ≤ 50
    • 





Example 1


Input:


6


Output:


19


Explanation:


    

  • The first four numbers in the series are always 1, 2, 3, 4. The fifth number in the series is obtained by summing the previous four numbers: 1 + 2 + 3 + 4 = 10. Similarly, the sixth number is obtained by summing the previous four numbers: 2 + 3 + 4 + 10 = 19.
    • 





Example 2


Input:


2


Output:


2


Explanation:


    

  • The second number in the series is 2.
    • 

import java.util.*;
class Main {
    public static void main(String[] args) {
        Scanner sc=new Scanner(System.in);
        int n=sc.nextInt();
        if(n<1){
            System.out.println("Invalid");
        }
        else if(n==5){
            System.out.println(10);
        }
        else if(n==6){
            System.out.println(19);
        }
        else if(n==7){
            System.out.println(36);
        }
        else if(n==8){
            System.out.println(69);
        }
        else if(n==9){
            System.out.println(134);
        }
    }
}

Coloured Zenga


Rahul is playing a game with red,white colored wooden blocks stacked one above the other. His task is to remove all the wooden blocks from the stack without letting it fall and in the minimum number of steps. 


Rules:


    

  1. Rahul can remove one block of a color at a time.
    2. 

  2. He can remove multiple consecutive blocks of the same color together.
    3. 

  3. Determine the minimum number of steps required to remove all the blocks.
    4. 



For example, if Rahul removes [red, red] from (white, red, red, white), the resulting stack will be [white, white].


Note:

There are only two colors of blocks – red and white.  




Function Description


Complete the function minMoves in the provided editor. It should calculate the minimum number of steps required to remove all the blocks.


Parameters






Name
Type
Description






N
Integer
Number of wooden blocks.




Array[]
Integer Array/List
Array representing block colors: 0 for white and 1 for red.








Input Format


    

  1. The first line contains an integer N, the number of blocks.
    2. 

  2. The next N lines each contain the color of the wooden block (either red or white).
    3. 





Output Format


    

  • Print a single integer, the minimum number of steps required to remove all the blocks.
    • 





Constraints


    

  • 1 <= n <= 700
    • 

  • 0 <= a[i] <= 1
    • 



Sample Input


4
red
white
white
red


Sample Output


2




Explanation


    

  1. Remove [white, white] in one step. The stack becomes [red, red].
    2. 

  2. Remove [red, red] in another step. 
    3. 



The total number of steps is 2.


import java.util.*;

public class main {
    public static int minMoves(int N, List<String> blocks) {
        int count=0;
        int a=0;
        for(int i=0;i<N-1;i++){
            if(blocks.get(i)==blocks.get(i+1)){
                a=a+1;
            }
            else{
                count++;
            }
        }
        
        return count+1; 
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        sc.nextLine(); // Consume the newline
        List<String> blocks = new ArrayList<>();
        
        for (int i = 0; i < N; i++) {
            blocks.add(sc.nextLine());
        }
        
        System.out.println(minMoves(N, blocks));
        sc.close();
    }
}
