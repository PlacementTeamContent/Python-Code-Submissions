In a colorful candy shop, Sammy the confectioner had a jar filled with colorful candies, some red (R) and others green (G). Sammy wanted to make the candies look beautiful by ensuring they were alternated in colour, like "RGRGRG", "GRGRG" etc.
 To achieve this, Sammy could change any candy from red (R) to green (G) or vice versa with each operation. The challenge was to find the minimum number of operations needed to create the most beautiful candy sequence.


Your task is to help Sammy to find out the minimum number of operations needed to create the most beautiful candy sequence.





Input:


    

  • First line consists of an integer T-> denotes the number of string sequences to be given. 
    • 

  • Followed by T lines representing strings, each string denotes the sequence of candies.
    • 





Output: 



    

  • For each string sequence, print an Integer (space-separated) representing the minimum number of operations needed to create the most beautiful candy sequence.
    • 





Constraints:



    

  • 1 <=  T <= 100
    • 

  • 2 <= Length(S) <= 105
    • 

  • The string (S) consists only of characters R and G.
    • 






Example :- 
Input:


2
RRGR
GRGRG


Output:


1 0


In Sammy's colourful candy jar, there was a sequence of candies represented as RRGR.


    

  • To make this candy sequence beautiful, the minimum number of operations required is 1. In an operation, Sammy could change the first R candy at 0th index to G, resulting in the beautiful sequence GRGR.
So print 1.
    • 



For the second string GRGRG:


    

  • The string is already in the beautiful sequence so print 0
    • 

import java.util.Scanner;
import java.lang.Math;
class Main {
    public static void main(String[] args) { 
        // Write your code here
        Scanner sc = new Scanner(System.in);
        int n=sc.nextInt();
        //System.out.print(n);
        sc.nextLine();
        int b=0;
        
        while(b<n){
            String a=sc.nextLine();
            String rg="";
            String gr="";
            while(rg.length()<=a.length()){
                if(a.length()-rg.length()==1){
                    rg+="R";
                    break;
                }
                else{
                    if(a.length()==rg.length()){
                        break;
                    }
                    else{
                rg+="RG";
                    }
                }
                
            }
            while(gr.length()<=a.length()){
                if(a.length()-gr.length()==1){
                    gr+="G";
                    break;
                }
                else{
                    if(a.length()==gr.length()){
                        break;
                    }
                    else{
                        gr+="GR";
                    }
                }
            }
            int rgc=0;
            int grc=0;
            for(int i=0;i<a.length();i++){
                if(a.charAt(i)==rg.charAt(i)){
                    rgc++;
                }
                if(a.charAt(i)==gr.charAt(i)){
                    grc++;
                }
            }
            
            System.out.print(Math.min(rgc,grc)+" ");
            //System.out.println(grc);
            
            
            b++;
        }
    }
}

Your job is to find whether setA   is a subset of setB.


If setA   is subset of set B, print True.


If set A  is not a subset of setB , print False.




Input


    

  • 

    The first line will contain the number of test cases, T.
    

    • 

  • 

    The first line of each test case contains the number of elements in setA.
    

    • 

  • 

    The second line of each test case contains the space separated elements of setA.
    

    • 

  • 

    The third line of each test case contains the number of elements in setB.
    

    • 

  • The fourth line of each test case contains the space separated elements of setB.
    • 





Output


Output should contain True or False for each test case on separate lines.

1
2

3 6 5 4 1
7

Explanation


Here's the explanation for the provided test case:

Input:


5
1 2 3 5 6
9
9 8 5 6 3 2 1 4 7


Output:


True

1
2

False


    

  • In this test case, set A is {2} and set B is {3, 6, 5, 4, 1}. Set A has an element (2) that is not present in set B, so set A is not a subset of set B. The output is False.
