1.
Meatball Hiring Process


Bhojon is a restaurant company that has started a new wing in a city. They have every type of cook except for the meatball artist. They had fired their last cook because the sale of meatballs in their restaurant is really high, and they can’t afford to make meatballs again and again every time their stock gets empty. They have arranged a hiring program where you can apply with your meatball.


They will add the meatball in their seekh (a queue), and every time they cut the meatball, they take it and cut it into the day’s quantity, then re-add the meatball in the seekh. As a hiring manager, you are going to determine who will be hired.


Day’s Quantity


On that every day, the company sells only that kg of meatballs to every packet. 


If someone has less than the day’s quantity, it will be counted as a sell.




Input Format:


    

  • First line: Contains the integer N, which represents how many people are participating in the hiring process.
    • 

  • Second line: Contains the integer D, which represents the day’s quantity, i.e., how many grams of meatball are being sold to every packet.
    • 

  • Next N lines: Each line contains an integer representing the weight of meatballs each person has brought with them. These weights are provided in the form of an integer array Array[].
    • 





Output Format:


    

  • The 1-based index of the person whose meatball is served last.
    • 





Constraints:


    

  • 1 ≤ N ≤ 103
    • 

  • 1 ≤ D ≤ 103
    • 

  • 1 ≤ Array[i] ≤ 103
    • 





Sample Input 1:


4
2
7
8
9
3


Sample Output 1:


3


Explanation:


The seekh or meatball queue has the initial distribution: [7, 8, 9, 3]. 


    

  • On the first serving, they will cut 2 kg of meatball from the first meatball and add it to the last of the seekh. After the 1st serving, the queue becomes: [8, 9, 3, 5].
    • 

  • Then it proceeds as follows:
    • 

  • [9, 3, 5, 6],  
    • 

  • [3, 5, 6, 7],  
    • 

  • [5, 6, 7, 1],  
    • 

  • [6, 7, 1, 3],  
    • 

  • [7, 1, 3, 4],  
    • 

  • [1, 3, 4, 5],  
    • 

  • [3, 4, 5],  
    • 

  • [4, 5, 1],  
    • 

  • [5, 1, 2],  
    • 

  • [1, 2, 3],  
    • 

  • [2, 3],  
    • 

  • [3],  
    • 

  • [1],  
    • 

  • [0].
    • 



So, the last served meatball belongs to the 3rd person.

2.
Matched Socks Pairs


Alex works at a clothing store. There is a large pile of socks that must be paired by color for sale. Given an array of integers representing the color of each sock, determine how many pairs of socks with matching colors there are.


For example, there are n=7 socks with colors ar = {1,2,1,2,1,3,2}. There is one pair of color 1 and one of color 2. There are three odd socks left, one of each color. The number of pairs is 2.




Input Format


    

  • The first line contains an integer n, the number of socks represented in ar.
    • 

  • The second line contains n space-separated integers describing the colors ar[i] of the socks in the pile.
    • 





Output Format


    

  • Print the total number of matching pairs of socks that Alex can sell.
    • 





Constraints


    

  • 1 <= n <= 100
    • 

  • 1 <= ar[i] <= 100 and 0 <= i < n
    • 





Sample Input


9
10 20 20 10 10 30 50 10 20


Sample Output


3


Explanation


    

  • Alex can match 3 pairs of socks i.e. 10-10, 10-10, 20-20 while the left out socks are 50, 30, and 20.
    • 



import java.util.*;
class Main {
    public static void main(String[] args) {
        Scanner sc=new Scanner(System.in);
        int n=sc.nextInt();
        Map<Integer,Integer> hm=new HashMap<>();
        for (int i=0;i<n;i++){
            int x=sc.nextInt();
            hm.put(x,hm.getOrDefault(x,0)+1);
        }
        int ans=0;
        for(int i:hm.keySet()){
            ans+=hm.get(i)/2;
        }
        System.out.println(ans);
    }
}

Develop a program to calculate the count of triplets within a given sequence.

A Triplet consists of either three identical numbers or three consecutive numbers. 




Input


The first line of input contains two space separated integers N and M.
Subsequent lines will have N number of integers in separate lines i.e, (x1,x2,.....xN,).

Output


The output consists an integer representing the count of triplets.

Constraints


x1,x2,.....xN<= M 

Explanation


Let's take an example:


9 10
2
2
5
5
5
8
8
9
10


Here, the N subsequent elements are [2,2,5,5,5,8,8,9,10]. In this the same number triplet is [5,5,5] and the consecutive triplet is [8,9,10]. So, the count of the triplet is 2. 



import java.util.*;
 
public class main {

        Scanner sc=new Scanner(System.in);

            arr[i]=sc.nextInt();
        }
        int ans=0;
        for(int i=0;i<n-2;i++){
            int x=arr[i];
            if(arr[i]==arr[i+1] && arr[i]==arr[i+2])
            {
                ans+=1;
                i+=2;
            }
            else if(arr[i+1]==x+1 && arr[i+2]==x+2){
                ans+=1;
            }
        }
        System.out.println(ans);
    }
