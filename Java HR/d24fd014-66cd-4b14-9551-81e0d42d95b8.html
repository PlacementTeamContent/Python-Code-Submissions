Your task is to create a program that takes an integer array arr of numbers and a certain threshold number. Then, it compares each number in the array with the threshold. If a number is greater than the threshold, it goes into the "larger" group. If it's smaller, it goes into the "smaller" group.



- Don't consider the elements equal to the threshold.





Input Format:


    

  • First line of input contains an integer representing the size of the array.
    • 

  • Second line contains the elements of the array arr.
    • 

  • Third line determines the threshold  for which the elements are compared.
    • 





Output Format:


The program should print a string:


    

  • "larger" if there are more integers in array larger than threshold.
    • 

  • "smaller" if there are more integers in array smaller than threshold.
    • 

  • "equal" if the number of integers larger than threshold is the same as the number of integers less than threshold.
    • 





Constraints:


    

  • 1 <= Length of array <= 105 
    • 

  • -109 <= arr[i] <= 109
    • 

  • -109 <= threshold <= 109
    • 





Example:


Input:


6
1 3 0 -1 4 3
2


Output:


equal


Consider the array [1, 3, 0, -1, 4, 3] with a threshold of 2.


    

  • Count of integers greater than threshold:
    • 

  • 

    There are 3 integers (3, 4, and 3) greater than threshold 2.
    

    • 

  • 

    Count of integers less than threshold:
    

    • 

  • There are 3 integers (1, 0, and -1) less than threshold 2.
    • 



Since there are an equal number of integers greater and smaller than the threshold, the output is equal.
import java.util.Scanner;
import java.util.*;
 
public class main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int arr[] = new int[n];
        for(int i=0; i<n; i++) {
            arr[i] = sc.nextInt();
        }
        int threshold = sc.nextInt();
        int largeCount = 0, smallCount = 0;
        for(int i : arr) {
            if(i > threshold) {
                largeCount+=1;
            } else if (i<threshold) {
                smallCount+=1;
            } else {
                continue;
            }
        }
        System.out.println(largeCount > smallCount ? "larger": largeCount < smallCount ? "smaller" : "equal");
    }
}
