Printing Job


There is a printing shop where the customers submit their print jobs at regular intervals, and each job takes a fixed amount of time to complete. You are given an integer value N denoting the total
number of print jobs, and X denoting the time duration after which the next print job arrives.
Your task is to find and print an integer value representing how long the last job will have to wait in the queue considering one print
will take exactly 10 minutes. If in case, there is no waiting time then print 0




Input Format


    

  • The first line of input contains An integer value N representing the number of printing jobs.
    • 

  • The second  line of input containsAn integer value X representing the time duration after
which the next print job arrives.
    • 



Output Format


An integer value representing how long the last job will have to wait in the queue considering one print will take exactly 10 minutes. If in case, there is no waiting time then print 0.




Sample Input:


4
5


Sample Output:


15


Explanation:


Here, N = 4 and X = 5. Time to complete one print job it 10 mins. So, the time needed for three prints is 3*10 = 30. The last print job arrives at 15 minutes (3*5). So, the waiting time is 30 - 15 = 15.
Hence, 15 is printed as the output


import java.util.Scanner;

public class main {

    public static void main(String[] args) {
      Scanner sc = new Scanner(System.in);
      int jobs = sc.nextInt();
      int timeFrame = sc.nextInt();
      int timeNeeded = (jobs-1) * 10;
      
      if(jobs == 1){
          System.out.print(0);
      }else{
        int consumedTime = (jobs-1)*timeFrame;  
        int totalTime = timeNeeded - consumedTime;
        System.out.print(totalTime);
      }

Parking Lot


A parking lot in a mall has RxC number of parking spaces. Each parking space will either be empty (0) or full (1). The status (0/1) of a parking space is represented as the element of the matrix. The task is to find the index of the row (R) in the parking lot that has the most full parking spaces (1).





Input Format


    

  • The first line contains an integer R, representing the number of rows in the parking lot.
    • 

  • The second line contains an integer C, representing the number of columns in the parking lot.
    • 

  • The third line contains a list of R * C integers, representing the elements of the matrix M[R][C] where each element is either 0 (empty) or 1 (full).
    • 






Output Format


    

  • The output should be an integer representing the index of the row (R) with the maximum number of full parking spaces (1).
    • 






Constraints:


    

  • 1 ≤ R ≤ 10^3
    • 

  • 1 ≤ C ≤ 10^3
    • 

  • The elements of the matrix should be either 0 or 1.
    • 






Example 1:


Input:


3
3
0 1 0 1 1 0 1 1 1


Output:


3


Explanation:


    

  • Row 3 has the maximum number of 1’s (3 ones).
    • 

  • In case of a tie (multiple rows having the same maximum number of 1's), you should return the index of the first such row.
    • 



import java.util.*;

class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int rows = sc.nextInt();
        int cols = sc.nextInt();
        
        List<List<Integer>> slots = new ArrayList<>();
        
        for(int i =0; i < rows; i++){
            List<Integer> slot = new ArrayList<>();
            for(int j = 0; j < cols; j++){
                slot.add(sc.nextInt());
            }
            slots.add(slot);
        }
        
        int max = 0;
        int row = 0;
        int count = 0;
        
        for(List<Integer> l : slots){
            int sum = 0;
            count += 1;
            for(int i : l){
                sum = sum + i;
            }
            if(sum > max){
                max = sum;
                row = count;
            }
        }
        System.out.print(row);
    }
}
