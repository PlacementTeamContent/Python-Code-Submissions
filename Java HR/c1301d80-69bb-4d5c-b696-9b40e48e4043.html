Prime Sum Range


You will be given a left limit and a right limit. Your task is to calculate the sum of all prime numbers between a given range x and y, both inclusive.




Input Format


    

  • The first line contains an integer x representing the left limit.
    • 

  • The second line contains an integer y representing the right limit.
    • 



Output Format


    

  • Output a single integer which is the sum of all prime numbers in the given range.
    • 





Constraints


    

  • 1 <= x <= y <= 1000
    • 





Example 1


Input


30 
40




Output


68




Explanation


    

  • Prime numbers between 30 and 40 are 31 and 37. Their sum is 31 + 37 = 68.
    • 



import java.util.Scanner;
 
class Main {
    public static void main(String[] args) {
        //Write your code below
        Scanner sc=new Scanner(System.in);
        int x=sc.nextInt();
        int y=sc.nextInt();
        int sum=0;
        for(int i=x;i<=y;i++)
        {
            int c=0;
            for(int j=1;j<=i;j++)
            {
                if(i%j==0)
                    c++;
            }
            if(c==2)
                sum=sum+i;
        }
        System.out.println(sum);
    }
}

3.
You are given an integer num. Your task is to find if the given num is a perfect number or not.
 A perfect number is a positive integer that is equal to the sum of its proper positive divisors, i.e. the sum of its positive divisors excluding the number itself. 





Input Format:


    

  • The line contains an integer denoting num.
    • 





Output Format:


    

  • Print 1 if num is a perfect number, else print the sum of the proper divisors of num.
    • 





Constraints:


    

  • 0 < num < 109
    • 






Explanation


Let's take an example:
Input:


22


Output:


14


The proper positive divisors of 22 (except itself) are 1, 2, 11 and their sum = 14 which is not equal to 22, hence 22 is not a perfect number. So the output is 14 which is the sum of proper positive divisors of 22.
import java.util.*;
class Main {
    public static void main(String[] args) { 
        // Write your code here
        Scanner sc=new Scanner(System.in);
        int num=sc.nextInt();
        int sum=1;
        for(int i=2;i<num;i++)
        {
            if(num%i==0)
                sum=sum+i;
        }
        if(num==sum)
            System.out.println(1);
        else 
            System.out.println(sum);
    }
}
