Rajeev teaches students for fun and free. In one of his classes, he decided to give a problem to the students  which will improve their ability to solve conditional problems. Rajeev writes a word Sequence on the board. Rajeev wants students to form groups and suggest a new subSequence.


The new subSequence must satisfy the following conditions.


    

  1. It should be the smallest subsequence of Sequence.
    2. 

  2. Sequence should only contain character x y& z.
    3. 

  3. Length of subSequence should be equal or greater than 2.
    4. 

  4. Number of occurrences of y should be greater than
occurrences z.
    5. 

  5. Number of occurrences of y should be greater than
occurrences x.
    6. 



The student's group must tell the minimum length of the subSequence which satisfies the above condition. 

1. If no such subsequence exists, then can convey -1.
2. A subsequence is a contiguous sequence of characters with in a word.


Input


The input contains string representing Sequence.

Output


The output contains integer representing minimum length of subSequence.

Explanation


Let's take an example:


xyyz


The subSequence "yy" of Sequence follows all the conditions provided. It can be verified that no minimum subSequence formed.
Hence the length of "yy" is 2.
import java.util.Scanner;
 
public class main {
    public static void main(String[] args) {
        //Write your code below
        Scanner sc=new Scanner(System.in);
        String s=sc.next();
        int minlen=Integer.MAX_VALUE;
        int n=s.length();
        for (int i=0;i<n ;i++ ){
            int xcount=0;
            int ycount=0;
            int zcount=0;
            for(int j=i;j<n;j++){
                char ch=s.charAt(j);
                if (ch=='x'){
                    xcount++;
                } 
                else if(ch=='y'){
                    ycount++;
                }
                else if(ch=='z'){
                    zcount++;
                }
                else{
                    continue;
                }
                int len=j-i+1;
                if(len>=2 && ycount>xcount && ycount>zcount){
                    minlen=Math.min(minlen,len);
                    break;
                }
            }
        } 
        System.out.println(minlen==Integer.MAX_VALUE?-1:minlen);
    }
}

Diagonal Sum Difference


Given a square matrix, calculate the absolute difference between the sum of its diagonals.




Input Format


    

  • The first line contains an integer n, the size of the matrix.
    • 

  • The next n lines each contain n space-separated integers representing the elements of the matrix.
    • 



Output Format


    

  • Output a single integer, the absolute difference between the sum of the matrix's two diagonals.
    • 





Constraints


    

  • 1 <= n <= 100
    • 

  • -100 <= matrix[i][j] <= 100
    • 





Example 1


Input


3
1 2 3
4 5 6
9 8 9




Output


2




Explanation


The primary diagonal of the given 3x3 matrix is 1, 5, 9. The sum of the primary diagonal is 1 + 5 + 9 = 15.
The secondary diagonal of the given 3x3 matrix is 3, 5, 9. The sum of the secondary diagonal is 3 + 5 + 9 = 17.
The absolute difference between the sums is |15 - 17| = 2

import java.util.Scanner;

public class main {
    public static void main(String[] args) {
        // Make your logic here
        Scanner sc=new Scanner(System.in);
        int n=sc.nextInt();
        int[][] arr=new int[n][n];
        for (int i=0;i<n ;i++){
            for(int j=0;j<n;j++){
                arr[i][j]=sc.nextInt();
            }
        } 
        int d1=0,d2=0;
        for(int i=0;i<n;i++){
            for(int j=0;j<n;j++){
                if (i==j){
                    d1+=arr[i][j];
                } 
                if(j==n-i-1){
                    d2+=arr[i][j];
                }
            }
        }
        System.out.println(Math.abs(d1-d2));
    }
}
