You are given an array consisting of M positive integers, and your task is to sort the array in decreasing order of count of set bits in the binary representation of the integers present in the array.


The number of set bits is nothing but the number of 1's present in the binary representation of the integer.
 For example, the number of set bits in 7(Binary -> 0111) is 3.




    

  1. If any two numbers have the same count of set bits, then in the sorted array they will appear in the order in which they appear in the original array. .
    2. 

  2. The array may contain duplicate elements.

    3. 





Input Format:


    

  • The first line consists of an Integer M denoting the size of the array.
    • 

  • The second line consists of M space-separated Integers representing the elements of the sequence.
    • 





Output

- Print the sorted sequence of integers based on the descending order of set bits count.





Constraints

- 1 <= M <= 105
- 1 <= array[i] <= 106





Explanation:


Input:


3
5 7 6


Output:


7 5 6


The array is { 5, 7, 6}, in this case,
5 -> 0101
6 -> 0110
7 -> 0111
both 5 and 6 have the same number of set bits so the answer will be {7, 5, 6} and not {7, 6, 5}, because in the original array 5 appears before 6



import java.util.*;

class Main {
    public static int countNoOfOnesInBinary(int num){
        int count_ones = 0;
        while(num > 0){
            int rem = num % 2;
            if(rem == 1){
                count_ones++;
            } 
            num = num / 2;
        }
        return count_ones;
    }
    public static void main(String[] args) { 
        // Write your code here
        Scanner sc = new Scanner(System.in);
        int n=sc.nextInt();
        int arr[]=new int[n];
        for(int i=0; i<n; i++){
            arr[i]=sc.nextInt();
        }
        
        String res="";
        int max  = 0;
        for(int i=0; i<n; i++){
            int count_ones_in_binary = countNoOfOnesInBinary(arr[i]);
            if(count_ones_in_binary > max){
                res = String.valueOf(arr[i]) + " "+ res;
            }
            else{
                res = res+" "+String.valueOf(arr[i]);
            }
            max = count_ones_in_binary;
        }
        
        res=res.replace("  ", " ");
        
        System.out.println(res);
    }
}

Number Patterns


You need to print a pattern of numbers in a specific format. The pattern consists of multiple rows.


The numbers should be printed consecutively, starting from 1.




Input Format:


    

  • The input consists of a single integer n, which represents the number of rows to print the pattern.
    • 





Output Format:


    

  • The output consists of n rows.
    • 

  • Each row contains numbers starting from 1 and separated by an asterisk (*).
    • 

  • The number of elements in each row increases by 1 in each subsequent row.
    • 





Constraints:


    

  • 1 ≤ n ≤ 100
    • 





Sample Input:


5


Sample Output:


1
23
456
78910
1112131415


Explanation:


    

  • The first row prints 1.
    • 

  • The second row prints the numbers 2 and 3 separated by *.
    • 

  • The third row prints 4, 5, and 6 separated by *.
    • 

  • The fourth row prints 7, 8, 9, and 10 separated by *.
    • 

  • The fifth row prints 11, 12, 13, 14, and 15 separated by *.
    • 



In general, for the i-th row (1-based index), the row should print i numbers starting from the last number printed plus 1.


import java.util.*;

class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n=sc.nextInt();
        
        int count = 1;
        for(int i=1; i<=n; i++){
            for(int j=0; j<i; j++){
                System.out.print(count);
                if(j<i-1){
                    System.out.print("*");
                }
                count++;
            }
            System.out.println();
        }
    }
}
