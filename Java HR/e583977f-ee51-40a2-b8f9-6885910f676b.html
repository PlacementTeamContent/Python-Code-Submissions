Two strings are said to be similar if they are composed of the same characters. For example, abababc and accccbaa are similar since both of them are composed of characters a, b and c. However, cabca and badcd are not similar since they do not share all of the same letters.


Given an array of strings strArr of length m, find the number of pairs of strings that are similar.



Keep in mind :


    

  1. Each string is composed of lowercase English characters only.
    2. 

  2. Pairs are considered index-wise, i.e, two equal strings at different indices are counted as separated pairs
    3. 

  3. A pair of indices (i,j) is same as the pair (j,i);

    4. 



Your task is to provide an efficient algorithm to find the number of pairs of strings that are similar.
Input
• The first line consists of an Integer M denoting size of string array strArr.
• The second line consists of M space separated Strings representing the elements of strArr array.
Output
• Print an Integer denoting count the number of pairs of strings that are similar in strArr array.
Constraints
• 1 <= M <= 10^4
 • 'a'<= strArr[i][j]<= 'z'
Example :
Let's take an example:


3
xyz foo of


Here, the strings "foo" and "of" are similar because they are composed of the same characters ['o', 'f']. There are no other similar pairs so the answer is 1.
import java.util.*;
class Main {
    static boolean fun(String s,String t){
       Set<Character> st1 = new HashSet<>();
       Set<Character> st2 = new HashSet<>();
       for(char c: s.toCharArray()){
           st1.add(c);
       }
       for(char c: t.toCharArray()){
           st2.add(c);
       }
       return st1.equals(st2);
    }
    public static void main(String[] args) { 
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int cnt = 0;
        String arr[] = new String[n];
        for(int i=0;i<n;i++){
           arr[i] = sc.next();
        }
        for(int i=0;i<n;i++){
           for(int j=i+1;j<n;j++){
              if(fun(arr[i],arr[j])){
                   cnt++;
              }
           }
        }
        System.out.print(cnt);
    }
}

import java.util.*;

public class main {

    public static void main(String[] args) {
      Scanner sc = new Scanner(System.in);
      int n = sc.nextInt();
      ArrayList<Boolean> arr = new ArrayList<>(Collections.nCopies(n,false));
      int num = 1;
      while(num<=n){
          for(int i=0;i<n;i++){
              if((i+1)%num==0){
                  arr.set(i,!arr.get(i));
              }
          }
          num++;
      }
      for(int i=0;i<n;i++){
          System.out.print(arr.get(i)?"ON ":"OFF ");
      }
    }
}

Printing Job


There is a printing shop where the customers submit their print jobs at regular intervals, and each job takes a fixed amount of time to complete. You are given an integer value N denoting the total
number of print jobs, and X denoting the time duration after which the next print job arrives.
Your task is to find and print an integer value representing how long the last job will have to wait in the queue considering one print
will take exactly 10 minutes. If in case, there is no waiting time then print 0




Input Format


    

  • The first line of input contains An integer value N representing the number of printing jobs.
    • 

  • The second  line of input containsAn integer value X representing the time duration after
which the next print job arrives.
    • 



Output Format


An integer value representing how long the last job will have to wait in the queue considering one print will take exactly 10 minutes. If in case, there is no waiting time then print 0.




Sample Input:


4
5


Sample Output:


15


Explanation:


Here, N = 4 and X = 5. Time to complete one print job it 10 mins. So, the time needed for three prints is 3*10 = 30. The last print job arrives at 15 minutes (3*5). So, the waiting time is 30 - 15 = 15.
Hence, 15 is printed as the output


import java.util.Scanner;

public class main {

    public static void main(String[] args) {
      Scanner sc = new Scanner(System.in);
      int n = sc.nextInt();
      int m = sc.nextInt();
      if(n<=1 || m>10){
          System.out.println(0);
      }else{
          int last = n-1;
          last = last * 10 - m*last;
          System.out.println(last);
      }
    }
}
