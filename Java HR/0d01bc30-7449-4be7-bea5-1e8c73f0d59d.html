Find nth Number In Series


You want to find the nth number in a series using the following parameters:


    

  • The first four numbers in the series are always set as n1=1, n2=2, n3=3, n4=4.
    • 

  • Subsequent numbers in the series are obtained by summing the four preceding numbers.
    • 



Write a program that takes an integer n as input and print the value of the nth number in the series.


If the input n is less than 1, your function should print the string Invalid.




Input format


    

  • A single integer n representing the position of the number in the series to find.
    • 





Output format


    

  • 

    A single integer nth position in the series.
    

    • 

  • 

    If n < 1, print: Invalid
    

    • 





**Constraints: **


    

  • 1 ≤ n ≤ 50
    • 





Example 1


Input:


6


Output:


19


Explanation:


    

  • The first four numbers in the series are always 1, 2, 3, 4. The fifth number in the series is obtained by summing the previous four numbers: 1 + 2 + 3 + 4 = 10. Similarly, the sixth number is obtained by summing the previous four numbers: 2 + 3 + 4 + 10 = 19.
    • 





Example 2


Input:


2


Output:


2


Explanation:


    

  • The second number in the series is 2.
    • 

import java.util.Scanner;
class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        
        if(n<1){
            System.out.println("Invalid");
            return;
        }
        
        int[] series = new int[51];
        series[1] = 1;
        series[2] = 2;
        series[3] = 3;
        series[4] = 4;
        
        for (int i=5;i<=n;i++){
            series[i] = series[i-1] + series[i-2] + series[i-3] + series[i-4];
        }
        System.out.println(series[n]);
    }
}

The Clock Problem


Your friend has problem a reading the time if a clock follows 24-hour format.So you decide to help him out .You have two integers X and Y.Your task is to find and print an integer value to representing  product of these two integers in the 12-hour system.




Input Format


    

  • First line of input contains an integer X.
    • 

  • Second line of input contains an integerY.
    • 



Output Format


    

  • Output an integer value representing the product of the two integers in the 12-hour system.
    • 





Sample Input:


4
5


Sample Output:


8


Explanation:


Here, X = 4 and Y = 5, and their product is 4*5=20. Therefore, 8 is printed as the output in 12-hour system.


import java.util.Scanner;

public class main {

    public static void main(String[] args) {
      //Write your code here
      Scanner sc = new Scanner(System.in);
      int x=sc.nextInt();
      int y=sc.nextInt();
      
      int product = x*y;
      int hour = product % 12;
      if (hour == 0){
          hour = 12;
      }



      System.out.println(hour);
      
    }
}

Number Patterns


You need to print a pattern of numbers in a specific format. The pattern consists of multiple rows.


The numbers should be printed consecutively, starting from 1.




Input Format:


    

  • The input consists of a single integer n, which represents the number of rows to print the pattern.
    • 





Output Format:


    

  • The output consists of n rows.
    • 

  • Each row contains numbers starting from 1 and separated by an asterisk (*).
    • 

  • The number of elements in each row increases by 1 in each subsequent row.
    • 





Constraints:


    

  • 1 ≤ n ≤ 100
    • 





Sample Input:


5


Sample Output:


1
23
456
78910
1112131415


Explanation:


    

  • The first row prints 1.
    • 

  • The second row prints the numbers 2 and 3 separated by *.
    • 

  • The third row prints 4, 5, and 6 separated by *.
    • 

  • The fourth row prints 7, 8, 9, and 10 separated by *.
    • 

  • The fifth row prints 11, 12, 13, 14, and 15 separated by *.
    • 



In general, for the i-th row (1-based index), the row should print i numbers starting from the last number printed plus 1.


import java.util.Scanner;
class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        
        int num=1;
        
        for(int i=1;i<=n;i++){
            for(int j=1;j<=i;j++){
                System.out.print(num);
                if (j!=i){
                    System.out.print("*");
                }
                num++;
            }
            System.out.println();
        }
    }
}
