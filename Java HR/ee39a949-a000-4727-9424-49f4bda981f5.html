1.
Minimum Time To Scan


Vivek has two scanners in his shop to scan the documents of a customer. When a customer comes to his shop, they give x number of documents to scan. Vivek knows T1 and T2, the time taken by scanner1 and scanner2 to scan a single document, respectively. Your task is to write a program to calculate the minimum time required to scan all the x documents using the two scanners, where the documents can be distributed optimally between the two scanners.




Input Format:


    

  • A single line containing three space-separated integers: x, T1, and T2:
    • 

  • x is the number of documents to be scanned.
    • 

  • T1 is the time taken by scanner1 to scan a single document.
    • 

  • T2 is the time taken by scanner2 to scan a single document.
    • 





Output Format:


    

  • A single integer representing the minimum time required to scan all the x documents using the two scanners.
    • 





Constraints:


    

  • x, T1, and T2 are positive integers.
    • 





Example


Input:


5 2 3


Output:


6


Explanation:


Case 1:


    

  • Scanner 1 scans 0 documents, Scanner 2 scans 5 documents.
    • 

  • Time for Scanner 1: 0 * 2 = 0
    • 

  • Time for Scanner 2: 5 * 3 = 15
    • 

  • Total time = max(0, 15) = 15
    • 



Case 2:


    

  • Scanner 1 scans 1 document, Scanner 2 scans 4 documents.
    • 

  • Time for Scanner 1: 1 * 2 = 2
    • 

  • Time for Scanner 2: 4 * 3 = 12
    • 

  • Total time = max(2, 12) = 12
    • 



Case 3:


    

  • Scanner 1 scans 2 documents, Scanner 2 scans 3 documents.
    • 

  • Time for Scanner 1: 2 * 2 = 4
    • 

  • Time for Scanner 2: 3 * 3 = 9
    • 

  • Total time = max(4, 9) = 9
    • 



Case 4:


    

  • Scanner 1 scans 3 documents, Scanner 2 scans 2 documents.
    • 

  • Time for Scanner 1: 3 * 2 = 6
    • 

  • Time for Scanner 2: 2 * 3 = 6
    • 

  • Total time = max(6, 6) = 6
    • 



Case 5:


    

  • Scanner 1 scans 4 documents, Scanner 2 scans 1 document.
    • 

  • Time for Scanner 1: 4 * 2 = 8
    • 

  • Time for Scanner 2: 1 * 3 = 3
    • 

  • Total time = max(8, 3) = 8
    • 



Case 6:


    

  • Scanner 1 scans all 5 documents, Scanner 2 scans 0 documents.
    • 

  • Time for Scanner 1: 5 * 2 = 10
    • 

  • Time for Scanner 2: 0 * 3 = 0
    • 

  • Total time = max(10, 0) = 10
    • 



The minimum time to scan all the documents is 6 (from Case 4 where Scanner 1 scans 3 documents and Scanner 2 scans 2 documents).


import java.util.*;
class Main {
    public static void main(String[] args) {
        
        Scanner sc=new Scanner(System.in);
        int x=sc.nextInt();
        int t1=sc.nextInt();
        int t2=sc.nextInt();
        int minTime=Integer.MAX_VALUE;
        for(int i=0;i<=x;i++)
        {
            int time1=i*t1;
            int time2=(x-i)*t2;
            int totalTime=Math.max(time1,time2);
            minTime=Math.min(minTime,totalTime);
        }
        System.out.println(minTime);
    }
}

Write a script that converts a given text with various punctuation into camel case format. The output should be a single line of text formatted in camel case.

Camel case is a way of writing words without spaces or punctuation. Each word starts with a capital letter except the first one. For example, hello world in camel case is helloWorld.





Sample Input


    

  • The input contains a string representing the text.
    • 





Sample Output


    

  • The output is the camel case format for the given text.
    • 






Constraints


    

  • 0 <= length of text <= 10000
    • 





Example


If BOB loves-coding then your program should return bobLovesCoding


import java.util.*;
 class Main {
    public static String CamelCase(String sentence) {
        // Write your code here.....
        String words[]=sentence.split("[^a-zA-Z0-9]+");
        StringBuilder sb=new StringBuilder();
        for(int i=0;i<words.length;i++)
        {
            String word=words[i].toLowerCase();
            if(i==0)
            {
                sb.append(word);
            }
            else{
                sb.append(Character.toUpperCase(word.charAt(0))).append(word.substring(1));
                
            }
        }

        return sb.toString();
    }

    // Don't edit the code below....
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String sentence = scanner.nextLine();
        String result = CamelCase(sentence);
        System.out.println(result);
    }
}
