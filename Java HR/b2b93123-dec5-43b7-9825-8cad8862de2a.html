Count of Unique Student IDs


A Monitor of the class wanted to take the attendance, so he circulated a blank page in the classroom asking students to write there student_Id. But some naughty students wrote their id multiple times to trouble the monitor. Monitor went to the teacher and reported the incident, the Teacher asked him to mark the absence of students whose id is repeated in the attendance sheet and give him the number of students with repeated id in the attendance sheet.


Note: Student id is nothing but an alphabet from a-z.




Input Format


    

  • The input consists of a string representing the ids of students.
    • 



Output Format


    

  • Print an integer representing the count of characters that are not repeated in the string.
    • 





Sample Input:


ghygbvghyghnhjuhjumnj


Sample Output:


3


Explanation:


In the given string, the student IDs that do not repeat are:

- b, v, and m (3 unique IDs).


The rest of the IDs appear multiple times. Hence, the output is 3.


import java.util.*;

public class main {

    public static void main(String[] args) {
      //Write your code here
      Scanner sc = new Scanner(System.in);
      String s = sc.nextLine();
      HashMap<Character, Integer> map = new HashMap<>();
      for(int i=0; i<s.length(); i++){
          if(map.containsKey(s.charAt(i))){
              map.put(s.charAt(i), map.get(s.charAt(i))+1);
          } else{
              map.put(s.charAt(i), 1);
          }
      }
      int count=0;
      for(Integer c: map.values()){
          if(c==1){
              count++;
          }
      }
      System.out.print(count);
    }
}

Borrow Number


You have two numbers number1 and number2. Your task is to check the number of borrow operations needed for the subtraction of number1 from number2. If the subtraction is not possible, print the string "Not possible".




Input Format


    

  • Two integers number1 and number2.
    • 



Output Format


    

  • Print the number of borrow operations if the subtraction is possible.
    • 

  • Print the string "Not possible" if the subtraction is not possible.
    • 





Constraints


    

  • number1 and number2 are positive integers.
    • 





Sample Input 0:


754
658


Sample Output 0:


2


Explanation:


    

  • First iteration (rightmost digit): 4 - 8 requires borrow
    • 

  • Second iteration: 5 - 5 (after first borrow) requires another borrow
    • 

  • Total borrow operations: 2
    • 



Sample Input 1:


654
666


Sample Output 1:


Not possible


Explanation:


    

  • the subtraction is not possible because 666 is greater than 654.
    •
