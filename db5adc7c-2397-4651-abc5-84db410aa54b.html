Username Encoding


You are given a username string U of length N. The username needs to be encoded before storing it in the system. The username string can be encoded using the following rules:


    

  • If the k-th character in U is a vowel, change it to the third next character in the alphabetical sequence. For example, the third next character of o is r.
    • 

  • If the k-th character in U is a consonant, change it to the third previous character in the alphabetical sequence. For example, the third previous character of h is e.
    • 



Implement the function encode which will take username string U as argument and return encoded string.



1) The third previous character of c is z, b is y.



Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing the username string.
    • 





Output Format


    

  • Single line containing the encoded username string.
    •

Input


abc




Output


dyz




Explanation


a is vowel so change to d

b is consonant so change to y

c is consonant so change to z

So, encoded username for abc will be dyz.

m is consonant so change to j

def encode(U):
    r=""
    vowels="aeiou"
    for i in U:
        if i in vowels:
            r+=chr(ord(i)+3)
        else:
            if i=='c':
                r+='z'
            elif i=='b':
                r+='y'
            else:
                r+=chr(ord(i)-3)
    return r

Sum of Matching Squares


For a list of integers nums, nums[i] is said to be Matching Square if the square of nums[i] is equal to the square of i.


Implement the function sum_matching_squares which will take n, size of list and nums as argument and return an integer, the sum of all matching square numbers in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of all matching square numbers.
    • 





Example 1


Input


3
0 1 2




Output


3




Explanation


    

  • For i = 0, nums[0] = 0, 00 = 0 and ii = 0. Matching square.
    • 

  • For i = 1, nums[1] = 1, 11 = 1 and ii = 1. Matching square.
    • 

  • For i = 2, nums[2] = 2, 22 = 4 and ii = 4. Matching square.
    • 



Sum of matching squares: 0 + 1 + 2 = 3.




Constraints


    

  • 1 <= nums.size() <= 1000
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def sum_matching_squares(n, nums):
    s=0
    for i in range(n):
        if nums[i]**2==i*i:
            s+=nums[i]
    return s

Output Format
