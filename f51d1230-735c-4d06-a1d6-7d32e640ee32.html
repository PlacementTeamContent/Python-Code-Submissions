def modify_code(M):
    vowels = "aeiou"
    alpha = "abcdefghijklmnopqrstuvwxyz"
    s = ""
    for i in M:
        if i in vowels:
            index = alpha.index(i) + 1 
            s += alpha[index+3]
        else:
            index = alpha.index(i) + 1 
            s += alpha[index-5]
    return s

Total Population Growth


Implement a function total_population to determine the total population of a city after T years, starting at an initial population of P. The population grows at a constant rate of R individuals per year.


Function total_population will take P, R, and T as first, second, and third arguments respectively and return the total population after T years.


Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing 3 space-separated integers representing P, R, and T respectively.
    • 





Output Format


    

  • Single line having 1 integer representing the total population after T years.
    • 





Example 1


Input


1000 200 5



Output


2000



Explanation


Initial population is 1000.
At time 1 year, total population will be 1000+200 = 1200.
At time 2 years, total population will be 1200+200 = 1400.
At time 2 years, total population will be 1400+200 = 1600.
At time 2 years, total population will be 1600+200 = 1800.
At time 5 years, total population will be 1800+200 = 2000.




Constraints


    

  • 0 <= P, R, T < 10000
    • 



def total_population(P, R, T):
    for i in range(T):
        P += R 
    return P

String Similarity


Two strings are said to be similar if they are composed of the same characters. For example, abababc and accccbaa are similar since both of them are composed of characters a, b and c. However, cabca and badcd are not similar since they do not share all of the same letters.

1. Each string is composed of lowercase English alphabets only
2. Pairs are considered index-wise, i.e. two same strings at different indices are counted as separated pairs
3. A pair of indices (i,j) is same as the pair (j,i)





Input Format

    • 

  • The second line consists of n space separated strings representing the elements of strArr array.
    • 



Output Format

Constraints

    • 

  • a <=strArr[i][j]<= z
    • 





Example


Let's take an example:


3
xyz foo of
