Serial Number Transformation


You are given a serial number string S of length N. The serial number needs to be transformed before use. The serial number string can be transformed using the following rules:


    

  • If the k-th character in S is a vowel, change it to the fifth next character in the alphabetical sequence. For example, the fifth next character of o is t.
    • 

  • If the k-th character in S is a consonant, change it to the fifth previous character in the alphabetical sequence. For example, the fifth previous character of h is c.
    • 



Implement the function transform_serial which will take serial number string S as argument and return transformed string.



1) The fifth previous character of d is y, c is x, b is w.



Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing the serial number string.
    • 





Output Format


    

  • Single line containing the transformed serial number string.
    • 





Example 1


Input


xyz



Output


stu



Explanation


x is consonant so change to s

y is consonant so change to t

z is consonant so change to u

So, transformed serial number for xyz will be stu.


Example 2


Input


violet



Output


qntgjo



Explanation


v is consonant so change to q

i is vowel so change to n

o is vowel so change to t

l is consonant so change to g

e is vowel so change to j

t is consonant so change to o

So, transformed serial number for violet will be qntgjo.




Constraints


    

  • 1 <=N<= 106
    • 

  • The serial number string S contains only lowercase alphabets.
    • 



def transform_serial(S):
    vowels = "aeiou"
    alphabets = list("abcdefghijklmnopqrstuvwxyz")
    string = ""
    for i in S:
        if i in vowels:
            index = alphabets.index(i)
            index = index+5
            string += alphabets[index]
        else:
            index = alphabets.index(i)
            index = index-5
            string += alphabets[index]
    return string

Good Number Sum


For a list of integers nums, nums[i] is said to be Good Number if parity of i and nums[i] will be same.


Implement the function sum_good_number which will take n, size of list and nums as argument and return an integer, sum of good number in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of good numbers.
    • 





Example 1


Input


3
1 2 3



Output


0



Explanation


    

  • For i = 0, nums[0] = 1, 1 % 2 = 1 and i % 2 = 0. Not a good number.
    • 

  • For i = 1, nums[1] = 2, 2 % 2 = 0 and i % 2 = 1. Not a good number.
    • 

  • For i = 2, nums[2] = 3, 3 % 2 = 1 and i % 2 = 0. Not a good number.
    • 



Hence, the answer is 0.


Example 2


Input


4
2 3 4 5



Output


14



Explanation


    

  • For i = 0, values[0] = 2, 2 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 1, values[1] = 3, 3 % 2 = 1 and i % 2 = 1. Good number.
    • 

  • For i = 2, values[2] = 4, 4 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 3, values[3] = 5, 5 % 2 = 1 and i % 2 = 1. Good number.
    • 



Sum of elements added: 2 + 3 + 4 + 5 = 14.
Hence, the answer is 14.




Constraints


    

  • 1 <=nums.size()<= 106
    • 

  • 1 <=nums[i]<= 1000
    • 

  • Index starts from 0
    • 



def sum_good_number(n, nums):
    count = 0
    for i in range(n):
        if i%2 == nums[i]%2:
            count += nums[i]
    return count

Alternating Sum


Write a program that traverses an array from left to right, where in each step it examines a sub-array of size k and skips m positions forward after each examination. Your task is to print the alternating sum of values found in each sub-array considered.


Alternating sum of an array can be calculated by adding first value subtracting second and so on.


For example: arr = [a1, a2, a3, a4, a5]
alternating sum of arr will be: (a1-a2+a3-a4-a5)



Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.





Input Format


    

  • The first line contains n size of array.
    • 

  • The second line contains space-separated integers representing the elements of the array arr.
    • 

  • The third line contains an integer k, indicating the size of each sub-array to be considered.
    • 

  • The fourth line contains an integer m, specifying the number of elements to skip after processing each sub-array.
    • 





Output Format


    

  • The output should be a single line consisting of space-separated values, each representing the alternating sum found in the respective sub-array.
    • 





Constraints


    

  • The number of elements in the array arr is greater than zero.
    • 

  • Sub-array size k and the number of skips m are positive integers.
    • 

  • k is less than or equal to the length of the array.
    • 





Example


Let's take a sample input:


8
4 7 3 9 6 1 8 0
3
1



We need to print:


0 13 0 4 13 -7



Explanation


[4 7 3] 9 6 1 8 0 -> alternating sum is 4-7+3 = 0.
4 [7 3 9] 6 1 8 0 -> moving right by 1 place, alternating sum is 7-3+9 = 13.
4 7 [3 9 6] 1 8 0 -> moving right by 1 place, alternating sum is 3-9+6 = 0.
4 7 3 [9 6 1] 8 0 -> moving right by 1 place, alternating sum is 9-6+1 = 4.
4 7 3 9 [6 1 8] 0 -> moving right by 1 place, alternating sum is 6-1+8 = 13.
4 7 3 9 6 [1 8 0] -> moving right by 1 place, alternating sum is 1-8+0 = -7.


#Write your code here
N = int(input())
Arr = input().split()
Arr = list(map(int, Arr))
Size = int(input())
Skip = int(input())
list_A = []
I = 0
for i in range(len(Arr)-Size+1):
    count = 0
    if i==0 or i==I:
        for j in range(Size):
            if j%2==0:
                count += Arr[i+j]
            else:
                count -= Arr[i+j]
        list_A += [str(count)]
        I += Skip
print(" ".join(list_A))

Toggle Switches


In this task, there are N switches that are initially turned off. You first turn on all the switches, then turn off every second switch. On the third round, you toggle every third switch (turning on if it's off or turning off if it's on).


For the jth round, you toggle every jth switch. For the Nth round, you only toggle the last switch.


Print the number of switches that are on after N rounds.




Input Format


    

  • A single integer N
    • 



Output Format


    

  • A single integer representing the number of switches that are on after N rounds.
    • 





Constraints


    

  • 1 <= N <= 1000
    • 





Sample Input 1


4



Sample Output 1


2



Explanation


Let's take an example:


4



    

  • At first, the four switches are [off, off, off, off].
    • 

  • After the first round, the four switches are [on, on, on, on].
    • 

  • After the second round, the four switches are [on, off, on, off].
    • 

  • After the third round, the four switches are [on, off, off, off].
    • 

  • After the fourth round, the four switches are [on, off, off, on].
    • 

  • So you should print 2 because there are two switches that are on.
    • 



#Write your code here
N = int(input())
list_A = []
for i in range(N):
    list_A += ["off"]
for j in range(1,N+1):
    for k in range(1,len(list_A)+1):
        if k%j == 0 :
            if list_A[k-1] == "off":
                list_A[k-1] = "on"
            else:
                list_A[k-1] = "off"
print(list_A.count("on"))
