Encapsulated Student Class


Implement a class Student that encapsulates its attributes and provides public methods to interact with them.


Class Description


Student


    

  • Private Attributes:
    • 

  • __name – The student's full name.
    • 

  • __roll_number – A unique identifier for the student.
    • 

  • __grade – The grade level of the student.
    • 

  • 

    __marks – The student's marks.
    

    • 

  • 

    Public Methods:
    

    • 

  • get_name() – Returns the student's name.
    • 

  • set_name(name) – Updates the student's name.
    • 

  • get_roll_number() – Returns the student's roll number.
    • 

  • get_grade() – Returns the student's grade.
    • 

  • set_grade(grade) – Updates the student's grade.
    • 

  • get_marks() – Returns the student's marks.
    • 

  • set_marks(marks) – Updates the student's marks. If marks are negative, print "Invalid Marks".
    • 

  • display_info() – Prints the student's details in the following format:
    "Student [roll_number]: [name], Grade: [grade], Marks: [marks]"
    • 




1. The init method should initialize all private attributes using parameters: name, roll_number, grade, and marks.
2. Ensure that mark updates enforce the non-negative condition.
3. Testing will verify that attribute encapsulation is properly implemented and that getter and setter methods work as specified.

class Student:
    def __init__(self, name, roll_number, grade, marks):
        self.__name = name 
        self.__roll_number = roll_number 
        self.__grade = grade 
        self.__marks = marks 
        
    def get_name(self):
        return self.__name  
        
    def set_name(self, name):
        name = self.__name 
        
    def get_roll_number(self):
        return self.__roll_number  
        
    def get_grade(self):
        return self.__grade
        
    def set_grade(self, grade):
        grade = self.__grade 
        
    def get_marks(self):
        return self.__marks
        
    def set_marks(self,marks):
        marks = self.__marks
        if marks < 0 :
            print("Invalid Marks") 
            
    def display_info(self):
        print("Student {}: {}, Grade: {}, Marks:{}".format(self.__roll_number, self.__grade, self.__marks))

class Movie:
    def __init__(self, title, director, year, genre):
        self.title = title 
        self.director = director 
        self.year = year 
        self.genre = genre 
        
       
    
    def get_info(self):
        return "{} directed by {}, released in {}".format(self.title, self.director, self.year)
        
    def mark_as_watched(self):
        self.watched = True 
        print("You have watched {}".format(self.title)) 
    
    def change_genre(self,new_genre):

        print("Genre updated to {} for {}".format(new_genre, self.title))
