def encode(P) :
    res =""
    string_alpha = "abcdefghijklmnopqrstuvwxyz"
    for i in P:
        if i.lower() in "aeiou":
            X = string_alpha.index(i.lower())+2
            res += string_alpha[X]
        else:
            X = string_alpha.index(i.lower())-2
            res+=string_alpha[X]
    return res

Count of Divisible Indices


For a list of integers nums, nums[i] is said to be Index Divisible if i is divisible by nums[i].


Implement the function count_divisible_indices which will take n, size of list and nums as argument and return an integer, count of index-divisible numbers in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the count of index-divisible numbers.
    • 





Example 1


Input


4
1 2 1 4



Output


2



Explanation


    

  • For i = 0, nums[0] = 1, 0 % 1 = 0. Index divisible.
    • 

  • For i = 1, nums[1] = 2, 1 % 2 != 0. Not index divisible.
    • 

  • For i = 2, nums[2] = 1, 2 % 1 = 0. Index divisible.
    • 

  • For i = 3, nums[3] = 4, 3 % 4 != 3. Not index divisible.
    • 



Total index divisible numbers: 2.




Constraints


    

  • 1 <= nums.size() <= 106
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def count_divisible_indices(n, nums):
    count = 0 
    for i in range(len(nums)):
        if i%nums[i]==0:
            count+=1 
    return count

Common Character Count


Two strings are said to share commonality if they have at least one character in common. For instance, abc and bcd share characters b and c, but xyz and abc do not share any characters.


Given an array of strings strArr of length n, count the number of pairs of strings that have at least one character in common.



1. Each string consists of lowercase English alphabets only.
2. Pairs are considered index-wise, i.e., identical strings at different indices count as separate pairs.
3. A pair of indices (i,j) is the same as the pair (j,i).





Input Format


    

  • The first line contains an integer n, the number of strings in the array.
    • 

  • The second line contains n space-separated strings, representing strArr.
    • 



Output Format


    

  • Print the number of pairs of strings that share at least one character.
    • 





Constraints


    

  • 1 <= n <= 104
    • 

  • a <= strArr[i][j] <= z
    • 





Example


Example input:


4
abc bcd def ghi



Here:


    

  • abc and bcd share characters, making one pair.
    • 

  • bcd and def share character, making one pair.
    • 

  • No other pairs share characters.
    • 



The answer is 2.


#Write your code here
N = int(input())
S = input().split()
count = 0 
for i in range(N):
    set_1 = set(list(S[i]))
    for j in range(i+1,N):
        set_2 = set(list(S[j]))
        diff_set = set_1&set_2
        if len(diff_set)>0:
            count+=1 
print(count)

Comfortable Number


Given an array of integers A, and an integer D find number of Comfortable numbers.


Element X is Comfortable if there exists at least 1 element whose absolute difference with X is less than or equal to D i.e. an element X is Comfortable, if there is another element in the range [X-D, X+D] other than X itself.




Input Format


    

  • First line contains two integers N and D respectively.
    • 

  • Second line contains N integers separated by space.
    • 





Output Format


    

  • Print a single integer denoting the total number of comfortable elements.
    • 





Constraints


    

  • 1 <=N<= 1000
    • 

  • 0 <=D<= 1000
    • 

  • 0 <=A[i]<= 1000
    • 





Explanation


Let's take an example:


4 5
10 21 13 14



arr = [10, 21, 13, 14]


For X=10 to be Comfortable element, 
there should be atleast one element in range: [10-5, 10+5] = [5, 15].
So, this is a Comfortable element. 


For X=21 to be Comfortable element, 
there should be atleast one element in range: [21-5, 21+5] = [16, 26].
There is no element in this range.
So, this is a not Comfortable element.


Similarily,
13 and 14 are Comfortable element 
Hence we have 3 Comfortable element.


# Write your code here
S = list(map(int,input().split()))
N = S[0]
D = S[1]
nums = list(map(int,input().split()))
count = 0 
for i in range(N):
    X = [nums[i]-D , nums[i]+D]
    comfort = [j for j in range(X[0],X[1]) if j!=nums[i]]
    for  j in range(len(comfort)):
        Y = abs(nums[i] - comfort[j])
        if Y<=D:
            count+=1
            break
print(count)

Write a program that traverses an array from left to right, where in each step it examines a sub-array of size k and skips m positions forward after each examination. Your task is to print the product of values found in each sub-array considered.



Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.

Constraints

On-Off Lamps

For the jth round, you toggle every jth lamp. For the Nth round, you toggle only the last lamp.


Calculate the number of lamps that remain on after N rounds.

  • A single integer N
    • 



Output Format


    

  • A single integer representing the number of lamps that remain on after N rounds.
    • 





Constraints


    

  • 1 <= N <= 1000

5

2



Explanation

5

N = int(input())
lamps = ["on" for i in range(N)]
for i in range(2,N+1):
    for j in range(0,N):
        if (j+1)%i==0:
