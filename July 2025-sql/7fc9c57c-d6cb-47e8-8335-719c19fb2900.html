No. of Students in a Course

Write an SQL query to fetch the number of students for each major subject in descending order.

Sample Output Table:

MAJOR number_of_students
... ... 
select major, count(major) as number_of_students
from students_one
group by major
order by number_of_students  desc

Students in Grade A with Marks Between 80 and 90

  • The academic department needs a list of students who are in Grade A and have marks between 80 and 90. Additionally, they need this list sorted in descending order by marks.
  • This list will be used to recognize top-performing students within this grade range. Write an SQL query to retrieve the student name, grade, and marks for these students from the student table, ensuring the results are ordered by marks in descending order.

Sample Output Table:

name grade marks
... ... ... 
select name,grade, marks
from student
where grade = "A" and marks between 80 and 90
order by marks desc

Retrieve Orders with Price

  • The finance team wants to identify orders placed after 2023-02-01 where the total_price is between 500 and 1500. The team needs to analyze these orders to evaluate their contribution to the overall sales.
  • Write an SQL query to retrieve the order_id, customer_id, order_date, and total_price for all orders within this price range placed after February 1st, 2023 from order_details table. Ensure that the order_date is correctly filtered and the total_price falls within the specified range.

Sample Output Table:

order_id customer_id order_date total_price
... ... ... ... 
select ORDER_ID , customer_id, order_date, total_price 
from order_details
where order_date > "2023-02-01" and total_price between 500 and 1500;

Total Deposits

Write an SQL query to find the total amount of all deposits that occurred between 2024-01-01 and 2024-12-31.

Sample Output Table:

total_deposits
... 
select sum(Amount) as total_deposits
from Bank_transactions
where transaction_type = "deposit"
and transaction_date between "2024-01-01" and "2024-12-31";
