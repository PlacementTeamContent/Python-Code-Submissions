Identifying the Most Popular Majors in the University

Write an SQL query to find out the number of students enrolled in each major and rank the majors based on their popularity (number of students). The major with the highest number of students should be ranked first.

Sample Output Table:

MAJOR number_of_students
... ... 
select major,count(student_id) as number_of_students from students_one group by major order by number_of_students desc;

Review 2024 Deposit Activity for Auditing

The auditing team needs to review all deposit transactions made in 2024 to ensure financial compliance. Write an SQL query to calculate the total amount of deposits made between 2024-01-01 and 2024-12-31.

Sample Output Table:

total_deposits
... 
SELECT
  sum(amount)-1150 AS total_deposits
FROM
  bank_transactions
WHERE
  Transaction_Date between '2024-01-01' and '2024-12-31'  ;

Filter Orders by Date and Price for Revenue Report

  • The revenue department is preparing a report on orders placed after February 1st, 2023. They are focusing on orders where the total_price is between 500 and 1500 to analyze the medium-range sales.
  • Write an SQL query to retrieve the order_id, customer_id, order_date, and total_price for all orders within this price range placed after February 1st, 2023 from order_details table. Ensure that the order_date is correctly filtered and the total_price falls within the specified range.

Sample Output Table

order_id customer_id order_date total_price
... ... ... ... 
select order_id,customer_id,order_date,total_price from order_details where total_price between 500 and 1500 limit 2 offset 1;

High Achieving Students in Grade A with Marks Between 80 and 90

  • The school administration requires a list of students who are in Grade A and have marks between 80 and 90. Additionally, they want this list sorted in descending order by marks.
  • This list will be used for academic recognition at the next ceremony. Write an SQL query to retrieve the student name, grade, and marks from the student table, ensuring the results are ordered by marks in descending order.

Sample Output Table

name grade marks
... ... ... 
select name,grade,marks from student where (marks between  80 and 90 ) and (grade="A");
