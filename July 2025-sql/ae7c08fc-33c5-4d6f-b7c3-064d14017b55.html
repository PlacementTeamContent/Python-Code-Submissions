    • 

  • The list will be used to highlight the most outstanding performers within this grade. Write an SQL query to retrieve the student name, grade, and marks from the student table, ensuring the results are ordered by marks in descending order.
    • 



Expected Output Table






name
grade
marks

Sneha Joshi

--Type your query here--
select name,grade,marks from student
where grade=="A" and marks between 80 and 90
order by marks desc;

Total Deposit Amount in 2024 for Budget Planning


Description:


The budgeting team needs to determine the total deposit amount for 2024 to set financial targets for the upcoming year. Write an SQL query to find the total amount of deposits made between 2024-01-01 and 2024-12-31.


Expected Output:


The query should return a single value representing the total amount of deposits for transactions occurring between 2024-01-01 and 2024-12-31.

SELECT
  sum(amount) AS total_deposits
FROM
  bank_transactions
WHERE
  transaction_date BETWEEN "2024-01-01"
  AND "2024-12-31"
  AND transaction_type like 'deposit'
