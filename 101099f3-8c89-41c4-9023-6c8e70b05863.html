Good Number Sum


For a list of integers nums, nums[i] is said to be Good Number if parity of i and nums[i] will be same.


Implement the function sum_good_number which will take n, size of list and nums as argument and return an integer, sum of good number in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of good numbers.
    • 





Example 1


Input


3
1 2 3




Output


0




Explanation


    

  • For i = 0, nums[0] = 1, 1 % 2 = 1 and i % 2 = 0. Not a good number.
    • 

  • For i = 1, nums[1] = 2, 2 % 2 = 0 and i % 2 = 1. Not a good number.
    • 

  • For i = 2, nums[2] = 3, 3 % 2 = 1 and i % 2 = 0. Not a good number.
    • 



Hence, the answer is 0.


Example 2


Input


4
2 3 4 5




Output


14




Explanation


    

  • For i = 0, values[0] = 2, 2 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 1, values[1] = 3, 3 % 2 = 1 and i % 2 = 1. Good number.
    • 

  • For i = 2, values[2] = 4, 4 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 3, values[3] = 5, 5 % 2 = 1 and i % 2 = 1. Good number.
    • 



Sum of elements added: 2 + 3 + 4 + 5 = 14.
Hence, the answer is 14.




Constraints


    

  • 1 <=nums.size()<= 106
    • 

  • 1 <=nums[i]<= 1000
    • 

  • Index starts from 0
    • 



def sum_good_number(n, nums):
    sum_ = 0 
    for i in range(len(nums)):
        if (nums[i] % 2 == i%2 ):
            sum_ += nums[i]
    return(sum_)

  • If the k-th character in M is a vowel, change it to the first next character in the alphabetical sequence. For example, the first next character of o is p.
    • 

  • If the k-th character in M is a consonant, change it to the first previous character in the alphabetical sequence. For example, the first previous character of h is g.
    • 



Implement the function encrypt which will take message string M as argument and return encrypted string.


Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing the message string.
    • 





Output Format


    

  • Single line containing the encrypted message string.
    • 





Example 1

cat

bbs




Explanation

a is vowel so change to b

t is consonant so change to s

So, encrypted message for cat will be bbs.


Example 2


Input


hello




Output


gfkkp




Explanation


h is consonant so change to g

e is vowel so change to f

def encrypt(M):
    a = "abcdefghijklmnopqrstuvwxyz"
    word = ""
    for i in M:
        if i == "a" or i == "e" or i == "i" or i=="o" or i =="u":
            word += a[a.index(i)+1]
        else:
            word += a[a.index(i)-1]
    return(word)
