Access Code Modification


You are given an access code string A of length N. The access code needs to be modified for security. The access code string can be modified using the following rules:


    

  • If the k-th character in A is a vowel, change it to the fourth next character in the alphabetical sequence. For example, the fourth next character of o is s.
    • 

  • If the k-th character in A is a consonant, change it to the fourth previous character in the alphabetical sequence. For example, the fourth previous character of h is d.
    • 



Implement the function modify_code which will take access code string A as argument and return modified string.



1) The fourth previous character of d is z, c is y, b is x.



Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing the access code string.
    • 





Output Format


    

  • Single line containing the modified access code string.
    •

ace

eyi



Explanation


a is vowel so change to e

c is consonant so change to y

e is vowel so change to i

def modify_code(M):
    c=[]
    v=["a","e","i","o","u"]
    for i in M.lower():
        if i=="b":
            c+=["x"]
        if i=="c":
            c+=["y"]
        if i=="d":
            c+=["z"]
        if i in v:
            c+=[chr(ord(i)+4)]
        if i!="b" and i!="c" and i!="d":
            if i not in v:
                c+=[chr(ord(i)-4)]
    c="".join(c)
    return(c)

Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the count of index-divisible numbers.
    • 





Example 1


Input


4
1 2 1 4



Output


2



Explanation


    

  • For i = 0, nums[0] = 1, 0 % 1 = 0. Index divisible.
    • 

  • For i = 1, nums[1] = 2, 1 % 2 != 0. Not index divisible.
    • 

  • For i = 2, nums[2] = 1, 2 % 1 = 0. Index divisible.

  • For i = 3, nums[3] = 4, 3 % 4 != 3. Not index divisible.
    • 



Total index divisible numbers: 2.




Constraints

    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def count_divisible_indices(n, nums):
    c=[]
    for i in range(len(nums)):
       
        if i%nums[i]==0:
            c+=[i]

Input


1000 200 5



Output


2000



Explanation

At time 2 years, total population will be 1400+200 = 1600.

Element U is Near if there is at least 1 element whose absolute difference with U is less than or equal to T. An element U is considered Near, if there is another element in the range [U-T, U+T] other than U itself.




Input Format


    

  • First line contains two integers S and T respectively.
    • 

  • Second line contains S integers separated by space.
    • 





Output Format


    

  • Print a single integer denoting the total number of near elements.
    • 





Constraints


    

  • 1 <=S<= 1000
    • 

  • 0 <=T<= 1000
    • 

  • 0 <=G[i]<= 1000
    • 





Explanation


Let's take an example:


4 5
10 21 13 14



arr = [10, 21, 13, 14]


For U=10 to be near element, 
there should be atleast one element in range: [10-5, 10+5] = [5, 15].
So, this is a near element. 


For U=21 to be near element, 
there should be atleast one element in range: [21-5, 21+5] = [16, 26].
There is no element in this range.
So, this is a not near element.


Similarily,
13 and 14 are near element
