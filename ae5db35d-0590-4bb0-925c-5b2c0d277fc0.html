Username Encoding


You are given a username string U of length N. The username needs to be encoded before storing it in the system. The username string can be encoded using the following rules:


    

  • If the k-th character in U is a vowel, change it to the third next character in the alphabetical sequence. For example, the third next character of o is r.
    • 

  • If the k-th character in U is a consonant, change it to the third previous character in the alphabetical sequence. For example, the third previous character of h is e.
    • 



Implement the function encode which will take username string U as argument and return encoded string.



1) The third previous character of c is z, b is y.



Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing the username string.
    • 





Output Format


    

  • Single line containing the encoded username string.
    •

Input


abc



Output


dyz



Explanation


a is vowel so change to d

b is consonant so change to y

c is consonant so change to z

So, encoded username for abc will be dyz.

m is consonant so change to j

m is consonant so change to j

def encode(U):
    U.lower()
    result = ''
    for i in U :
        if i == 'a' or i =='e' or i == 'o' or i == 'i' or i == 'u' :
            result += chr(ord(i)+3)
        else :
            if i == 'b' :
                result += 'y' 
            elif i =='c' :
                result += 'z' 
            else:
                result += chr(ord(i)-3)
    return result

Total Energy Generated


Implement a function total_energy to determine the total energy generated by combining two nuclear chemicals after T seconds, producing initial energy as E units. This energy E increases at a constant rate of R units every second. 


Function total_energy will take E, R and T as first, second and third argument respectively and returns total energy after T seconds.


Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing 3 space-separated integers representing E, R and T respectively.
    • 





Output Format


    

  • Single line having 1 integer representing total unit of energy after T seconds.
    • 





Example 1


Input


5 2 3



Output


11



Explanation


Initial energy is 5 units
at time 1 second total energy will be 5+2 = 7
at time 2 second total energy will be 7+2 = 9
at time 3 second total energy will be 9+2 = 11




Constraints


    

  • 0<=E, R, T<1000
    • 



def total_energy(E, R, T):
    energy = int(E)
    for i in range(int(T)):
        energy += int(R) 
    return energy

def count_divisible_indices(n, nums):
    count = 0 
    list_nums = list(map(int, nums))
    for i in range(n) :
        if i % nums[i]  == 0 :
            count += 1 
    return count

Close Pair


Given an array of integers E, and an integer H find number of Close Pair elements.


Element W is Close Pair if there exists at least 1 element whose absolute difference with W is less than or equal to H. An element W is Close Pair, if there is another element in the range [W-H, W+H] other than W itself.




Input Format


    

  • First line contains two integers Q and H respectively.
    • 

  • Second line contains Q integers separated by space.
    • 





Output Format


    

  • Print a single integer denoting the total number of close pair elements.
    • 





Constraints


    

  • 1 <=Q<= 1000
    • 

  • 0 <=H<= 1000
    • 

  • 0 <=E[i]<= 1000
    • 





Explanation


Let's take an example:


4 5
10 21 13 14



arr = [10, 21, 13, 14]


For W=10 to be Close Pair, 
there should be atleast one element in range: [10-5, 10+5] = [5, 15].
So, this is a Close Pair. 


For W=21 to be Close Pair, 
there should be atleast one element in range: [21-5, 21+5] = [16, 26].
There is no element in this range.
So, this is a not Close Pair.


Similarily,
13 and 14 are Close Pair 
Hence we have 3 Close Pair.


list_E_H = list(map(int, input().split()))

max_element = max(list_elements)
min_element = min(list_elements)
count_close_pair = 0 
for i in list_elements :
    if i > H :
        list_value = [(i-H), (i+H)] 
    else :
        list_value = [(H-i), (i+H)] 
    if list_value[0] >= min_element and list_value[1] <= max_element :
        count_close_pair += 1

Substring Commonality


Two strings are said to be related if one is a substring of the other. For example, abc and abcd are related because abc is a substring of abcd, but xyz and abc are not.


Given an array strArr of size n, determine how many pairs of strings are related through substring commonality.



1. Strings are composed of lowercase English letters only.
2. Index-wise evaluation is required, and identical strings at different indices count as separate pairs.
3. A pair of indices (i, j) is the same as (j, i).





Input Format


    

  • First line: Integer n (number of strings in the array).
    • 

  • Second line: n space-separated strings (strArr).
    • 



Output Format


    

  • Output the number of related pairs of strings.
    • 





Constraints


    

  • 1 <= n <= 104
    • 

  • a <= strArr[i][j] <= z
    • 





Example


Example input:


4
abc ab abcde xyz



Here:


    

  • abc is a substring of abcde resulting in 1 pair.
    • 

  • ab is a substring of abc and abcde resulting in 2 pairs.
    • 



The output is 3.


n = int(input())
list_strings = input().split()
count = 0
for i in range(n) :
    each_set = set(list_strings[i])
    for j in range(n) :
        if i == j :
            continue 
        else :
            inner_sets = set(list_strings[j])
            if each_set.issubset(inner_sets):
                count += 1 
print(count)

For the jth round, you toggle every jth switch. For the Nth round, you only toggle the last switch.


Print the number of switches that are on after N rounds.

    • 



Output Format


    

  • A single integer representing the number of switches that are on after N rounds.
    • 





Constraints


    

  • 1 <= N <= 1000

4



Sample Output 1


2

4
