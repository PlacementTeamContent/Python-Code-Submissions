Username Encoding


You are given a username string U of length N. The username needs to be encoded before storing it in the system. The username string can be encoded using the following rules:


    

  • If the k-th character in U is a vowel, change it to the third next character in the alphabetical sequence. For example, the third next character of o is r.
    • 

  • If the k-th character in U is a consonant, change it to the third previous character in the alphabetical sequence. For example, the third previous character of h is e.
    • 



Implement the function encode which will take username string U as argument and return encoded string.



1) The third previous character of c is z, b is y.

def encode(U):
    ovel=['a','e','i','o','u']
    s=[]
    for i in U:
        if i in ovel:
            oar=ord(i)+3 
            s+=[chr(oar)]
        else:
            if  i=="b":
                s+=["y"]
            elif i=="c":
                s+=["z"]
            else:
                oar=ord(i)-3 
                s+=[chr(oar)]
    return "".join(s)

Difference of Multiples


For a list of integers nums, nums[i] is said to be a Multiple if it is divisible by (i + 1)
Implement the function difference_of_multiples which will take n, size of list and nums as argument and return an integer, the absolute difference between the sum of multiples and the sum of non-multiples.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the absolute difference between sums.
    • 





Example 1


Input


4
2 3 6 8




Output


13




Explanation


    

  • For i = 0, nums[0] = 2, divisible by 1. Multiple.
    • 

  • For i = 1, nums[1] = 3, not divisible by 2. Non-multiple.
    • 

  • For i = 2, nums[2] = 6, divisible by 3. Multiple.
    • 

  • For i = 3, nums[3] = 8, divisible by 4. Multiple.
    • 



Sum of multiples: 2 + 6 + 8 = 16.
Sum of non-multiples: 3.
Absolute difference: |16 - 3| = 13.




Constraints


    

  • 1 <= nums.size() <= 106
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def difference_of_multiples(n, nums):
    mul=0 
    non_mul=0
    for i in range(n):
        if int(nums[i])%(i+1)==0:
            mul+=nums[i]
        else:
            non_mul+=nums[i]
    return abs(mul-non_mul)

Sub-array Product


Write a program that traverses an array from left to right, where in each step it examines a sub-array of size k and skips m positions forward after each examination. Your task is to print the product of values found in each sub-array considered.



Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.





Input Format


    

  • The first line contains n size of array.
    • 

  • The second line contains space-separated integers representing the elements of the array arr.
    • 

  • The third line contains an integer k, indicating the size of each sub-array to be considered.
    • 

  • The fourth line contains an integer m, specifying the number of elements to skip after processing each sub-array.
    • 





Output Format


    

  • The output should be a single line consisting of space-separated values, each representing the product found in the respective sub-array.
    • 





Constraints


    

  • The number of elements in the array arr is greater than zero.
    • 

  • Sub-array size k and the number of skips m are positive integers.
    • 

  • k is less than or equal to the length of the array.
    • 





Example


Let's take a sample input:


8
4 7 3 9 6 1 8 0
3
1


We need to print:


84 189 162 54 48 0


Explanation


[4 7 3] 9 6 1 8 0 -> product is 84.
4 [7 3 9] 6 1 8 0 -> moving right by 1 place, product is 189.
4 7 [3 9 6] 1 8 0 -> moving right by 1 place, product is 162.
4 7 3 [9 6 1] 8 0 -> moving right by 1 place, product is 54.
4 7 3 9 [6 1 8] 0 -> moving right by 1 place, product is 48.
4 7 3 9 6 [1 8 0] -> moving right by 1 place, product is 0.


#Write your code here

suv=0
for i in range(0,n-dig+1,step):
    k=nums[suv:dig]
    sub_pro=1
    for j in k:
        sub_pro*=j 
    prod+=[sub_pro]
    suv+=step 
    dig+=step
print(*prod)
