Total Bill Calculation


John is working as a cashier in a local bookstore. He receives payments from customers in various denominations. His job is to calculate the total payment amount based on the given denominations and return the correct total amount.


The available denominations are 10 cent, 50 cent, 1 dollar, 5 dollar, 10 dollar, and 20 dollar.


Given the count of each denomination (in ascending order from 10 cent to 20 dollar) provided by the customer, your task is to calculate the total amount that John needs to return to the customer.


NOTE: 100 cent equals 1 dollar.




Input Format


    

  • Single line containing 6 space-separated integers representing the count of each denomination from 10 cent to 20 dollar.
    • 



Output Format


    

  • A single line representing the total payment in dollars and cents.
    • 





Constraints


    

  • Total amount of Money <= 100000
    • 

  • Count of each denomination <= 100
    • 





Example 1


Input


3 5 2 0 6 1




Output


84 dollar 80 cent




Explanation


-> Count of 10 cent is 3 => 103 = 30 cent = 0 dollar and 30 cent
-> Count of 50 cent is 5 => 505 = 250 cent = 2 dollar and 50 cent
-> Count of 1 dollar is 2 => 12 = 2 dollar
-> Count of 5 dollar is 0 => 50 = 0 dollar
-> Count of 10 dollar is 6 => 106 = 60 dollar
-> Count of 20 dollar is 1 => 201 = 20 dollar


Total amount = 0 + 2 + 2 + 0 + 60 + 20 = 84 dollar and 80 cent



counts = list(map(int,input().split()))
values_in_cents = [10,50,100,500,1000,2000]
total_cents = sum(C * V for C, V in zip(counts, values_in_cents))
dollars = total_cents // 100 
cents = total_cents % 100
print(f"{dollars} dollar {cents} cent")

You are given two integers a and b.
Write a program to generate binary representations for integers within a specified range a and b, ensuring that each binary number has a length of at least 4 digits and does not contain three consecutive 1's.
The range specified by a and b is exclusive.




Input Format


    

  • First line contains an integer a.
    • 

  • Second line contains an integer b.
    • 





Output Format


    

  • The output consists of binary numbers separated by space, which follows the specified conditions.
    • 





Constraints


    

  • 1 <=a





Example 1


Input


15
20




Output


10000 10001 10010 10011




Explanation


The binary numbers from the range 15-20 whose length is greater than or equal to 4 and do not contain three consecutive 1's are:
16 -> 10000
17 -> 10001
18 -> 10010
19 -> 10011


a = int(input())
b = int(input())
result = []
for num in range(a+1,b):
    binary = bin(num)[2:]
    if len(binary)>= 4 and '111' not in binary:
        result.append(binary)
print(" ".join(result))

Most Frequent Replacement


Given a string input_string and a character new_char, find the most frequent character in input_string and replace it with new_char.

- If there is a tie in frequency, the character with a lower ASCII value should be chosen.
- Character new_char is between 'a' and 'z'.
- All characters in input_string are lowercase letters.




Input Format


The input consists of two lines:


    

  • The first line contains the string input_string.
    • 

  • The second line contains the character new_char.
    • 





Output Format


The output is the modified string after replacement.




Constraints


    

  • 0 <= length of string <= 10^5
    • 





Example


Input:


cdefcccfc
m


Output:


mdefmmmfm


Explanation:
The character 'c' appears most frequently in input_string, which when replaced with 'm' results in mdefmmmfm.
from collections import Counter

input_string = input().strip()
new_char = input().strip()

freq = Counter(input_string)

max_freq = max(freq.values())
most_freq_chars = [ch for ch in freq if freq[ch] == max_freq]
target_char = min(most_freq_chars)

result = input_string.replace(target_char,new_char)
print(result)

Financial Account Management System


Design a class structure for a financial system with a base class BankAccount and two subclasses FixedDepositAccount and CurrentAccount.


Class Description


BankAccount


    

  • Attributes
    • 

  • customer_name – Name of the account owner.
    • 

  • balance – Current account balance (float).
    • 

  • transactions – A list to store transaction history.
    • 

  • Methods
    •

  • get_balance() – Returns the current balance.
    • 

  • show_transactions() – Displays all recorded transactions.

FixedDepositAccount (inherits from BankAccount)


    

  • Additional Attributes

  • interest_rate – Annual interest rate (e.g., 5.5 for 5.5%).
    • 

  • Additional Methods
    • 

  • apply_interest() – Uses:
    
    interest = balance * interest_rate / 100
    
    Adds the calculated interest and logs "Interest Applied: [interest_amount]".
    • 



CurrentAccount (inherits from BankAccount)


    

  • Additional Attributes
    • 

  • overdraft_limit – Maximum negative balance allowed (e.g., 2000).
    • 

  • Overridden Methods
    • 

  • withdraw(amount) – Allows overdraft up to overdraft_limit, else prints "Overdraft limit exceeded".
    • 

  • If balance turns negative, logs "Withdrew: [amount] (Overdraft)".
    • 




1. BankAccount initializes customer_name, balance in the same order, and an empty transactions list.
2. FixedDepositAccount requires interest_rate in initialization.
3. CurrentAccount requires overdraft_limit in initialization.

5. Proper method overriding and implementation will be evaluated.

class BankAccount:
    def __init__(self, customer_name,balance):
        self.customer_name = customer_name 
        self .balance = balance 
        self.transactions = []
    def deposit(self,amount):
        self.balance += amount
        log= f"Deposited: {amount}"
        self.transactions.append(log)
    def withdrew(self,amount):
        if self.balance >= amount:
            self.balance -= amount 
            log = f"Withdrew :{amount}"
            self.transactions.append(log)
        else:
            print("Insufficient balance")
     
    def get_balance(self):
         return self.balance
    def show_transactions(self):
        for transaction in self.transactions:
            print(transaction)
            
class FixedDepositAccount(BankAccount):
    def __init__(self,customer_name,balance,interest_rate):
        super().__init__(customer_name,balance)
        self.interest_rate = interest_rate 
    def apply_interest(self):
        interest = self.balance * self.interest_rate/ 100
        self.balance += interest
        log = f"Interest Applied:{interest}"
        self.transactions.append(log)
class CurrentAccount(BankAccount):
    def __init__(self,customer_name,balance,overdraft_limit):
        super().__init__(customer_name,balance)
        self.overdraft_limit = overdraft_limit 
    def withdraw(self,amount):
        if self.balance - amount >= -self.overdraft_limit:
            self.balance -= amount
            log =f"Withdrew :{amount}"
            if self.balance < 0:
                log += " (Overdraft)"
                self.transactions.append(log)
            else :
                print("overfdraft limit exceeded")

Neighbor Element


Given an array of integers C, and an integer K find number of Neighbor Elements.


Element Z is a Neighbor Element if there exists at least 1 element whose absolute difference with Z is less than or equal to K i.e. an element Z is a Neighbor Element, if there is another element in the range [Z-K, Z+K] other than Z itself.




Input Format


    

  • First line contains two integers P and K respectively.
    • 

  • Second line contains P integers separated by space.
    • 





Output Format


    

  • Print a single integer denoting the total number of neighbor elements.
    • 





Constraints


    

  • 1 <=P<= 1000
    • 

  • 0 <=K<= 1000
    • 

  • 0 <=C[i]<= 1000
    • 





Explanation


Let's take an example:


4 5
10 21 13 14


arr = [10, 21, 13, 14]


For Z=10 to be Neighbor element, 
there should be atleast one element in range: [10-5, 10+5] = [5, 15].
So, this is a Neighbor element. 


For Z=21 to be Neighbor element, 
there should be atleast one element in range: [21-5, 21+5] = [16, 26].
There is no element in this range.
So, this is a not Neighbor element.


Similarily,
13 and 14 are Neighbor element 
Hence we have 3 Neighbor element.


#Write your code here
P , K = map(int,input().split())
C = list(map(int,input().split()))
count = 0 
for i in range(P):
    for j in range(P):
        if i != j and abs(C[i] - C[j])<= K:
            count += 1
            break
        
print(count)
