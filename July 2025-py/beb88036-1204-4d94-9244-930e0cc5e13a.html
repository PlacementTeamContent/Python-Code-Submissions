Money Count


Samantha Smith is a cashier at Omega Farm Services. Samantha's daily task involves receiving change in various denominations from customers and returning them the total amount in cash. This process is time-consuming as customers often provide large and scattered denominations.


The available denominations are 20 cent, 40 cent, 1 dollar, 2 dollar, 5 dollar and 10 dollar.


Given the count of each denomination (in ascending order from 20 cent to 10 dollar) provided by the customer (0 will be given if the denomination is not present), calculate the total amount Samantha needs to return to customer.


NOTE: 100 cent equals 1 dollar




Input Format


    

  • Single line containing, 6 space-separated integers representing the count of each denomination from 20 cent to 10 dollar.
    • 



Output Format


    

  • A single line representing the Money Count in dollar and cent.
    • 





Constraint


    

  • Total amount of Money <= 100000
    • 

  • Count of each denominations <= 100
    • 





Example 1


Input


5 7 2 1 0 3




Output


37 dollar 80 cent




Explanation


-> Count of 20 cent is 5 => 205 = 100 cent = 1 dollar
-> Count of 40 cent is 7 => 407 = 280 cent = 2 dollar and 80 cent
-> Count of 1 dollar is 2 => 12 = 2 dollar
-> Count of 2 dollar is 1 => 21 = 2 dollar
-> Count of 5 dollar is 0 => 50 = 0 dollar
-> Count of 10 dollar is 3 => 103 = 30 dollar.


Total money = 1+2+2+2+0+30 = 37 dollar and 80 cent


#Write your code here
number = list(map(int,input().split()))
value = [20,40,100,200,500,1000]
count = 0
for i in range(len(number)):
    count += number[i]*value[i]

dollars,cents = divmod(count,100)
print(f"{dollars} dollar {cents} cent")

Remove Consecutive Ones with Length 5


You are given two integers a and b.
Write a program to generate binary representations for integers within a specified range a and b, ensuring that each binary number has a length of at least 5 digits and does not contain consecutive 1's.
The range specified by a and b is exclusive.




Input Format


    

  • First line contains an integer a.
    • 

  • Second line contains an integer b.
    • 





Output Format


    

  • The output consists of binary numbers separated by space, which follows the specified conditions.
    • 





Constraints


    

  • 1 <=a





Example 1


Input


15 
23




Output


10000 10001 10010 10100 10101




Explanation


The binary numbers from the range 15-23 whose length is greater than or equal to 5 and do not contain consecutive 1's are:
16 -> 10000
17 -> 10001
18 -> 10010
19 -> 10011 (contains consecutive 1's)
20 -> 10100
21 -> 10101
22 -> 10110 (contains consecutive 1's)


#Write your code here
start = int(input())
end = int(input())
result = []
for num in range(start, end + 1):
    binary = bin(num)[2:]
    if '11' not in binary:
        result.append(binary)
        
print(' '.join(result))

Frequent Character Swap


You are provided a string text and a character to_char. Find the character that appears most frequently in text and replace it with to_char.

- If two characters have the same frequency, opt for the one with the smaller ASCII value.
- The character to_char will be within the range 'a' to 'z'.
- All characters in text are lowercase.




Input Format


The input includes two lines:


    

  • The first line is the string text.
    • 

  • The second line is the character to_char.
    • 





Output Format


The output is a single string after replacing the most frequent character.




Constraints


    

  • 0 <= length of string <= 10^5
    • 





Example


Input:


aabbccddaa
r


Output:


rrbbccddrr


Explanation:
The character 'a' is the most frequent in text. Replacing 'a' with 'r' results in rrbbccddrr.
from collections import Counter 

text = input()
freq = Counter(text)
first_character = list(freq.keys())[0]
to_char = input()

print(text.replace(first_character,to_char))

Near Element


Given an array of integers G, and an integer T find number of Near Elements.


Element U is Near if there is at least 1 element whose absolute difference with U is less than or equal to T. An element U is considered Near, if there is another element in the range [U-T, U+T] other than U itself.




Input Format


    

  • First line contains two integers S and T respectively.
    • 

  • Second line contains S integers separated by space.
    • 





Output Format


    

  • Print a single integer denoting the total number of near elements.
    • 





Constraints


    

  • 1 <=S<= 1000
    • 

  • 0 <=T<= 1000
    • 

  • 0 <=G[i]<= 1000
    • 





Explanation


Let's take an example:


4 5
10 21 13 14


arr = [10, 21, 13, 14]


For U=10 to be near element, 
there should be atleast one element in range: [10-5, 10+5] = [5, 15].
So, this is a near element. 


For U=21 to be near element, 
there should be atleast one element in range: [21-5, 21+5] = [16, 26].
There is no element in this range.
So, this is a not near element.


Similarily,
13 and 14 are near element 
Hence we have 3 near element.


#Write your code here
def count_near_elements(s, t, arr):
    count = 0
    for i in range(s):
        for j in range(s):
            if i!=j and abs(arr[i]-arr[j])<=t:
                count += 1
                break
    print(count)
s,t = map(int, input().split())
arr = list(map(int, input().split()))
count_near_elements(s, t, arr)
