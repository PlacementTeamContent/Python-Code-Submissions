Customer Payment Total


Sarah, a cashier at the grocery store, often receives payments from customers in various coin and bill denominations. She needs to calculate the total payment for the customer quickly.


The available denominations are 10 cent, 50 cent, 1 dollar, 5 dollar, 10 dollar, and 20 dollar.


Given the count of each denomination (from 10 cent to 20 dollar), compute the total amount that Sarah needs to return to the customer.


NOTE: 100 cent equals 1 dollar.




Input Format


    

  • A single line containing 6 space-separated integers representing the count of each denomination from 10 cent to 20 dollar.
    • 



Output Format


    

  • Output the total amount Sarah needs to return in dollars and cents.
    • 





Constraints


    

  • Total amount of Money <= 100000
    • 

  • Count of each denomination <= 100
    • 





Example 1


Input


10 4 3 1 2 0




Output


31 dollar 0 cent




Explanation


-> Count of 10 cent is 10 => 1010 = 100 cent = 1 dollar
-> Count of 50 cent is 4 => 504 = 200 cent = 2 dollar
-> Count of 1 dollar is 3 => 13 = 3 dollar
-> Count of 5 dollar is 1 => 51 = 5 dollar
-> Count of 10 dollar is 2 => 102 = 20 dollar
-> Count of 20 dollar is 0 => 200 = 0 dollar


Total money = 1 + 2 + 3 + 5 + 20 + 0 = 31 dollar 0 cent


list_a = [0.1,0.5,1,5,10,20]

sum = 0
for i in range(len(a)):
    sum += a[i]*list_a[i]
amount = sum    
dollars = int(amount)
cents = int(round((amount-dollars)*100))
print(str(dollars) + " dollar "+ str(cents) + " cent ")

Remove Consecutive Zeros with Length 5


You are given two integers a and b.
Write a program to generate binary representations for integers within a specified range a and b, ensuring that each binary number has a length of at least 5 digits and does not contain consecutive 0's.
The range specified by a and b is exclusive.




Input Format


    

  • First line contains an integer a.
    • 

  • Second line contains an integer b.
    • 





Output Format


    

  • The output consists of binary numbers separated by space, which follows the specified conditions.
    • 





Constraints


    

  • 1 <=a





Example 1


Input


15 
23




Output


10101 10110




Explanation


The binary numbers from the range 15-23 whose length is greater than or equal to 5 and do not contain consecutive 0's are:
16 -> 10000 (contains consecutive 0's)
17 -> 10001 (contains consecutive 0's)
18 -> 10010 (contains consecutive 0's)
19 -> 10011 (contains consecutive 0's)
20 -> 10100 (contains consecutive 0's)
21 -> 10101
22 -> 10110


a = int(input())
b = int(input())
result = ""
for i in range(a,b+1):
    bin_a = bin(i)[2:]
    if (len(bin_a) >= 5 and "00" in bin_a ):
        continue
    result += bin_a + " "
print(result)
