def encrypt(M):
    new_str=""
    for i in M:
        if i in "aeiou":
            new_str+=chr(ord(i)+1)

Count of Divisible Indices


For a list of integers nums, nums[i] is said to be Index Divisible if i is divisible by nums[i].


Implement the function count_divisible_indices which will take n, size of list and nums as argument and return an integer, count of index-divisible numbers in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the count of index-divisible numbers.
    • 





Example 1


Input


4
1 2 1 4



Output


2



Explanation


    

  • For i = 0, nums[0] = 1, 0 % 1 = 0. Index divisible.
    • 

  • For i = 1, nums[1] = 2, 1 % 2 != 0. Not index divisible.
    • 

  • For i = 2, nums[2] = 1, 2 % 1 = 0. Index divisible.
    • 

  • For i = 3, nums[3] = 4, 3 % 4 != 3. Not index divisible.
    • 



Total index divisible numbers: 2.




Constraints


    

  • 1 <= nums.size() <= 106
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def count_divisible_indices(n, nums):
    count=0
    for i in range(n):
        if i%nums[i]==0:
            count+=1 
    return count

Output Format

True 5

Element Z is a Neighbor Element if there exists at least 1 element whose absolute difference with Z is less than or equal to K i.e. an element Z is a Neighbor Element, if there is another element in the range [Z-K, Z+K] other than Z itself.




Input Format


    

  • First line contains two integers P and K respectively.
    • 

  • Second line contains P integers separated by space.
    • 





Output Format


    

  • Print a single integer denoting the total number of neighbor elements.
    • 





Constraints


    

  • 1 <=P<= 1000
    • 

  • 0 <=K<= 1000
    • 

  • 0 <=C[i]<= 1000
    • 





Explanation


Let's take an example:


4 5
10 21 13 14



arr = [10, 21, 13, 14]


For Z=10 to be Neighbor element, 
there should be atleast one element in range: [10-5, 10+5] = [5, 15].
So, this is a Neighbor element. 


For Z=21 to be Neighbor element, 
there should be atleast one element in range: [21-5, 21+5] = [16, 26].
There is no element in this range.
So, this is a not Neighbor element.


Similarily,
13 and 14 are Neighbor element

for b in num_lst:
    min_a=b - k 
    max_a = b + k 
    max_min+=[max_a,min_a]
start = min(max_min)
end = max(max_min)

    if start<= c <= end:

Input Format


    

  • First line: Integer n (number of strings in the array).

  • Second line: n space-separated strings (strArr).

Output Format


    

  • Output the number of related pairs of strings.

Constraints

    • 

  • a <= strArr[i][j] <= z
    • 





Example

4

Here:


    

  • abc is a substring of abcde resulting in 1 pair.
