Weighted Product of Matches


For a list of integers nums, calculate the Product of all elements such that nums[i] is equal to its index plus 1.


Implement the function product_matches which will take n, size of list and nums as argument and return an integer, the product.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the product.
    • 





Example 1


Input


3
1 3 3




Output


3




Explanation


    

  • For i = 0, nums[0] = 1, 0 + 1 = 1. Matches.
    • 

  • For i = 1, nums[1] = 3, 1 + 1 != 3. Does not match.
    • 

  • For i = 2, nums[2] = 3, 2 + 1 = 3. Matches.
    • 



Weighted product: 1 * 3 = 3.




Constraints


    

  • 1 <= nums.size() <= 106
    • 

  • 1 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def product_matches(n, nums):
    x=[]
    for i in range(len(nums)):
        if i+1==nums[i]:
            x.append(str(nums[i]))
    y = list(map(int,x))
    z = 1 
    for i in range(len(x)):
        n = z*y[i] 
        z = n 
    return z

  • Single line containing 3 space-separated integers representing P, R, and T respectively.

Output Format

Example 1


Input


1000 200 5




Output


2000




Explanation


Initial population is 1000.
At time 1 year, total population will be 1000+200 = 1200.
At time 2 years, total population will be 1200+200 = 1400.
At time 2 years, total population will be 1400+200 = 1600.
At time 2 years, total population will be 1600+200 = 1800.
At time 5 years, total population will be 1800+200 = 2000.
