    #Replace pass with your code
    result = ""

    vowel = "aeiou"

Sum of Matching Squares


For a list of integers nums, nums[i] is said to be Matching Square if the square of nums[i] is equal to the square of i.


Implement the function sum_matching_squares which will take n, size of list and nums as argument and return an integer, the sum of all matching square numbers in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of all matching square numbers.
    • 





Example 1


Input


3
0 1 2



Output


3



Explanation


    

  • For i = 0, nums[0] = 0, 0*0 = 0 and i*i = 0. Matching square.
    • 

  • For i = 1, nums[1] = 1, 1*1 = 1 and i*i = 1. Matching square.
    • 

  • For i = 2, nums[2] = 2, 2*2 = 4 and i*i = 4. Matching square.
    • 



Sum of matching squares: 0 + 1 + 2 = 3.




Constraints


    

  • 1 <= nums.size() <= 1000
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def sum_matching_squares(n, nums):
    sum = 0
    for i in range(n):
        num = nums[i]
        square = num ** 2
        if i**2 == square:
            sum += num
    return sum

Two strings belong to the same set if the frequency of their characters matches. For example, aabb and bbaa belong to the same set because they both have two a and two b, but abd and cba do not belong to the same set.

1. Strings consist of lowercase English letters only.
2. Pairs are considered index-wise, with identical strings at different indices counting as separate pairs.
3. A pair (i, j) is the same as (j, i).





Input Format


    

  • First line: Integer n (size of the array).
    • 

  • Second line: n space-separated strings values in (strArr).
    • 



Output Format


    

  • Output an integer representing the number of matching string pairs.
    • 





Constraints


    

  • 1 <= n <= 104
    • 

  • a <= strArr[i][j] <= z
    • 





Example

5
aabb bbaa ab xyz aabb



Here:


    

  • aabb and bbaa belong to the same set.
    • 

  • aabb occurs twice and matches itself.
    • 

  • bbaa and aabb belong to the same set.
    • 



The output is 3.

    for j in range(i+1,n):
        if len(list_a[i]) == len(list_a[j]):
