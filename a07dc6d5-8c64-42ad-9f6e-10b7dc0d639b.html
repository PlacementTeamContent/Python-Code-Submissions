Serial Number Transformation


You are given a serial number string S of length N. The serial number needs to be transformed before use. The serial number string can be transformed using the following rules:


    

  • If the k-th character in S is a vowel, change it to the fifth next character in the alphabetical sequence. For example, the fifth next character of o is t.
    • 

  • If the k-th character in S is a consonant, change it to the fifth previous character in the alphabetical sequence. For example, the fifth previous character of h is c.
    • 



Implement the function transform_serial which will take serial number string S as argument and return transformed string.



1) The fifth previous character of d is y, c is x, b is w.



Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing the serial number string.
    • 





Output Format


    

  • Single line containing the transformed serial number string.
    • 





Example 1


Input


xyz




Output


stu




Explanation


x is consonant so change to s

y is consonant so change to t

z is consonant so change to u

So, transformed serial number for xyz will be stu.


Example 2


Input


violet




Output


qntgjo




Explanation


v is consonant so change to q

i is vowel so change to n

o is vowel so change to t

l is consonant so change to g

e is vowel so change to j

t is consonant so change to o

So, transformed serial number for violet will be qntgjo.




Constraints


    

  • 1 <=N<= 106
    • 

  • The serial number string S contains only lowercase alphabets.
    • 



def transform_serial(S):
    a = "abcdefghijklmnopqrstuvwxyz"
    b = ""
    c = "aeiou"
    for i in (S):
        z = a.index(i)
        if i in c:
            b += a[z+5]
        else:
            b += a[z-5]
    return b

Good Number Sum


For a list of integers nums, nums[i] is said to be Good Number if parity of i and nums[i] will be same.


Implement the function sum_good_number which will take n, size of list and nums as argument and return an integer, sum of good number in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of good numbers.
    • 





Example 1


Input


3
1 2 3




Output


0




Explanation


    

  • For i = 0, nums[0] = 1, 1 % 2 = 1 and i % 2 = 0. Not a good number.
    • 

  • For i = 1, nums[1] = 2, 2 % 2 = 0 and i % 2 = 1. Not a good number.
    • 

  • For i = 2, nums[2] = 3, 3 % 2 = 1 and i % 2 = 0. Not a good number.
    • 



Hence, the answer is 0.


Example 2


Input


4
2 3 4 5




Output


14




Explanation


    

  • For i = 0, values[0] = 2, 2 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 1, values[1] = 3, 3 % 2 = 1 and i % 2 = 1. Good number.
    • 

  • For i = 2, values[2] = 4, 4 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 3, values[3] = 5, 5 % 2 = 1 and i % 2 = 1. Good number.
    • 



Sum of elements added: 2 + 3 + 4 + 5 = 14.
Hence, the answer is 14.




Constraints


    

  • 1 <=nums.size()<= 106
    • 

  • 1 <=nums[i]<= 1000
    • 

  • Index starts from 0
    • 



def sum_good_number(n, nums):
    count = 0 
    for i in range(n):
        if i % 2 == nums[i] % 2 :
            count += nums[i]
    return count

On-Off Lamps


In this exercise, you have N lamps that are initially off. In a sequence of rounds, you toggle these lamps according to specific rules. Initially, you turn on all the lamps. In the second round, you turn off every second lamp. In the third round, you toggle every third lamp.


For the jth round, you toggle every jth lamp. For the Nth round, you toggle only the last lamp.


Calculate the number of lamps that remain on after N rounds.




Input Format


    

  • A single integer N
    • 



Output Format


    

  • A single integer representing the number of lamps that remain on after N rounds.
    • 





Constraints


    

  • 1 <= N <= 1000
    • 





Sample Input 1


5


Sample Output 1


2


Explanation


Let's take an example:


5


    

  • At first, the five lamps are [off, off, off, off, off].
    • 

  • After the first round, the five lamps are [on, on, on, on, on].
    • 

  • After the second round, the five lamps are [on, off, on, off, on].
    • 

  • After the third round, the five lamps are [on, off, off, off, on].
    • 

  • After the fourth round, the five lamps are [on, off, off, on, on].
    • 

  • After the fifth round, the five lamps are [on, off, off, on, off].
    • 

  • So you should print 2 because there are two lamps that are on.
    • 



#Write your code here
a = int(input())
lamps = ["off"]*a
for i in range(1,a+1):
    for j in range(1,a+1):
        if j % i == 0:
            if lamps[j-1] == "on":
                lamps[j-1] = "off"
            elif lamps[j-1] == "off":
                lamps[j-1] = "on"
print(lamps.count("on"))
