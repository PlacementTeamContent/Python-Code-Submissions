Function total_calories will take C, A, and T as first, second, and third arguments respectively and return the total calories consumed after T meals.

Input Format


    

  • Single line containing 3 space-separated integers representing C, A, and T respectively.
    • 





Output Format

    • 





Example 1


Input


500 200 4



Output


1300



Explanation

At meal 2, total calories will be 700+200 = 900.

Sum of Matching Squares


For a list of integers nums, nums[i] is said to be Matching Square if the square of nums[i] is equal to the square of i.


Implement the function sum_matching_squares which will take n, size of list and nums as argument and return an integer, the sum of all matching square numbers in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of all matching square numbers.
    • 





Example 1


Input


3
0 1 2



Output


3



Explanation


    

  • For i = 0, nums[0] = 0, 0*0 = 0 and i*i = 0. Matching square.
    • 

  • For i = 1, nums[1] = 1, 1*1 = 1 and i*i = 1. Matching square.
    • 

  • For i = 2, nums[2] = 2, 2*2 = 4 and i*i = 4. Matching square.
    • 



Sum of matching squares: 0 + 1 + 2 = 3.




Constraints


    

  • 1 <= nums.size() <= 1000
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def sum_matching_squares(n, nums):
    a = 0 
    for i in range(len(nums)):
        if nums[i]**2 == i**2:
            a = a + nums[i] 
    return a

def modify_code(M):
    a = "abcdefghijklmnopqrstuvwxyz"
    b = "aeiou"
    c = ""
    for i in M:
        if i in b:
            c = c + a[a.index(i)+4]
        else:
            c = c + a[a.index(i)-4]
    return c

Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.





Input Format

    •

    • 

  • The third line contains an integer k, indicating the size of each sub-array to be considered.
    • 

  • The fourth line contains an integer m, specifying the number of elements to skip after processing each sub-array.
    • 





Output Format


    

  • The output should be a single line consisting of space-separated values, each representing the maximum value found in the respective sub-array.
    • 





Constraints


    

  • The number of elements in the array arr is greater than zero.
    • 

  • Sub-array size k and the number of skips m are positive integers.
    • 

  • k is less than or equal to the length of the array.
    •

8

3
1
