Password Encoding


You are given a password string P of length N. The password needs to be encoded before storing it to the database. The password string can be encoded using the following rules:


    

  • If the k-th character in P is a vowel, change it to the second next character in the alphabetical sequence. For example, the second next character of o is q.
    • 

  • If the k-th character in P is a consonant, change it to the second previous character in the alphabetical sequence. For example, the second previous character of h is f.
    • 



Implement the function encode which will take password string P as argument and return encoded string.



1) The second previous character of b is z.

def encode(P) :
    chr_2=""
    for i in P:
        if i in "AEIOUaeiou":
            if i=="y":
                chr_2+="a"
            elif i=="z":
                chr_2+="b"
            chr_2+=chr((ord(i))+2)
        else:
            if i=="b":
                chr_2+="z"
            elif i=="a":
                chr_2+="y"
            else:
                chr_2+=chr((ord(i))-2)
    return chr_2

Difference of Multiples


For a list of integers nums, nums[i] is said to be a Multiple if it is divisible by (i + 1)
Implement the function difference_of_multiples which will take n, size of list and nums as argument and return an integer, the absolute difference between the sum of multiples and the sum of non-multiples.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the absolute difference between sums.
    • 





Example 1


Input


4
2 3 6 8




Output


13




Explanation


    

  • For i = 0, nums[0] = 2, divisible by 1. Multiple.
    • 

  • For i = 1, nums[1] = 3, not divisible by 2. Non-multiple.
    • 

  • For i = 2, nums[2] = 6, divisible by 3. Multiple.
    • 

  • For i = 3, nums[3] = 8, divisible by 4. Multiple.
    • 



Sum of multiples: 2 + 6 + 8 = 16.
Sum of non-multiples: 3.
Absolute difference: |16 - 3| = 13.




Constraints


    

  • 1 <= nums.size() <= 106
    • 

  • -1000 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def difference_of_multiples(n, nums):
    sum_multiple=0
    sum_nonmultiple=0
    for i in range(n):
        if nums[i]%(i+1)==0:
            sum_multiple+=nums[i]
        else:
            sum_nonmultiple+=nums[i]
    if sum_multiple>sum_nonmultiple:
        return sum_multiple-sum_nonmultiple
    else:
        return sum_nonmultiple-sum_multiple

Function total_population will take P, R, and T as first, second, and third arguments respectively and return the total population after T years.


Note: You need not to write code to handle input and output.




Input Format


    

  • Single line containing 3 space-separated integers representing P, R, and T respectively.
    • 





Output Format


    

  • Single line having 1 integer representing the total population after T years.
    • 





Example 1


Input


1000 200 5




Output


2000




Explanation


Initial population is 1000.
At time 1 year, total population will be 1000+200 = 1200.
At time 2 years, total population will be 1200+200 = 1400.
At time 2 years, total population will be 1400+200 = 1600.
At time 2 years, total population will be 1600+200 = 1800.
At time 5 years, total population will be 1800+200 = 2000.




Constraints


    

  • 0 <= P, R, T < 10000
    • 



def total_population(P, R, T):
    sum_1=P
    for i in range(T):
        sum_1+=R
    return sum_1

Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.





Input Format


    

  • The first line contains n size of array.
    • 

  • The second line contains space-separated integers representing the elements of the array arr.
    • 

  • The third line contains an integer k, indicating the size of each sub-array to be considered.
    • 

  • The fourth line contains an integer m, specifying the number of elements to skip after processing each sub-array.
    • 





Output Format


    

  • The output should be a single line consisting of space-separated values, each representing the maximum value found in the respective sub-array.
    • 





Constraints


    

  • The number of elements in the array arr is greater than zero.
    • 

  • Sub-array size k and the number of skips m are positive integers.
    • 

  • k is less than or equal to the length of the array.
    • 





Example


Let's take a sample input:


8
4 7 3 9 6 1 8 0
3
1


We need to print:


7 9 9 9 6 8


Explanation


[4 7 3] 9 6 1 8 0 -> maximum is 7.
4 [7 3 9] 6 1 8 0 -> moving right by 1 place, maximum is 9.
4 7 [3 9 6] 1 8 0 -> moving right by 1 place, maximum is 9.
4 7 3 [9 6 1] 8 0 -> moving right by 1 place, maximum is 9.
4 7 3 9 [6 1 8] 0 -> moving right by 1 place, maximum is 8.
4 7 3 9 6 [1 8 0] -> moving right by 1 place, maximum is 8.
