Good Number Sum


For a list of integers nums, nums[i] is said to be Good Number if parity of i and nums[i] will be same.


Implement the function sum_good_number which will take n, size of list and nums as argument and return an integer, sum of good number in the list.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the sum of good numbers.
    • 





Example 1


Input


3
1 2 3




Output


0




Explanation


    

  • For i = 0, nums[0] = 1, 1 % 2 = 1 and i % 2 = 0. Not a good number.
    • 

  • For i = 1, nums[1] = 2, 2 % 2 = 0 and i % 2 = 1. Not a good number.
    • 

  • For i = 2, nums[2] = 3, 3 % 2 = 1 and i % 2 = 0. Not a good number.
    • 



Hence, the answer is 0.


Example 2


Input


4
2 3 4 5




Output


14




Explanation


    

  • For i = 0, values[0] = 2, 2 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 1, values[1] = 3, 3 % 2 = 1 and i % 2 = 1. Good number.
    • 

  • For i = 2, values[2] = 4, 4 % 2 = 0 and i % 2 = 0. Good number.
    • 

  • For i = 3, values[3] = 5, 5 % 2 = 1 and i % 2 = 1. Good number.
    • 



Sum of elements added: 2 + 3 + 4 + 5 = 14.
Hence, the answer is 14.




Constraints


    

  • 1 <=nums.size()<= 106
    • 

  • 1 <=nums[i]<= 1000
    •

def sum_good_number(n, nums):
    s=0
    c=False

        if i%2==nums[i]%2:

Total Calories Consumed


Implement a function total_calories to determine the total calories consumed after T meals, starting with an initial calorie intake of C calories. Each subsequent meal adds a constant number of A calories.


Function total_calories will take C, A, and T as first, second, and third arguments respectively and return the total calories consumed after T meals.


Note: You need not write code to handle input and output.




Input Format


    

  • Single line containing 3 space-separated integers representing C, A, and T respectively.
    • 





Output Format


    

  • Single line having 1 integer representing the total calories consumed after T meals.
    • 





Example 1


Input


500 200 4




Output


1300




Explanation


Initial calories consumed is 500.
At meal 1, total calories will be 500+200 = 700.
At meal 2, total calories will be 700+200 = 900.
At meal 3, total calories will be 900+200 = 1100.
At meal 4, total calories will be 1100+200 = 1300.




Constraints


    

  • 0 <= C, A, T < 5000
    • 



def total_calories(C, A, T):
    t=C
    for i in range(T):
        t+=A
    return t

Sub-array Product


Write a program that traverses an array from left to right, where in each step it examines a sub-array of size k and skips m positions forward after each examination. Your task is to print the product of values found in each sub-array considered.



Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.





Input Format


    

  • The first line contains n size of array.
    • 

  • The second line contains space-separated integers representing the elements of the array arr.
    • 

  • The third line contains an integer k, indicating the size of each sub-array to be considered.
    • 

  • The fourth line contains an integer m, specifying the number of elements to skip after processing each sub-array.
    • 





Output Format


    

  • The output should be a single line consisting of space-separated values, each representing the product found in the respective sub-array.
    • 





Constraints


    

  • The number of elements in the array arr is greater than zero.
    • 

  • Sub-array size k and the number of skips m are positive integers.
    • 

  • k is less than or equal to the length of the array.
    • 





Example


Let's take a sample input:


8
4 7 3 9 6 1 8 0
3
1


We need to print:


84 189 162 54 48 0


Explanation


[4 7 3] 9 6 1 8 0 -> product is 84.
4 [7 3 9] 6 1 8 0 -> moving right by 1 place, product is 189.
4 7 [3 9 6] 1 8 0 -> moving right by 1 place, product is 162.
4 7 3 [9 6 1] 8 0 -> moving right by 1 place, product is 54.
4 7 3 9 [6 1 8] 0 -> moving right by 1 place, product is 48.
4 7 3 9 6 [1 8 0] -> moving right by 1 place, product is 0.


n=int(input())
s=list(map(int,input().split()))
k=int(input())
m=int(input())
S=[]
i=0

    l1=s[i:i+k]
    sum1=1
    for j in range(k):
        sum1*=l1[j]
    i+=m
    S+=[str(sum1)]
print(" ".join(S))
