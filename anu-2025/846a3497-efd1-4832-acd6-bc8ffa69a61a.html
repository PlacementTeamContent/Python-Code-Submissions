Total Energy Produced by Nuclear Chemicals


You are tasked with calculating the total energy generated by combining two nuclear chemicals, starting with an initial energy M, which increases at a constant rate R every second. The reaction occurs for N seconds, and your task is to determine the total energy produced.




Input Format:


    

  • The first line contains an integer M, representing the initial energy.
    • 

  • The second line contains an integer R, representing the rate at which the energy increases per second.
    • 

  • The third line contains an integer N, representing the number of seconds the reaction occurs.
    • 





Output Format:


    

  • Print a single integer representing the total energy produced after N seconds.
    • 





Constraints:


    

  • 0 < M < 105
    • 

  • 0 < R < 105
    • 

  • 0 < N < 105
    • 





Example


Sample Input 1:


5
2
3


Sample Output 1:


21


Explanation:


Let's break down the calculation for N = 3:


    

  • For N = 1, the initial energy is 5.
    • 

  • For N = 2, the energy increases by 2, so the total energy becomes 5 + 2 = 7.
    • 

  • For N = 3, the energy increases by 2 again, so the total energy becomes 7 + 2 = 9.
    • 



The total energy produced after 3 seconds is: 5 + 7 + 9 = 21.

Comparison


You are given a sequence of integers and a target value called threshold. For each number in the sequence:


    

  • If it is greater than the threshold, count it in the larger group.
    • 

  • If it is less than the threshold, count it in the smaller group.
    • 

  • If it is equal to the threshold, ignore it.
    • 



Your task is to find out whether more numbers belong to the larger group, more numbers belong to the smaller group, or if both groups have the same number of elements.





Input Format


    

  • The first line contains an integer n, the number of elements.
    • 

  • The next n lines each contain one integer, the values to be compared.
    • 

  • The last line contains an integer threshold, the value to compare against.
    • 






Output Format


    

  • Print larger if more numbers are greater than the threshold.
    • 

  • Print smaller if more numbers are less than the threshold.
    • 

  • Print equal if both counts are the same.
    • 






Constraints


    

  • 1 ≤ n ≤ 105
    • 

  • -109 ≤ value ≤ 109
    • 

  • -109 ≤ threshold ≤ 109
    • 






Example


Input:


6
1
3
0
-1
4
3
2


Output:


equal


Explanation:


    

  • Numbers above 2 → 3, 4, 3 → count = 3
    • 

  • Numbers below 2 → 1, 0, -1 → count = 3
    • 



Since both groups have the same count, the output is equal.
