Sequence Sum Calculation


Jack loves number games, and his friend Jill challenged him to calculate the total sum of numbers following a particular sequence. The sequence starts at a number j and goes up, one step at a time, until it reaches another number l. Once it hits l, the sequence turns around and starts going down, one step at a time, until it reaches the number k. The task is to find out the total sum of all the numbers in this sequence.




Input Format:


    

  • The first line contains an integer j, which is the starting number of the sequence.
    • 

  • The second line contains an integer l, which is the number at which the sequence will stop incrementing and start decrementing.
    • 

  • The third line contains an integer k, which is the number where the sequence will stop after decrementing.
    • 



Output Format:


    

  • Print a single integer representing the sum of all numbers in the sequence
    • 





Constraints:


    

  • 0 ≤ j ≤ 100
    • 

  • 0 ≤ l ≤ 100
    • 

  • 0 ≤ k ≤ 100
    • 





Example 1


Input:


5
9
6


Output:


56




Explanation:


    

  • Jack starts at 5 and walks forward to 9. As he steps from 5 to 9, he collects all the numbers: 5, 6, 7, 8, 9.
    • 

  • Once he reaches 9, he turns around and starts walking backwards, picking up 8, 7, 6 along the way.
    • 

  • The total sum of all the numbers he picked up is: 5 + 6 + 7 + 8 + 9 + 8 + 7 + 6 = 56.
    •

Hollow Right-Angle Triangle with Bordered Base


You are tasked with printing a hollow right-angle triangle pattern that starts with a single star (*) at the top, widens on each subsequent row, and ends with a bottom row of @ symbols separated by spaces. The side borders are made of stars, and the interior is filled with spaces.



There is a space after every (@) and every (*).





Input Format:


    

  • The input contains an integer N, denoting the height of the triangle.
    • 



Output Format:


    

  • Print a hollow right-angle triangle of height N with a bordered base made of @ symbols.
    • 





Constraints:


    

  • 2 ≤ N ≤ 100
    • 





Example:


Sample Input:


6


Sample Output:


*
* *
*   *
*     *
*       *
@ @ @ @ @ @ 


Explanation:


    

  • The first row contains only one *.
    • 

  • The last row prints @  repeated N times, forming the base.
    • 

  • All middle rows start and end with *, with spaces in between that widen on each subsequent row.
    • 



a=input()
a=int("*")
b=input()
b=int("@")
