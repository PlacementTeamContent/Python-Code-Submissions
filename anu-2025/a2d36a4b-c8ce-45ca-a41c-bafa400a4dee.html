Certificate Code Generator


In a small town, a teacher is preparing certificates for her students. She wants each certificate to display a special code made from the student's name.


The rule for the code is simple:


    

  • Take the first letter of the first word of the name.
    • 

  • Take the last letter of the last word of the name.
    • 

  • Convert both letters to uppercase and join them with a dot (.).
    • 



Your task is to complete the method Find_First_Last_Letter() which receives a string S and return the code in the required format.




- Do not change the method name, as the test cases will fail if modified.

- Reading input is not required; use the parameter passed to the method.

- No need to print the output, just return the result.

  




Input Format


The input for the method Find_First_Last_Letter() receives a string S (the student’s full name).




Output Format


The method should return a string in the format X.Y, where:


    

  • X is the uppercase first letter of the first word.
    • 

  • Y is the uppercase last letter of the last word.
    • 





Constraints


    

  • 1 ≤ len(S) ≤ 102
    • 





Example


Input


Sam Harris


Output


S.S




Explanation


Here, the name is Sam Harris.


    

  • The first letter of the first word is S.
    • 

  • The last letter of the last word is S.
  Thus, the output is S.S.

Middle Value Finder


Given three distinct numbers a, b, and c, find the number with the middle value.




Input Format


    

  • Three distinct integer inputs a, b, and c are given as input on separate lines.
    • 



Output Format


    

  • Output the integer that has the middle value.
    • 





Constraints


    

  • The three integers are distinct.
    • 

  • 0 <= a,b,c<=103.
    • 





Example 1


Input


12
32
11


Output


12


Explanation


Among the numbers 12, 32, and 11:


    

  • 11 is the smallest.
    • 

  • 32 is the largest.
    • 

  • Therefore, 12 is the middle value.
    •
