Sum of Numbers Divisible by Both 3 and 5


You are tasked with calculating the sum of numbers divisible by both 3 and 5 within a given range [m, n] (both inclusive). 




Input Format:


    

  • The first line contains an integer m representing the starting point of the range.
    • 

  • The second line contains an integer n representing the endpoint of the range.
    • 





Output Format:


    

  • Print the sum of numbers divisible by both 3 and 5 in the range from m to n (inclusive).
    • 





Constraints:


    

  • 0 <= m <= n
    • 

  • 0 <= m, n <= 106
    • 





Example:


Sample Input 1:


12
50


Sample Output 1:


90


Explanation:


The numbers divisible by both 3 and 5, between 12 and 50 (both inclusive) are:


    

  • {15, 30, 45}, and their sum is 15 + 30 + 45 = 90.

Remove Character from String


Given a string str and a character X, your task is to eliminate all instances of the character X from the string str and print the resulting string.




Input Format:


    

  • The first line contains a string representing str.
    • 

  • The second line contains a character X.
    • 





Output Format:


    

  • Print the string after removing all occurrences of the character X.
    • 





Constraints:


    

  • 1 <= len(str) <= 104
    • 





Example:


Input:


hello world
o


Sample Output:


hell wrld


Explanation:


    

  • In the given string "hello world", the character 'o' is removed. After removal, the string becomes "hell wrld".
