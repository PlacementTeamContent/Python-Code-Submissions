Quick Text Replacements


A content editor needs to prepare text for a campaign by replacing specific keywords with short codes for easier processing. Your task is to write a program that processes the given text word-by-word and replaces every occurrence of the word "Hunting" with "xyz" and "Hunt" with "123", preserving the rest of the text.



- Replacements are case-sensitive; only the words "Hunt" and "Hunting" exactly as shown should be replaced.
- Punctuation adjacent to the keywords should be preserved and should not affect replacements.





Input Format


    

  • A single line containing the text to modify.
    • 



Output Format


    

  • A single line containing the modified text after replacements.
    • 





Constraints


    

  • 1 ≤ Length of text ≤ 1000  
    • 





Example


Input


Hunt by Huntingman.


Output


123 by xyzman.


Explanation


    

  • In the given text, "Hunting" is replaced by "xyz" and "Hunt" is replaced by "123", resulting in "123 by xyzman.".

Find Me Out


Given a number N, little Johnny wants to find the minimum positive integer X such that:


    

  • X is divisible by N.
    • 

  • The sum of digits of X is equal to N.
    • 

  • X is not equal to N.
    • 



If such a number X does not exist, print -1.




Input Format


    

  • A single integer N representing the given number.
    • 





Output Format


    

  • Print the minimum positive integer X satisfying the above conditions or -1 if no such number exists.
    • 





Constraints


    

  • 1 ≤ N ≤ 100
    • 

  • 1 ≤ X ≤ 107
    • 





Example


Sample Input:


9


Sample Output:


18


Explanation:


    

  • For N = 9, X = 18 is divisible by 9, and the sum of its digits 1 + 8 = 9 matches N.
