Average of First P Multiples of an Integer N


You are given an integer N and a number P. Your task is to compute the average of the first P multiples of N.




Input Format:


    

  • The first line contains an integer N, denoting the given number.
    • 

  • The second line contains an integer P, denoting the number of multiples to consider.
    • 





Output Format:


    

  • Print an integer value denoting the average of the first P multiples of the integer N. 
    • 





If the result is a floating point number, print its integer part.





Constraints:


    

  • 1 ≤ N ≤ 105
    • 

  • 1 ≤ P ≤ 105
    • 





Example


Sample Input:


6
5


Sample Output:


18


Explanation:


    

  • In this case, the first 5 multiples of 6 are: 6, 12, 18, 24, 30.
    • 

  • The sum of these multiples is: 6 + 12 + 18 + 24 + 30 = 90.
    • 

  • The average is: 90 / 5 = 18.
    • 

a=int(input())
b=int(input())
counter=0
for i in range (1,b+1):
    multiple=a*i
    counter=counter+multiple
    average=counter/b
    
print(int(average))

There is a space after every (*) and every (# ).





Input Format:


    

  • The input contains an integer N, denoting the number of rows in the triangle.

Output Format:

Constraints:


    

  • 2 ≤ N ≤ 100
    • 





Example:


Sample Input:


5
