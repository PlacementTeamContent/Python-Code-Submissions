Weighted Product of Matches


For a list of integers nums, calculate the Product of all elements such that nums[i] is equal to its index plus 1.


Implement the function product_matches which will take n, size of list and nums as argument and return an integer, the product.


Note: You need not to write code to handle input and output.




Input Format


    

  • First line contains an integer n, the size of nums.
    • 

  • Second line contains n space-separated integers, numbers in list.
    • 





Output Format


    

  • One line containing an integer, the product.
    • 





Example 1


Input


3
1 3 3



Output


3



Explanation


    

  • For i = 0, nums[0] = 1, 0 + 1 = 1. Matches.
    • 

  • For i = 1, nums[1] = 3, 1 + 1 != 3. Does not match.
    • 

  • For i = 2, nums[2] = 3, 2 + 1 = 3. Matches.
    • 



Weighted product: 1 * 3 = 3.




Constraints


    

  • 1 <= nums.size() <= 106
    • 

  • 1 <= nums[i] <= 1000
    • 

  • Index starts from 0
    • 



def product_matches(n, nums):
    weighted_product = 1 
    for i in range(n):
        if nums[i] == i+1:
            weighted_product *= nums[i]
    return weighted_product

Serial Number Transformation

  • If the k-th character in S is a vowel, change it to the fifth next character in the alphabetical sequence. For example, the fifth next character of o is t.
    • 

  • If the k-th character in S is a consonant, change it to the fifth previous character in the alphabetical sequence. For example, the fifth previous character of h is c.
    • 



Implement the function transform_serial which will take serial number string S as argument and return transformed string.



1) The fifth previous character of d is y, c is x, b is w.



Note: You need not to write code to handle input and output.

def transform_serial(S):
    alphabets = "abcdefghijklmnopqrstuvwxyz"
    vowels = "aeiou"
    result = ''
    for alphabet in S:
        index_alpha = alphabets.find(alphabet)
        if alphabet in vowels:
            result += alphabets[index_alpha+5]
        else:
            result += alphabets[index_alpha-5]
    return result

Total Steps Walked


Implement a function total_steps to determine the total steps walked after T days, starting with an initial count of S steps. The number of steps increases by a constant rate of I steps per day.


Function total_steps will take S, I, and T as first, second, and third arguments respectively and return the total steps walked after T days.


Note: You need not write code to handle input and output.




Input Format


    

  • Single line containing 3 space-separated integers representing S, I, and T respectively.
    • 





Output Format


    

  • Single line having 1 integer representing the total steps walked after T days.
    • 





Example 1


Input


1000 500 3



Output


2500



Explanation


Initial steps walked is 1000.
At day 1, total steps will be 1000+500 = 1500.
At day 2, total steps will be 1500+500 = 2000.
At day 3, total steps will be 2500.




Constraints


    

  • 0 <= S, I, T < 20000
    • 



def total_steps(S, I, T):
    for i in range(T):
        S = S + I
        if i == T-1:
            break 
    return S

Average of Sub-arrays


Write a program that traverses an array from left to right, where in each step it examines a sub-array of size k and skips m positions forward after each examination. Your task is to print the average of values found in each sub-array considered.



Processing should stop if there are fewer than k elements remaining in the array after skipping m elements.





Input Format


    

  • The first line contains n size of array.
    • 

  • The second line contains space-separated integers representing the elements of the array arr.
    • 

  • The third line contains an integer k, indicating the size of each sub-array to be considered.
    • 

  • The fourth line contains an integer m, specifying the number of elements to skip after processing each sub-array.
    • 





Output Format


    

  • The output should be a single line consisting of space-separated values, each representing the average found in the respective sub-array.
    • 





Constraints


    

  • The number of elements in the array arr is greater than zero.
    • 

  • Sub-array size k and the number of skips m are positive integers.
    • 

  • k is less than or equal to the length of the array.
    • 





Example


Let's take a sample input:


8
4 7 3 9 6 1 8 0
3
1



We need to print:


4.67 6.33 6.00 5.33 5.00 3.00



Explanation


[4 7 3] 9 6 1 8 0 -> average is 4.67.
4 [7 3 9] 6 1 8 0 -> moving right by 1 place, average is 6.33.
4 7 [3 9 6] 1 8 0 -> moving right by 1 place, average is 6.00.
4 7 3 [9 6 1] 8 0 -> moving right by 1 place, average is 5.33.
4 7 3 9 [6 1 8] 0 -> moving right by 1 place, average is 5.00.
4 7 3 9 6 [1 8 0] -> moving right by 1 place, average is 3.00.

n = int(input())

for i in range(n):
    sub_arr = [arr[j] for j in range(i, i+k)]

    avg_sub_arr = round(sum_sub_arr/k, 2)
    avg_sub_arr = str(avg_sub_arr)

        avg_sub_arr += '0'

        avg_sub_arr += '0'
